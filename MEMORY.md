## 记忆进化规则

- 记忆不是一次性配置，要随着用户复盘、交割单、短线心得和纠错记录持续进化。
- 每次复盘后必须主动总结：新术语、新偏好、新规则、反复错误、仓位习惯、买卖触发条件。
- 原始口述只进 RAW；提炼后的长期规则才进入 MEMORY 或 WIKI。
- 不把单日情绪、单只股票临时判断直接升级为长期规则。
- 出现 2 次以上的重复表达、重复错误、重复成功模式，要记录为“待验证记忆”。
- 用户明确确认的规则，才能升级为“稳定记忆”。
- 记忆更新要短、准、可执行，避免堆长文。
- 如果发现旧记忆和新复盘冲突，要标注冲突，不直接覆盖。
- 飞书对话里暴露出来的交互问题也属于记忆进化输入，不限于交易复盘本身。
- 只要用户在飞书里指出体验问题，默认动作是：定位原因 -> 改桥或改规则 -> 写入记忆/进化日志 -> 后续继续迭代。
- 飞书里用户纠正判断的短句默认进入校准闭环，例如“这条没用，降权”“这条有效，升权”“写成规则”“明天验证这条”“只给我结论”。系统要自动写 RAW、消息权重人工校准表、待验证队列和记忆进化日志，不把这类短句当普通聊天。
- 飞书里的回复偏好要尽量做成“持久偏好 + 即时回显”，例如切换到快答模式、分析模式、复盘模式、盯盘模式后，后续路由和回答风格都要跟着变。
- 只要是在判断消息、题材、个股的有效性或权重，默认输出结构是：判断对象 -> 结论 -> 2-3条理由 -> 后续验证点。这样用户才能直接做校准。
- 飞书会话要同时保留“最近对话”和“会话要求笔记”。前者承接上下文，后者承接用户反复强调的要求；回答前优先读取两者。
- 飞书里要提供自检命令，至少支持“当前会话记住了什么”“查看上下文”，让用户随时抽查系统到底记住了什么。

## 常用交易术语

- "窗滚" = 异动窗口向前滚动（挤出旧收盘、纳入新收盘）。
- "绕异动" = 主动断板/回调，等最低收盘被窗口滚出后再涨，规避异动监管。
- "分歧" = 市场对方向观点不一致，分歧买入是最佳入场点。
- "一冰/二冰/三冰" = 冰点序列，一冰清仓、二冰三冰是买点，越冰越买。
- "对等换仓" = 短线自然形态，仓位只跟确定性有关。
- "炒新不炒旧" = 题材大小决定是否切换，要信早信。
- "半路" = 盘中介入（非打板），>=6%只能板上买，2-5%半路有盈亏比。
- "退潮日" = 市场情绪下行日。
- "强退潮日" = 情绪0-3分。
- "中等退潮日" = 情绪4-5分。
- "出监管日" = 解除异动监管限制的日期。
- "预期差" = 市场预期与实际表现之间的差距。
- "大票/小票" = 大盘股/小盘股。
- "天地板" = 涨停到跌停。
- "炸板" = 涨停后打开未回封。

## 核心策略框架

- 一切围绕确定性做决策。
- 纠错能力 = 短线第一原则。
- 分歧买入 = 最佳入场点。
- 买卖决策独立，不可互相绑架。
- 仓位只跟确定性有关，没有刚性百分比规则。
- 出监管日卖点：核心判断是明天会不会被监管。
- 外部利好映射：新题材刚启动可追，老题材高开要跑。
- 异动卖出：特指10天100%/30天200%监管异动规则。

## 强退潮日规则

- 一冰清仓，二冰三冰拿先手。
- 越冰越买，而不是越冰越空仓。

## 通达信异动计算记忆

- 3日/10日/30日异动，各板块阈值不同。
- 沪主板/深主板：3日20%。
- 创业板/科创板：3日30%。
- 10日100%，30日200%：全部板块统一。
