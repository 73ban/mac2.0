## 四层嵌套决策体系

| 层级 | 名称   | 职责               | 对应 wiki    |
| -- | ---- | ---------------- | ---------- |
| L1 | 市场环境 | 当前是什么市？主升/震荡/退潮？ | 市场环境/      |
| L2 | 方向选择 | 哪个板块/概念是真正主线？    | 分析/ 概念/    |
| L3 | 个股选择 | 主线里哪个标的逻辑最硬？     | 股票/        |
| L4 | 执行控制 | 买不买、买多少、何时止损？    | 策略/L4执行控制层 |

## 五个支撑模块

| 模块  | 职责                       |
| --- | ------------------------ |
| 模式/ | 可复用市场模式（恐慌修复、尾盘抢筹、高位分歧等） |
| 错误/ | 反面教材库（每次犯错归档）            |
| 进化/ | 系统迭代日志                   |
| 概念/ | 领域知识卡片                   |
| 股票/ | 个股跟踪（逻辑/催化/估值）           |

## 工作流程

```
开盘前：市场环境/ → 分析/ → 策略/四层嵌套（L1→L2→L3→L4）
收盘后：分析/主线识别 → 更新市场环境 → 错误归档
周末：  进化/系统进化日志 → 错误/批量回顾
```
