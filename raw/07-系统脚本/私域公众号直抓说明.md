# 私域公众号直抓说明

适用场景：

- 公众号关闭搜索
- WeRSS 里 `mp_id -> /articles` 拉不到历史列表
- 只能通过你手里的文章链接或转发链接抓取

当前已接通的私域入口：

- 链接种子文件：`/Users/qixinchaye/wiki/73神话/.system/wechat-mp-url-seeds.json`
- 每个作者一个目录，例如：
  - `/Users/qixinchaye/wiki/73神话/raw/05-研报新闻/公众号/游资号/爱在冰川`
  - `/Users/qixinchaye/wiki/73神话/raw/05-研报新闻/公众号/游资号/作手三少`

投喂方式：

1. 把微信文章链接加入 `.system/wechat-mp-url-seeds.json`
2. 15 分钟主任务自动执行 URL 直抓
3. 已导入过的链接会按状态和去重规则自动跳过
4. 抓到图片时进入 OCR，OCR 文本随 RAW 进入游资学习池

示例：

```text
https://mp.weixin.qq.com/s/AZfN6WHmsHecIbKPQyid8g
https://mp.weixin.qq.com/s/JeQlLQtIGjZsoWAXSbhBcg
```

后台处理链路：

`wechat-mp-url-seeds.json -> fetch-wechat-mp-seeds.mjs -> 正文清洗 -> 图片本地化/OCR -> RAW -> 游资学习状态 -> 作者页/主题页/质量审计`

落地位置：

- RAW：`/Users/qixinchaye/wiki/73神话/raw/05-研报新闻/公众号/游资号/作者名/`
- URL 种子：`/Users/qixinchaye/wiki/73神话/.system/wechat-mp-url-seeds.json`
- 15 分钟主任务：`/Users/qixinchaye/wiki/73神话/.system/scripts/run-cloud-data-connectors.mjs`

当前限制：

- 私域号最新文章无法靠公众号搜索自动发现
- 必须先拿到文章链接，系统才能抓取
- 一旦拿到链接，后续正文、图片、OCR、学习闭环都是自动的
