# 空方证据占优时的多头偏见

## 错误模式定义

**当空方信号数量、强度、时效性均明显优于多方信号时，仍倾向于寻找买入/持仓理由，而非优先评估风险控制。**

## 案例：2026-06-10 日韩早报

### 信号格局

| 维度 | 空方 | 多方 |
|------|------|------|
| 信号数量 | 5个 | 2个 |
| 强信号 | 2个（KOSPI熔断、全球撤退） | 0个（涨价是中长期） |
| 时效性 | 即时发生 | 3周后生效 |
| 综合强度 | ⭐⭐⭐⭐ | ⭐⭐⭐ |

### 作者的矛盾表述

**正文（多头倾向）**：
> "村田涨价逻辑未变 → 跌多了可能是分歧买入机会"

**风险提示（空头承认）**：
> "KOSPI 4天2次熔断是系统性风险信号，不可忽视"

两句话在同一篇文档中，**未给出权衡框架**。这是经典的多头偏见表现：承认风险，但行动建议仍倾向多头。

### 为什么这是偏见

1. **时间框架错配**：用3周后的涨价逻辑对冲今日的全球抛售
2. **信号选择性过滤**：5个空方信号被概括对待，2个多方信号被强调
3. **缺乏风险量化**："分歧买入机会"未附带任何止损条件或仓位限制
4. **获利盘压力被低估**：A股6/9多只MLCC涨停是独立利空，仅一笔带过

## 这不是孤立案例

参见：[[达实智能]] — 物理AI板块减持崩盘事件中，同样存在"逻辑正确掩盖短期风险"的模式。

**共同特征**：
- 持有标的的"基本面逻辑"被用来对冲利空
- 利空信号的强度和时效性被低估
- 风险控制措施缺位或后置

## 识别信号

当出现以下组合时，警惕"多头偏见"：

| 信号 | 含义 |
|------|------|
| ✅ 你持有该标的 | 利益相关，天然偏多 |
| ✅ 空方信号数量 > 多方 | 格局不利 |
| ✅ 多方逻辑是"中长期"的 | 时间框架不匹配 |
| ✅ 你没有写出止损条件 | 风险控制缺位 |
| ✅ 你觉得"跌多是机会" | 经典的偏见触发词 |

## 防范机制

### 规则
1. **空方信号强度≥多方时，先执行风险审计，再考虑机会**
2. 所有"分歧买入"建议必须附带：止损位、最大仓位、失效条件
3. 警惕"中长期逻辑支撑短期操作"的时间错配
4. 撰写盘前分析时，明确区分"客观信号评估"和"个人倾向"
