# 冰点抄底

## 定义

**冰点抄底**是指在市场情绪周期处于[[概念/退潮日|冰点]]时，基于“价格已经跌了很多”、“比昨天便宜”等锚定思维，在下跌趋势中买入前期强势股（尤其是高标），预期其反弹的错误交易行为。

核心公式：
> **冰点 ≠ 买点** — 冰点是观望信号，不是行动信号。

## 经典案例：2024-09-12 深圳华强

### 市场环境

| 指标 | 数值 | 信号含义 |
|------|------|----------|
| 两市成交 | 5156亿（地量） | 资金观望，反弹动能不足 |
| 高标补跌 | 老百姓3跌停、大众交通4跌停 | 前期龙头仍在杀跌，退潮未结束 |
| 连板可参与性 | 最高标全部一字板 | 无真实参与窗口 |
| 情绪周期 | 冰点 | 应空仓或极轻仓等待 |

### 用户操作

| 行为 | 细节 |
|------|------|
| 买入标的 | 深圳华强（000062），前期强势股，5日跌21% |
| 成交价 | 33.72元（当日收盘32.75，建仓即浮亏2.9%） |
| 仓位 | 约60%，融资账户单吊 |
| 锚定参考 | “低于前日收盘34.28” |

### 错误拆解

1. **忽略趋势** — 5日跌21%不是止跌信号，可能是加速下跌中继
2. **忽略环境** — 老百姓/大众交通已示范高标补跌，华强同属此列
3. **锚定错误** — 以34.28和41.55为锚，而非32.75的市场价格
4. **仓位倒挂** — 最高不确定性 + 最高仓位（60%融资单吊）= 必然亏损

### 用户的认知-行动悖论

用户在复盘中明确判断市场“真实参与感极差”，却仍融资重仓买入。这暴露了：
- **认知层面**：知道市场没机会
- **行动层面**：仍强行制造交易
- **断裂点**：从认知到执行的纪律缺失

## 防范规则

### 冰期交易检查清单

进入交易前，必须确认以下三项全部通过：

- [ ] **冰点确认**：是否处于退潮/冰点？（是 → 考虑暂停）
- [ ] **高标补跌**：前期高标是否仍在补跌？（是 → 不碰同类标的）
- [ ] **参与窗口**：是否有换手充分的可参与机会？（无 → 不交易）

### 仓位约束

- 冰期总仓位：≤20%（不融资）
- 冰期单票上限：≤10%
- 不确定性越高，仓位越低——这是仓位与确定性的匹配原则
