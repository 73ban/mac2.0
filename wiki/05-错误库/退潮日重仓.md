# 退潮日重仓

## 定义

在明确识别市场进入**退潮日**（涨停家数腰斩、旧龙头A杀跌停、情绪骤降）的情况下，仍进行大幅加仓、甚至全仓操作的行为。这是“认知-执行”断裂的极端表现，是交易纪律崩溃的核心错误类型。

## 为什么这是致命错误？

| 维度 | 正常日 | 退潮日 |
|------|--------|--------|
| 上涨概率 | 50%+ | ＜30% |
| 板块持续性 | 主线有接力 | 板块快速轮动、一日游 |
| 高标表现 | 分歧后修复 | 连续跌停、A杀 |
| 仓位策略 | 可重仓主线 | **必须轻仓或空仓** |

退潮日的核心特征是 **流动性迅速萎缩**，当日买入的股票次日极易遭遇低开、甚至低开跌停（参见 [[概念/一字跌停流动性归零]]）。

## 案例：2024-11-12 退潮日重仓信创

### 市场环境

- 涨停从199家骤减至104家，单日减少95家。
- 旧周期龙头 [[股票/华映科技]]、蓝黛科技、双成药业连续一字跌停。
- 涨跌比580:1687，**74%个股下跌**。
- 用户自评：**“全面退潮！”**，市场情绪评分4/10。

### 操作行为

| 代码 | 名称 | 操作 | 金额 |
|------|------|------|------|
| 600536 | 中国软件 | 融资+证券买入 | ~11.5万（占总仓位67%） |
| 000066 | 中国长城 | 融资买入 | ~4.1万 |
| 601138 | 工业富联 | 证券买入 | ~0.26万 |
| **合计** | | | **~15.8万净买入** |

- 总仓位从接近空仓（11/11）→ **约17.2万全仓**。
- 操作方向：**信创（国产软件/国产CPU）**。
- 买点：**早盘追高**（中国软件66.87-67.80，当日未涨停）。

### 错误诊断

1.  **认知-执行断裂**：用户已清晰判断“全面退潮”，但操作上却反向满仓。
2.  **仓位失控**：单票（中国软件）占仓位67%，在退潮期极度危险。
3.  **选股与买点粗糙**：中国软件当日不在涨停榜，非最强票，且买在早盘高位。
4.  **动机推测**：11月累计亏损-9,846元，急于扭亏。属 [[概念/贪婪驱动的纪律撤销]] 典型表现——为抓住“内部缩圈”机会而不计后果。

### 后果

此日重仓埋下了后续亏损的伏笔（次日走势需查阅后续复盘交割单验证）。

## 案例：2026-06-05 豫能控股天地板

详见：[[2026-06-05-豫能控股天地板三次卖点错过]]

核心事实：

| 项目 | 内容 |
|---|---|
| 市场环境 | 中等退潮，情绪约 4/10 |
| 板块环境 | 电力板块集体崩塌，豫能控股天地板，华电能源/广西能源/粤电力A跌停 |
| 账户影响 | 当日盈亏 -38,134（-8.6%） |
| 核心错误 | 满仓满融未降仓，三次卖点全部错过 |

规则强化：

- 情绪≤5 且持仓板块负反馈，先降仓到半仓以下。
- 龙头开盘 5 分钟该强不强，确定性自动扣分。
- 龙头从高位砸到零轴，确定性归零，立即清仓。
- 跌停封死时先挂卖出，不幻想翘板。

## 正确做法：退潮日的操作铁律

1.  **仓位**：总仓位不超过20%，或直接空仓。
2.  **买点**：退潮日不开新仓。即使开，必须尾盘（14:50后）确认涨停封死再小仓位排板。
3.  **选股**：只考虑当日**最强势、有板块效应、且封单巨大的涨停票**（如当日的和而泰18.4亿一字）。中国软件这类非涨停票，完全没有买入理由。
4.  **纪律优先于判断**：当自己的“退潮”判断与“想买”的冲动冲突时，纪律必须胜出。这正是[[策略/退潮票早盘清仓]]的反向应用——退潮日不仅不能买，而且是卖出旧仓位的最后窗口。
