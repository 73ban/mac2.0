# 回血后激进反弹

## 定义

**回血后激进反弹** 是指：交易者在经历亏损并回本后，非但没有冷静反思风险，反而因“回血”带来的信心膨胀，以**比之前更高**的仓位重新杀入市场。

这是弹药装填链的核心驱动心理之一，与 [[概念/恐惧驱动的纪律执行]] 形成镜像对称。

## 核心机制

### 为什么回血会导致激进？
1. **幸存者偏差被强化**：“看，我亏了也能赚回来”——回血被错误解读为能力的证明，而非运气的眷顾。
2. **痛感消退**：亏损时的痛苦记忆在回血瞬间被覆盖，风险感知被重置。
3. **时间紧迫感**：回血后产生“应该趁机扩大战果”的FOMO，担心不立刻加仓就会错过。
4. **账户锚定效应**：回血到前高时，心理锚点变成了“现在就是安全的起点，可以放大仓位”。

### 与恐惧驱动纪律执行的对立
| 触发条件 | [[概念/恐惧驱动的纪律执行]] | [[错误/回血后激进反弹]] |
|---------|--------------------------|----------------------|
| 触发事件 | 亏损 | 回血 |
| 行为方向 | 降仓、严守止损、更谨慎 | 升仓、放松纪律、更激进 |
| 心理状态 | 恐惧→保护 | 自信膨胀→进攻 |
| 对账户的影响 | 正面（保命） | 负面（积累风险） |

两者构成了一个完整的对称模型：**亏损和回血都可能扭曲行为，但方向相反。** Wiki此前只系统化记录了前者（恐惧驱动的纪律执行），回血后激进反弹的识别填补了这个对称模型的后半部分。

## 在5月灾难中的角色

### 关键节点：2026年5月12日
1. 5/5-5/11期间清仓银之杰/综艺股份（4/30节前装填的头寸）。
2. 清仓后资金回血到接近前高水平。
3. 回血当日（5/12）即有融资重仓天通/吉翔的重大操作——**没有任何冷静期，仓位没有降低，反而更大。**
4. 这是弹药装填链从I（节前赌博）向II（回血反弹）的质变点。

### 与过度补偿的区别
- **过度补偿**（如5/23-26满仓风华高科）是由“**错过**”驱动的报复性追仓。
- **回血后激进反弹**是由“**回本**”驱动的信心膨胀式升仓。
- 两者可能叠加：回血后→错过反弹→过度补偿，形成复合型灾难。

## 与过度补偿的关系

5/23-26满仓风华高科的操作可以视为 **回血后激进反弹（5/12）+ 过度补偿（5/23-26）** 的复合作用：
1. 5/12回血→立刻升仓（回血后激进反弹）
2. 5/22高潮日满仓→次日亏损（损失）
3. 5/23-26→错过反弹→报复性满仓风华高科（过度补偿 = [[错误/锚定回追]] 的变体）

## 识别信号

| 信号 | 说明 |
|------|------|
| 回血当日或次日即开新仓 | 缺乏冷静期 |
| 新仓仓位 > 旧仓仓位 | 仓位放大 |
| 新仓使用融资 | 杠杆升级 |
| 开仓理由中不包含风险讨论 | 风险感知被重置 |
| 能回忆起“我之前亏过但赚回来了”的叙事 | 幸存者偏差强化 |

## 防范措施

1. **强制冷静期**：任何回血到前高±5%的位置时，强制24-48小时不开新仓。
2. **回血后降仓规则**：回血到位时，下一笔交易的仓位上限 = 前一笔仓位上限 × 50%（而非100%或更大）。
3. **回血日记**：每次回血时记录当时的心理状态和开仓动机，事后回溯审核。
4. **逆反测试**：每次开仓前问自己：“如果我这笔交易不是因为回血了而是因为亏损中，我还会开这个仓位吗？”

## 一句话总结
**回血不是绿灯，是黄灯。账户回到前高时，风险并未回到零点——恰恰相反，这是弹药装填链加速的起点。亏损和回血都能扭曲行为，恐惧驱动降仓、回血驱动升仓，两者必须用规则对冲，而非依赖情绪。**
