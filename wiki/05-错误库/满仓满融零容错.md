# 满仓满融零容错

## 错误描述

在交易中持续保持满仓满融状态，现金与负债极度接近，净资产仅数千元，无任何容错空间。

## 5/27 证据

| 指标 | 数据 |
|------|------|
| 现金 | 185,314 |
| 融资负债 | 184,021 |
| 净资产 | 1,293 |
| 容错空间 | 0.6% |

**历史模式**：5/25释放现金18万 → 5/26全部打光 → 5/27清仓收回现金 → 同一天全部打入新仓。现金从未真正留存超过一天。

## 危害

1. **退潮日无法防御**：满仓满融状态下，退潮日杀跌直接重伤，无现金缓冲
2. **机会成本归零**：没有现金意味着看到更好机会时无法调仓，只能割旧买新
3. **心态绑架**：满仓满融使任何日内波动都牵动神经，容易触发 [[概念/恐惧驱动的纪律执行]] 或 [[概念/贪婪驱动的纪律撤销]]
4. **强制止损被动**：一旦方向错误，必须在亏损状态砍仓，没有扛单的余地

## 解决方案

1. **退潮日强制降仓**：与 [[策略/退潮票早盘清仓]] 绑定，退潮日清仓后不买新仓，保留现金过夜
2. **设置现金下限**：无论市场多好，始终保留10-20%现金
3. **融资额度分级**：主升期可用满融，震荡期半融，退潮期零融

## 关联错误

- 与 [[概念/集中暴击亏损模式]] 共振：满仓满融+方向过度集中=致命组合
- 与 **[[概念/正反馈循环的自我破坏]]** 相关：盈利后加仓→满仓满融→回调亏损
