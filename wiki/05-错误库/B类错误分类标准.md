# B 类错误分类标准

## 定义

B 类错误指因 **买入价位过高、追涨停或炸板后追入**，以及在出现明显卖出信号时 **未及时止损** 造成的亏损。

## 典型特征

- 买入点直接导致持仓成较高，安全边际极低。
- 盘中或盘后出现明确离场信号（如破位、利好兑现退潮），但未执行止损。
- 常发生在高潮次日、炸板回封失败等场景，情绪驱动大于逻辑驱动。

## 与其他错误类别的区分

| 类别 | 说明 |
|------|------|
| A 类错误 | 系统性问题（如仓位失控、违反风控红线） |
| **B 类错误** | **追高入场或未及时止损（战术层面纪律缺陷）** |
| C 类错误 | 基本面误判或逻辑错误（分析能力缺陷） |

> ⚠️ 此分类为暂定框架，后续可随更多案例归档进一步细化。

## 鹏鼎控股 5/23 案例（假设性）

- **疑似错误类型**：B 类 — PCB 高潮次日追入或未止损。
- **信号假设**：若鹏鼎在早盘高开低走且未执行纪律，则属典型 B 类错误。
- **防范规则**：高潮次日不追高；早盘放量下挫必须止损，不留隐患。

## 通用防范规则

1. **买点纪律**：不追单日高潮后的开盘冲高，不排涨停板开板。
2. **卖点纪律**：止损线一旦触发，必须执行，不找理由推延。
3. **仓位管理**：单票在高位区域的仓位不得超过常规上限，防止小过错放大亏损。
4. **复盘归档**：每次 B 类错误必须录入 [[错误/错误案例库]]，并追溯到具体信号和决策过程。
