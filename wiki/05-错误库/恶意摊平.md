# 恶意摊平

> 又名：补仓幻觉、成本下降错觉

## 定义

**恶意摊平**是指：在初始买入后股价向不利方向运行，交易者没有按计划止损，而是在下跌过程中持续加仓，试图通过降低平均持仓成本来"挽救"交易的行为。

## 为什么"恶意"？

摊平买入看似理性（"我在降低成本"），但本质是：

| 表象 | 实质 |
|------|------|
| "降低成本" | 拒绝承认初始判断失误 |
| "等待反弹回本" | 放大亏损敞口 |
| "越跌越有价值" | 没有重新评估逻辑 |
| "再买一点就能拉回均价" | 赌徒式追加 |

**核心幻觉**：你并没有"降低成本"，你只是在同一笔错误上押注了更多资金。

## 典型案例：合百集团 2026-05-22

```
买入路径：10.24 → 9.98 → 9.88 → 9.81 → 9.87 → 9.88 → 9.89 → 10.24 → 10.24
买入笔数：10笔
总金额：约24.4万
渠道：融资 + 证券
```

这10笔买入中，前3笔可能是主动建仓，后7笔是典型的被动摊平。最后一笔融资在10.24元买入，说明交易者仍未放弃，可能在价格反弹时再次追入——这正是恶性循环的开始。

## 恶意摊平的三个阶段

| 阶段 | 描述 | 心理状态 |
|------|------|----------|
| **第一阶段：追高** | 在相对高位首次买入，可能基于突破/龙头/消息 | 贪婪、怕错过 |
| **第二阶段：补仓** | 股价下跌2-5%时加仓，"降低平均成本" | 否认、合理化 |
| **第三阶段：沉没** | 股价持续下跌仍继续买入，动用融资 | 绝望、复仇式交易 |

## 与已有错误模式的关系

- **[[概念/集中暴击亏损模式]]**：恶意摊平是集中暴击的放大器——先集中、再摊平，双重叠加。
- **[[错误/锚定回追]]**：锚定于"我最初买入的价格是对的"，回追"补仓"以证明自己没错。
- **[[概念/贪婪驱动的纪律撤销]]**：如果初始买入有止损计划，下跌中加仓就是纪律的自我撤销。

## 识别恶意摊平的检查问题

1. 价格下跌后，我是在重新评估逻辑，还是在看"还能买多少"？
2. 我的加仓是为了执行计划，还是为了"拉低均价"？
3. 如果这笔交易最终亏光，我能否承受？

## 预防铁律

> **"下跌不加仓，除非有新的买入逻辑。"**
>
> 如果买入理由已被市场否定，任何加仓都是恶意摊平。
