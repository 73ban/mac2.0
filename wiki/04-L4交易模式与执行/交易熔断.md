# 交易熔断规则

## 目的
通过物理限制切断“亏损 → 清仓伪装 → 报复性交易 → 旧方向沉船跳”的情绪传导链，在账户出现显著亏损时强制进入冷静期，避免日内二次伤害。



## 核心原则

亏损本身不是问题。亏损后降低标准、用翻本动机驱动交易，才是问题。

每次开仓只看当前的确定性——上一笔盈亏与这一笔无关。

## 识别信号

以下信号出现说明动机已偏：
- [ ] 买入理由里包含"把刚才亏的赚回来"
- [ ] 仓位明显大于平常
- [ ] 标的在核心追踪池之外，临时扫描涨幅榜选的
- [ ] 不确定这笔交易的逻辑是什么，只知道"它在涨"

## 正确做法

1. 亏损后重新评估盘面——有没有确定性买点？
2. 有 → 正常开仓，仓位按确定性匹配
3. 没有 → 不交易
4. 绝不因"刚亏了"而降低选票标准
