# 模式切换决策框架

## 概述

此框架是交易系统的**元决策层**：在[[概念/趋势主升参与模式]]和[[模式/绕异动临界点参与规则]]之间，决定"何时用哪种模式"。

**核心命题**：当市场出现[[概念/龙头切换识别|主线切换]]时，必须执行完整的模式切换——包括清仓旧模式仓位、按新模式纪律重新建仓。

## 1. 两大模式的本质区别

| 维度 | 趋势主升模式 | 绕异动博弈模式 |
|---|---|---|
| **核心逻辑** | 跟随机构趋势资金，赚取主升浪收益 | 博弈游资连板情绪，套利偏离值修复 |
| **适用标的** | 中军大票/趋势容量票 | 连板情绪龙头/补涨标的 |
| **资金性质** | 机构+外资主导 | 游资抱团主导 |
| **参与方式** | 低吸、分批建仓、5日线持有 | 确定日尾盘介入、偏离值窗口博弈 |
| **持仓周期** | 数周至数月 | 1-3日，快进快出 |
| **决定性指标** | [[概念/中军不倒]]、板块梯队完整性 | 高度板断板、资金流出、新题材 |

## 2. 市场环境诊断清单

### 2.1 支持趋势主升的信号

- 大资金进场：中军单日净流入超20亿
- 板块四级梯队成型：[[概念/板块梯队成型]]
- 产业链扩散效应：[[概念/产业链扩散逻辑]]
- 成交量持续：板块整体放量且不缩量
- 中军不倒：沿5日线上行，无放量长阴
- 机构股涨停模式：涨停后调整→再涨，非连板形态

### 2.2 支持绕异动博弈的信号

- 连板高度打开：4板+标的出现
- 游资活跃：龙虎榜游资席位大举买入
- 情绪分化但未退潮：高位断板有修复，低位补涨活跃
- 量化偏离值窗口：标的逼近异动临界
- 低价容量股活跃（注意：这可能是情绪末段信号）

### 2.3 预警信号——应切换模式

**趋势走弱信号**：
- 中军放量滞涨或跌破5日线
- 板块成交量持续萎缩
- 跟风股率先走弱
- 新题材抢夺资金

**趋势走强信号**：
- 连板高度停滞但中大市值票持续新高
- 机构资金大规模进场
- 板块由纯情绪转向业绩验证
- 新主线确立完整四级梯队

## 3. 绕异动退出信号（"气数"信号）

以下4个信号出现，表明应放弃绕异动模式：

1. **连续3日无新高度板产生** → 情绪生态萎缩
2. **断板次日无修复（A杀频现）** → 资金不愿接力
3. **龙虎榜游资净流出连续2日** → 聪明钱撤退
4. **新题材确立趋势梯队** → 资金向趋势迁移

详见 [[概念/绕异动退出信号]]。

## 4. 趋势主升参与流程（五步）

| 步骤 | 内容 | 详参 |
|---|---|---|
| 第一步：主线确认 | 大资金+梯队+扩散+成交量 | [[分析/ai算力基础设施链主线确立]] |
| 第二步：买点选择 | 趋势启动低吸/分歧转一致/5日线加仓 | [[概念/趋势主升参与模式]] |
| 第三步：仓位管理 | 净资产三档，单日上限，防现金归零 | [[策略/净资产三档仓位管理]] |
| 第四步：卖出纪律 | 中军不放量滞涨持有；中军长阴全线减仓 | [[概念/中军不倒]] |
| 第五步：防卖飞 | 不能套用连板止盈纪律 | [[错误/趋势票卖飞综合症]] |

## 5. 模式切换决策流程图

```
市场状态诊断
    │
    ├─ 大资金进场+中军不倒+四级梯队？
    │       YES ──→ 启用【趋势主升模式】
    │
    ├─ 高度板+偏离值窗口+游资活跃？
    │       YES ──→ 启用【绕异动模式】
    │
    ├─ 两者信号模糊？
    │       └─→ 空仓观望
    │
    └─ 旧模式信号衰竭+新模式信号确立？
            └─→ 执行【模式切换】
                  ├─ 清仓旧模式仓位
                  └─ 按新模式纪律重新建仓
```

## 6. 绝对禁止项

| 禁止行为 | 后果 |
|---|---|
| 趋势票用打板思维追高 | 卖飞踏空 |
| 绕异动票做趋势持有 | T+1锁仓亏损 |
| 两种模式混用 | 思维混乱，进退失据 |

**模式错配是元错误** — 几乎所有已识别错误（卖飞、锁仓、踏空）都可追溯到模式混用。参见 [[概念/模式错配]]。

## 7. 待验证

- 此框架从未被实战检验。建议在每日开盘前用5分钟跑一遍诊断清单，收盘后与实际情况对比。
- "两者信号模糊→空仓"的执行难度最高，需收集执行数据。
