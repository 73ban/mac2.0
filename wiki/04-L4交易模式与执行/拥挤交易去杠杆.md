# 拥挤交易去杠杆

## 定义

**拥挤交易去杠杆**是指当某一资产/板块/主题经历了大幅上涨后，投资者仓位高度集中，一旦核心叙事出现裂痕，所有人同时争相退出，导致跌幅远超基本面恶化程度，形成暴力去杠杆（踩踏式出逃）。

## 核心特征

| 特征 | 描述 |
|------|------|
| 前期涨幅巨大 | 通常YTD或近几个月涨幅超过30-50% |
| 仓位高度集中 | 多头方向过于拥挤，持仓结构同质化严重 |
| 叙事极度乐观 | 共识认为“这次不一样”，忽视尾部风险 |
| 上涨斜率陡峭 | 加速赶顶阶段，波动率被压缩 |
| 退出通道狭窄 | 当反转来临时，流动性迅速枯竭 |

## 典型案例

### 宏观案例：2026年6月费城半导体
- 半导体指数YTD涨40%+
- 科技板块占标普总市值39%+，创历史新高
- 9周连涨创2023年12月以来最长纪录
- 博通指引miss触发去杠杆：费半单日-10.26%，芯片板块蒸发1.2万亿

### 微观案例（A股）
- 视觉中国的[[正反馈闭环-1]]→锚定回追→剧烈反转
- 有研新材等题材龙头的退潮周期

## 识别信号（拥挤度预警清单）

- [ ] 板块/个股连续多周跑赢大盘
- [ ] 媒体/研报一致性看多，几乎无分歧声音
- [ ] 估值指标（PE/PS/PB）处于历史极高分位
- [ ] 板块轮动信号出现（资金开始向其他方向扩散）
- [ ] 核心龙头股对利好消息反应钝化（price in过度）

## 去杠杆触发后的特征

1. **跌幅非线性**：不是技术性回调的5-8%，而是15-30%的暴力压缩
2. **不分好坏**：优质标的与垃圾标的同跌，相关性急剧上升
3. **流动性消失**：买盘撤单，卖盘踩踏
4. **修复缓慢**：需要时间消化套牢盘和重建信心

## 与退潮日的关系

拥挤交易去杠杆是[[退潮日-1]]的核心驱动机制之一。区别在于：

- **退潮日**：侧重现象描述（连板高度压缩、涨停数量骤降、主线题材大面积跌停）
- **拥挤交易去杠杆**：侧重内在机制解释（为什么涨多了就会跌这么多、为什么跌起来不分好坏）

## 应对

1. **事前预防**：动态评估持仓方向的拥挤度，在极端拥挤时主动降低仓位
2. **事中应对**：去杠杆开始时不为“便宜”所诱惑，不抄底——等波动率收敛、等修复结构出现
3. **事后复盘**：记录每次去杠杆的信号特征，建立个人预警清单
