# 主线匹配优先

## 定义
**主线匹配优先**是L2方向选择层的一条强制执行规则：任何开仓决策必须先确认该标的属于当日市场的最强主线。非主线标的，无论个人对其有多深的历史锚定或正反馈记忆，一律放弃。

## 来源

### 12/18 市场全景 vs. 用户操作的对照

| 维度 | 市场全景（此复盘） | 用户操作（同日诊断） |
|------|-------------------|---------------------|
| 最强主线 | 首发经济 + 芯片 | — |
| AI应用端状态 | 转弱，资金流出至硬件 | — |
| 视觉中国位置 | **未出现在任何主线/支线** | 高位追入31.13 |
| 结论 | 视觉中国=非主线轮空票 | 用户买入=锚定回追 |

**12/18 完美揭示了"市场好≠我赚钱"的根本机制**：
- 市场：105涨停、7.5:1涨停跌停比、双主线并行 → 强修复
- 用户：追入非主线票 → 亏损
- 断裂点：选票与主线的匹配度为零

## 规则

### 硬性要求
```
开仓前必须回答：
1. 这只票属于今日第几主线？
2. 这条主线今天是否仍保持强度？
3. 这只票在该主线中的角色是什么（龙头/中军/跟风/淘汰）？
4. 是否有更强的替代选择？

如果任一答案为「否」或「不确定」→ 不开仓
```

### 非主线轮空的识别
- 该板块/概念未出现在当日涨停榜前3
- 该板块龙头今日无表现或已炸板
- 资金从该板块明显流出（参考板块净流出排名）
- 该板块的涨停股数与封单量显著低于主线

### 禁止行为
| 行为 | 危险机制 |
|------|----------|
| 因历史记忆买入非主线票 | [[概念/锚定效应]] — 被过去盈利绑住 |
| 因预期"轮动会来"提前埋伏 | [[概念/预期先行业绩滞后]] — 预期≠现实 |
| 因其他票买不到而退而求其次 | 退而求其次=主动选择低确定性 |
| 在退潮板块"抄底" | 缩圈期的跌停是真淘汰，不是机会 |

## 与其他规则的配合

### L1→L2→L3 嵌套
```
L1：确认市场阶段（强修复期 → 可积极参与）
  ↓
L2：锁定最强主线（首发经济+芯片 → 资金方向）
  ↓
L3：主线内选最强票（友阿股份/益民集团 → 逻辑最硬）
  ↓
L4：执行买点+仓位纪律
```

如果L2不执行「主线匹配优先」，L3和L4的整个链条都会偏离——这正是12/18用户买视觉中国的问题链。

## 执行检查清单（每日开盘前）

- [ ] 如不在，我是否有"锚定效应"或"退而求其次"的心理动机？
- [ ] 是否愿意放弃非主线票，转而等待主线内的分歧低吸机会？
