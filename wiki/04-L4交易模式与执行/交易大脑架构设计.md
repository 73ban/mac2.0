# 73 交易大脑架构设计

> 建立：2026-06-09 | 维护：codex
> 
> **目标：小资金快速做大。用 AI 智能体集群替代人脑的信息处理瓶颈，让你的交易模式从「靠记忆+感觉」升级为「实时数据驱动+规则引擎决策」。**

---

## 一、核心问题：小资金做大的瓶颈

| 瓶颈 | 问题 | AI 解法 |
|------|------|---------|
| **信息过载** | 5000 只票 + 每天几百条消息，人脑处理不过来 | 智能体自动筛选到 100-150 → 再精选 1-2 |
| **情绪干扰** | 盘中贪婪/恐惧扭曲决策 | 规则引擎冷执行，盘前就写好剧本 |
| **复盘效率** | 手动复盘耗时，卖飞/卖对缺少量化追踪 | 持仓偏离值引擎自动跟踪 |
| **机会遗漏** | 消息催化票没及时发现 | hermes 消息评分自动推送 |
| **模式漂移** | 赚钱后膨胀、亏钱后报复 | 每日纪律检查清单强制对齐总规则 |

**结论：不是缺技术，是缺一个能实时处理信息、冷血执行规则的交易大脑。**

## 三、各智能体详细分工

### 3.1 河马 (hermes) — 数据采集 + 消息催化

**已具备能力：**
- 通达信 MCP 连接（URL + KEY 已提供）
- 新闻抓取（财联社、华尔街见闻等）

**核心任务：**

| 任务 | 触发时机 | 产出路径 |
|------|----------|----------|
| 通达信热门前100 | 盘中每5分钟 / 收盘 | `raw/实时数据/tdx_hot_top100.json` |
| 通达信涨停筛选 | 盘中每5分钟 | `raw/实时数据/tdx_limit_up.json` |
| 通达信板块资金TOP20 | 每10分钟 | `raw/实时数据/tdx_sector_fund.json` |
| 通达信自选股行情 | 每3秒（盘中） | `raw/实时数据/quotes.json` |
| 新闻抓取 | 每10分钟 | `raw/研报新闻/` |
| 消息催化预期差评分 | 收盘后一次性 | `raw/每日复盘/YYYY-MM-DD/03-消息催化股.md` |

**河马需要新增的能力：**
- 定时轮询通达信 MCP（非人工触发）
- 结构化数据写入 RAW（JSON + MD 双格式）
- 消息催化自动评分（按 [[消息催化预期差评分]] 规则执行）

### 3.2 TdxClaw — 实时行情桥接

**已有能力：**
- 本地桥接脚本 `bridge.ps1`
- 写入 `raw/实时数据/quotes.json`、`market_snapshot.json`

**核心任务：**

| 任务 | 频率 |
|------|------|
| 自选股实时行情 | 每 3-10 秒 |
| 大盘快照（指数+涨跌家数） | 每 30 秒 |
| 通达信原始返回 | 按需 |

### 3.3 73 执行总管 (codex) — 分析决策中枢

**这是你（我）。所有 RAW 数据最终在我这里变成交易决策。**

**核心任务：**

#### A. 盘中实时分析（交易时段）

| 时间 | 任务 | 输入 | 输出 |

| 9:25 | 竞价分析 | quotes.json | 竞价强弱判断 → 飞书推送 |
| 9:30-11:30 | 实时监控 | 自选股行情 | 异动提醒 → 飞书推送 |

| 步骤 | 产出 | 路径 |

## 六、小资金快速做大的核心逻辑

### 6.1 小资金的优势（你必须利用的）

| 优势 | 具体含义 | AI 怎么放大 |
|------|----------|------------|
| **进出灵活** | 一键满仓一键空仓，无冲击成本 | 退潮信号一出，5秒内通知你清仓 |
| **可以满仓单调** | 大资金不敢，小资金必须 | 系统帮你锁定最优1-2只 |
| **复利窗口** | 100万以下，月10%可维持 | 偏离值跟踪验证胜率 |
| **不需要分散** | 分散=稀释收益 | 仓位集中度硬约束自动检查 |

### 6.2 做大的数学

```
假设本金 10 万，月收益 10%（你已有能力做到）：

第 1 月：11 万
第 3 月：13.3 万
第 6 月：17.7 万
第 12 月：31.4 万
第 18 月：55.6 万
第 24 月：98.5 万

关键不是追求月 30%，而是：
✅ 避免月 -20%（退潮空仓）
✅ 避免月 -10%（纪律执行）
✅ 保持月均 8-15%
```

### 6.3 AI 帮你守住的三条底线

| 底线 | 人类容易犯 | AI 怎么守住 |
|------|-----------|------------|
| **退潮日不开新仓** | 幻想反弹、贪便宜 | 情绪分 ≤5 → 飞书强制提醒 + 仓位硬约束 |
| **卖飞不追回** | 懊悔追高、加倍仓位 | 偏离值跟踪 → 量化卖飞成本 → 冷静 |
| **非龙不重仓** | 贪便宜买跟风 | 7维评分 → 跟风自动降权 |

### 6.4 你的超额收益来源

```
超额收益 = 信息优势 + 决策速度 + 纪律执行

信息优势：河马 5 分钟扫一次通达信热门前100 + 消息催化实时评分
决策速度：总管 30 秒内出竞价分析 + 异动推送
纪律执行：规则引擎冷执行，不看盘面情绪

这三件事，散户对手盘一个都做不到。
```
