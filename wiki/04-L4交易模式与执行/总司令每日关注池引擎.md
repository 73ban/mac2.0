## 二、对核心池的7维深度分析

| # | 维度 | 权重 | 你关心什么 | 系统怎么算 | 数据来源 |
|---|------|------|-----------|-----------|----------|
| ① | **大盘环境** | 底线判断 | 今天能不能做 | 市场情绪指数0-10分 | [[市场环境评分框架]] |
| ② | **热门板块归属** | 20% | 是否属于今日主线 | 映射到板块资金TOP10；冷门板块降权 | 板块资金流向 |

## 三、与你交易模式的映射

```
市场环境 ≥6分 → 可以做
       ↓
热门板块确认 → 缩小到2-3个主线板块
       ↓
个股7维排序 → 三类候选：
       │
       ├─ A类：龙头（连板天梯前3 + 主线板块）
       │   适用：龙头战法、断板低吸、绕异动
       │
       ├─ B类：强势低吸（高位横盘/二波/该弱不弱 + 趋势新高）
       │   适用：趋势低吸、重仓博弈修复
       │
       └─ C类：消息催化（预期差评分≥4 + 热门板块）
           适用：事件驱动、明牌博弈
       ↓
每天锁定 1-2只 → 最强 + 盈亏比最优
```

### 两种行情两种打法

| 行情 | 特征 | 打法 |
|------|------|------|
| **有持续性** | 主线清晰、龙头不断板、容错率高 | A类龙头战法，直接打最强 |
| **无持续性** | 轮动/一日游、无明确主线 | B类强势低吸，博弈次日 |

## 七、关注池核心筛选逻辑（伪代码）

```python
def generate_focus_pool():
    # 1. 取通达信热门前100
    hot_stocks = tdx.get_hot_stocks(top_n=100)
    
    # 2. 硬过滤
    hot_stocks = [s for s in hot_stocks if not s.is_st]  # 去ST
    hot_stocks = [s for s in hot_stocks if s.price > 2]   # 去低价
    hot_stocks = [s for s in hot_stocks if s.volume > 5000_0000]  # 去流动性差
    
    # 3. 7维打分
    for stock in hot_stocks:
        score = (
            ①_market_env() * 0.00 +    # 底线，不参与加权
            ②_sector_strength(stock) * 0.20 +
            ③_volume_rank(stock) * 0.20 +
            ④_trend_strength(stock) * 0.15 +
            ⑤_ladder_position(stock) * 0.15 +
            ⑥_deviation_warning(stock) * 0.15 +
            ⑦_news_catalyst(stock) * 0.15
        )
        stock.total_score = score
    
    # 4. 排序输出
    hot_stocks.sort(key=lambda s: s.total_score, reverse=True)
    return hot_stocks[:150]
```
