# 炒新不炒旧

## 定义

退潮发生后，**全面切割上一主线方向的所有仓位**，将资金（或注意力）转向新主线的萌芽方向。核心原则是：**旧主线退潮后不补仓、不抄底、不幻想二次行情。**

## 核心原理

1. **退潮的主线无法在短时间内重生** — 龙头断板 + 后排负反馈 = 板块周期结束
2. **资金有记忆** — 在旧方向上亏钱的人，短期内不会回来接盘
3. **新方向才有预期差** — 旧方向的利好已经兑现/兑现失败，只有新方向才有未被消化的信息
4. **持有旧方向 = 时间成本浪费** — 即使不跌，也占用了子弹和时间

## 实战案例：2026-06-01

交易者在退潮中执行了完整的"炒新不炒旧"：

| 操作 | 方向 | 状态 |
|------|------|------|
| 清仓鹏鼎控股 | PCB（半导体相关） | ✅ 切掉 |
| 清仓同有科技 | 存储（科技） | ✅ 切掉 |
| 清仓东百集团 | 新零售后排 | ✅ 切掉 |
| 清仓华能蒙电 | 电力后排 | ✅ 切掉 |
| 清仓金螳螂 | 城市更新（涨停卖出） | ✅ 切掉 |
| 买入粤电力A | 电力龙头（但方向已在退潮）| ⚠️ 违反原则 |
| 买入豫能控股 | 电力跟风后排 | ⚠️ 违反原则 |

**教训**：虽然切掉了PCB/存储/新零售后排，但仍在电力方向开新仓——而电力恰恰是正在退潮的旧主线。完全遵守"炒新不炒旧"应该是不碰电力任何标的。

## 操作规则

1. **旧主线龙头断板 → 该板块所有持仓2日内清零**
2. **不抄底旧主线**：即使跌了20%，也不在退潮期接回
3. **新方向验证标准**：
   - 有新的催化事件（政策、收购、IPO等）
   - 有板块效应（至少2个涨停，梯队至少3层）
   - 不是旧主线的分支或补涨
4. **如果新方向未确认 → 空仓等待**

## 常见陷阱

- "龙头断板但我的持仓是龙二，应该还能补涨" — 错误。退潮时龙二比龙头跌得更快
- "已经跌了这么多，再跌空间有限" — 锚定效应。退潮期的底不是你能预测的
- "新方向还没确认，先持有旧方向观望" — 观望可以用空仓，没必要承担旧方向的持续风险
