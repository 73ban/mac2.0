## 自动切换逻辑

```
调用TDX MCP失败
  → 先自查：是否使用了正确的KEY？
  → 切龙虾的KEY重试
  → 仍失败 → 切河马的KEY重试
  → 三把全挂 → 报告用户 + 降级为本地文件分析
```

## 降级模式

当三把KEY全挂时：
- 不阻塞分析任务
- 用本地RAW/WIKI文件 + 用户提供的截图/数据继续
- 明确告知用户"当前为离线模式"
