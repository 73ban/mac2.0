# 一字定方向模式

更新时间：2026-06-13

## 模式定位

“一字定方向”属于竞价模式/题材确认模式，不是系统配置。

它的含义是：如果某个核心票或多个同题材前排在 9:15 被大单顶一字，且买不到，场外资金可能沿着同题材、同属性、同逻辑去挖掘可买的补涨、换手龙、容量中军，从而带动板块集体走强。

这个模式只能作为作战室确认工具，不能单独作为买入理由。

## 模式逻辑

```text
核心票大单一字 -> 资金买不到 -> 挖掘同题材可买票 -> 板块扩散 -> 形成日内方向
```

但它成立有前提：一字必须是真强，不是假封单；题材必须有持续性，不是单日小作文；板块必须有梯队，不是孤立个股。

## 适用市场

| 市场状态 | 是否适用 |
|---|---|
| 题材主升行情 | 最适用 |
| 连板接力行情 | 适用 |
| 趋势抱团行情 | 只用于确认方向，不追后排 |
| 题材轮动行情 | 谨慎使用 |
| 混沌退潮行情 | 不适用 |

## 成立前提

| 条件 | 要求 |
|---|---|
| 题材级别 | 必须是大题材、强催化、主线候选，不能是冷门孤立消息 |
| 一字质量 | 9:15 大单封住，9:20 不撤，9:25 继续加强 |
| 核心地位 | 一字票必须是题材核心、空间票、重组核心、消息最正宗个股 |
| 板块扩散 | 同题材中军、补涨、创业板弹性票、低位票有同步高开或加强 |
| 情绪环境 | 市场不是明显退潮，跌停和核按钮没有扩散 |
| 人气共振 | 热门榜、连板天梯、作战室候选池中能看到同题材共振 |
| 风险锚 | 高标、昨日负反馈、题材风险票没有明显崩溃 |

## 三段式验证

| 时间 | 看什么 | 判断 |
|---|---|---|
| 9:15 | 哪些票大单一字，集中在哪个题材 | 只能初判方向，不能直接下结论 |
| 9:20 | 9:15 的一字是否撤单，板块是否继续强 | 过滤假一字、假方向 |
| 9:25 | 一字是否继续封死，板块是否扩散，作战室票是否确认 | 决定计划继续、暂停、降仓 |

## 可交易对象

一字票本身通常买不到，作战室重点不是追一字，而是找：

1. 同题材换手核心。
2. 容量中军。
3. 低位补涨。
4. 20cm 弹性票。
5. 题材内最先确认的半路票。
6. 分歧后回封的换手龙。

## 买入方式

| 买入方式 | 条件 |
|---|---|
| 半路 | 一字方向成立，板块扩散，目标股竞价后主动走强 |
| 打板 | 换手核心分歧后回封，封单质量好 |
| 低吸 | 龙头或中军首次良性分歧，题材没有退潮 |
| 不买 | 一字孤立、9:20 撤单、板块无扩散、风险锚恶化 |

## 作战室动作

| 结果 | 动作 |
|---|---|
| 真一字 + 板块扩散 + 题材主升 | 提高该题材权重，寻找作战室内可买核心 |
| 真一字但无板块扩散 | 只观察一字票，不追后排 |
| 9:15一字但9:20撤单 | 判定假强，作战室计划降级或暂停 |

## 失效条件

1. 9:15 一字，9:20 明显撤单。
2. 一字票孤立，板块没有扩散。
3. 中军不跟，只有小票乱冲。
4. 高标 A 杀，跌停扩散。
5. 一字来自小作文或低可信来源。
6. 9:25 题材没有形成梯队。

## 禁止

1. 禁止只凭 9:15 一字直接追后排。
2. 禁止把孤立一字当题材主线。
3. 禁止忽略 9:20 撤单过滤。
4. 禁止在混沌退潮、高标 A 杀、跌停扩散时重仓跟随一字方向。
5. 禁止把小作文刺激的一字当 S1/S2 级催化。

## 一句话规则

```text
9:15 一字只能提示方向，9:20 不撤才有可信度，9:25 板块扩散才有作战价值。
```
