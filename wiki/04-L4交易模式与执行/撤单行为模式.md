# 撤单行为模式

## 定义
撤单行为模式是指通过在短时间内反复挂单、撤单、改价，暴露决策不坚定、价格锚定或试图"微调改善成交价"的交易行为集群。

## 两种撤单类型

### 类型一：策略性撤单（正常）
- **目的**：市场走势与预期不符，主动撤销原计划
- **特征**：有计划、有替代方案、撤单后明确下一步行动
- **撤单率**：通常<10%
- **示例**：挂8.00买入，但开盘即涨停→撤单，放弃当日交易

### 类型二：犹豫性撤单（异常）⚠️
- **目的**：试图在价格上"再争取一下"，而不接受当前事实
- **特征**：无替代方案、撤单后重新挂价、撤单率异常高
- **撤单率**：>30%即为警告信号，>50%为严重异常
- **示例**：[[股票/韶能股份]]5/15的57%撤单率

## 犹豫性撤单的诊断清单

| 诊断维度 | 信号 | 韶能案例表现 |
|----------|------|-------------|
| 撤单率 | >30%异常，>50%严重 | 57% |
| 时间间距 | 短时间密集撤单挂单 | 11:13→11:21→11:28，8分钟3笔 |
| 价格轨迹 | 撤单后重新挂的价格是否"追着市场跑" | 9.05→追至7.93→8.49→8.39→8.38 |
| 锚点来源 | 被撤的挂单价格是否有明确依据 | 9.05锚点来源不明 |
| 已发生事实 | 是否忽视已发生的成交 | 11:21已成交7.93，11:28仍挂9.05 |

## 撤单率作为纪律指标

```
撤单率 = 撤单笔数 / 总委托笔数

正常（策略性操作）：0-15%
轻度异常（需自检）：15-30%
中度异常（警告信号）：30-50%
严重异常（纪律崩溃）：>50%
```

## 系统性意义
撤单行为模式为诊断"知道该做但做不到"提供了**可量化的微观证据**。它与以下概念形成诊断链：
- [[概念/锚定效应]] → 解释了为何撤单（锚定高价，拒绝接受当前市价）
- [[概念/追低卖出]] → 解释了撤单后的结果（不断追低成交）
- [[概念/恐惧驱动的纪律执行]] → 撤单循环 = 恐惧驱动了行动，但未转化为有效纪律

## 在复盘中的应用
每日复盘时，检查委托明细的撤单率：
1. 撤单率<15%且无追低行为 → 纪律执行正常
2. 撤单率>30%且有追低行为 → 需要写"撤单行为自检"：为什么撤？锚点是什么？
3. 撤单率>50% → 触发纪律回顾，重新确认[[策略/退潮票早盘清仓]]等核心规则
