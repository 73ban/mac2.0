# 高位加速票规避规则

## 规则定义

**放弃任何当日已3连板（含）以上的标的**。高位接力是退潮期回撤的最大来源，加速过的票已不属于交易者的利润区间。

## 量化标准

| 市场 | 连板阈值 | 涨幅阈值 |
|------|---------|---------|
| 主板（10%） | 3连板 | 30%+ |
| 科创板/创业板（20%） | 2连板 | 40%+ |
| 北交所（30%） | 2连板 | 60%+ |

**北交所特殊规则**：30%涨跌幅下，2连板即达60%+涨幅，风险乘数极高。北交所的高位加速票在退潮期应列入**"禁止交易"清单**。

## 例外情况

**仅在大盘主升段可以有条件参与**：

| 条件 | 要求 |
|------|------|
| L1 环境 | 主升期确认（情绪 ≥ 8 且上升） |
| 主线确认 | 标的是 L2 确认的最强主线核心票 |
| 仓位限制 | 单票 ≤ 总仓位 10% |
| 止损 | 买入当日设置 -5% 硬止损 |

**在退潮期/修复期/震荡期：无例外，全部放弃。**

## 为什么加速票危险

1. **盈亏比已严重失衡**：3连板后向上的空间被压缩，向下的空间被放大
2. **流动性风险**：连板票的接盘资金多为情绪驱动，一旦转势流动性急剧萎缩
3. **[[概念/局部行情幻觉]]** 放大器：加速段制造"机会还在加速"的幻象，但实际是风险在加速
4. **卖飞焦虑的完美陷阱**：因为"错过了前面几个板"，所以"必须抓住这个"

## 典型案例

### 2024-10-22 天马新材
- 北交所3连板票，追高买入 @65.20
- 被局部行情幻觉+卖飞焦虑驱动
- 结果：追在加速段顶部区域

### 2024-10-16 中粮资本
- 主板连板票，下跌中补仓
- 同样的高位接力→补仓→扩大亏损模式

## 与 L4 执行控制层的绑定

本规则属于 [[04-L4交易模式与执行/L4执行控制层/每日5问执行检查清单]] 的买点筛选规则：

```
L4 买点检查清单：
□ 是否为3连板（含）以上标的？
  → 是：检查是否满足例外条件
  → 否：继续其他检查
□ 如满足例外条件，仓位是否 ≤ 10%？
□ 如不满足例外条件 → 直接拒绝
```

## 执行方式

- **物理屏蔽**：在交易软件中将3连板以上标的加入黑名单/预警过滤
- **自选股清理**：每日开盘前清理自选股，移除所有3连板以上标的
- **与熔断联动**：一级熔断触发后，本规则自动升级为"所有连板票一律禁止"
