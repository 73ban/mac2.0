# 20万小资金模式权限矩阵

更新时间：2026-07-05

## 定位

本页约束 Codex 的交易建议权限。任何买卖建议必须先经过模式权限判断，不能只因为题材逻辑、名人堂观点、淘股吧热度或三榜共振就给买入结论。

机器可读版本：

```text
.system/codex-mode-permission-matrix.json
```

## 总原则

- 持仓处理优先于新开仓。
- 退潮/恐慌阶段默认防守。
- A类模式 + 主线确认 + 个股核心 + 竞价确认 + 明确退出条件，才允许提高仓位。
- 资料、名人堂、淘股吧热度、三榜共振只能加权，不能直接授予买入权限。

## 权限等级

| 等级 | 仓位权限 | 最低条件 |
|---|---|---|
| 主计划 | 可按当日风险预算执行，是否重仓仍需用户确认 | L1不退潮、L2主线前排、L3个股核心、L4模式A级、竞价确认、退出条件明确 |
| 备选 | 小仓试错或等待触发，不允许先手重仓 | 题材有效、个股有辨识度、买点清楚、风险可控 |
| 观察 | 不得买入，只观察是否满足升级条件 | 信息有价值但盘面或权限不足 |
| 禁止 | 不得交易 | 退潮高位、后排补涨、非计划交易、卖飞补偿、持仓未确认时扩大风险 |

## 模式权限

| 模式 | 默认权限 | 升级条件 | 禁止场景 |
|---|---|---|---|
| 主线核心分歧低吸 | 备选 | 主线仍在前排；核心未破位；分歧后承接强；三榜或涨停全景有资金确认 | 退潮初期；个股已从核心变后排；踏空补偿追入 |
| 弱转强/反包 | 观察 | 前一日强分歧；次日竞价超预期；板块同步修复；量能和承接确认 | 退潮期高位反抽；后排借情绪修复骗炮 |
| 首板/新题材预期差 | 观察 | 首板原因硬；板块有扩散；次日竞价确认；可写出止损 | 单纯消息热；无涨停全景确认；追高20cm后排 |
| 连板接力 | 观察 | 市场情绪允许；梯队完整；空间板未负反馈；换手充分且承接强 | 退潮期高位加速；连续一字开板无承接；临盘追高 |
| 容量中军趋势 | 备选 | 成交额Top100；板块主线确认；趋势结构未坏；回踩承接而非追高 | 把大成交当安全；趋势破位后补仓 |
| 卖飞补偿追高 | 禁止 | 无 | 全部场景禁止，除非重新进入作战室并满足独立触发条件 |
| 情绪补仓/摊低成本 | 禁止 | 无 | 全部场景禁止，只能按原计划止损、降仓或等待确认 |

## 数据加权

| 数据 | 用法 |
|---|---|
| 三榜共振 | 提高关注等级，不直接提高仓位权限 |
| 淘股吧独热 | 标记预期差或噪音，必须看盘面确认 |
| 通达信独热 | 优先看资金盘口，等待题材解释 |
| 同花顺独热 | 大众热度，警惕滞后和一致性风险 |
| 名人堂观点 | 长期原则可入学习层；短期个股观点必须D+验证 |

## Codex回答约束

回答任何交易问题时，必须先给出：

```text
当前权限：主计划/备选/观察/禁止
权限理由：
升级条件：
降级条件：
退出条件：
D+验证是否支持：
```

没有权限，不给买入建议。
