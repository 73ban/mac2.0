# 淘股吧数据采集与使用规则

更新时间：2026-07-05

## 定位

淘股吧是短线交易、游资文化、实盘赛、高手复盘的重要信息源。

它不能直接替代作战室决策，但可以作为：

1. 短线情绪观察源
2. 游资模式学习源
3. 实盘高手行为样本源
4. 人气热榜和题材扩散验证源
5. 交易纪律和错误案例来源

补充：

淘股吧日更内容是大鸟外部学习里进化最快的素材之一，尤其适合训练“高手如何实战处理分歧、如何犯错、如何修正”的能力。

## 默认可信度

淘股吧内容默认 `truth_grade=S3`。

除非来自可核验的实盘赛公开数据、官方比赛页、完整交割单截图或可复核链接，才可上调。

## 实盘赛重点样本

千万组前排和高人气选手单独建样本池：

```text
.system/taoguba-contest-watchlist.json
```

规则见 [[淘股吧实盘赛样本学习规则]]。

处理口径：

1. 优先看同一比赛的千万组前10名；
2. 优先看比赛前排且公开交割单、当日分享完整的选手；
3. 每天提取持有、买入、卖出、今日盈亏、总收益、题材标签；
4. 连续跟踪同一选手的行为轨迹，而不是只看单日；
5. 提炼出来的模式必须和用户交易记录对照，帮助用户给自己的买票思路命名和补全；
6. 不得用全组排名、博客页排名或人气排名替代千万组组内排名。

## 使用原则

1. 不直接照抄淘股吧观点。
2. 只把它当作观察、验证、学习样本。
3. 必须与盘面数据、作战室、真实交割单交叉验证。
4. 没有证据链的内容不能进入主计划。

## 大鸟处理口径

大鸟看到淘股吧内容时，按四层处理：

1. 是不是事实
2. 能不能验证
3. 对今天或明天交易有没有用
4. 是否值得沉淀成模式或错误样本

## 淘股吧内容怎么用

淘股吧不是用来“抄作业”，而是用来训练三件事：

1. **看情绪**：热门讨论100、热帖和神评，判断散户和短线资金当天在兴奋、恐慌、分歧还是转一致。
2. **学高手处理**：重点作者的实盘/复盘，拆他们在不同周期里如何开仓、加仓、止损、空仓。
3. **做验证**：把作者观点和当日涨停全景、连板天梯、板块强度、龙虎榜、成交额Top100做对照，D+1/D+3/D+5验证有效性。

每篇重点文章必须输出：

- 市场阶段判断
- 题材/主线判断
- 提到的股票和概念
- 具体买卖条件
- 仓位态度
- 风险条件
- 次日验证点
- 后视镜风险

验证有效后才允许进入：

```text
wiki/04-L4交易模式/
wiki/05-错误库/
wiki/09-统计与进化/
```

## 日更学习要求

1. 每个交易日都要优先读取新增的高手复盘、热帖、实盘赛样本。
2. 不允许只看观点，必须拆成：
   盘面判断
   出手时机
   持仓处理
   错误暴露
   次日预期
3. 要重点盯“连续几天都在出现”的题材、模式和错误。
4. 经过筛选后，把内容分成：
   `现在有用`
   `未来两周可能有用`
   `当前噪音`

## 大鸟重点反复学习什么

1. 高手复盘里对市场阶段和龙头地位的判断
2. 实盘赛选手真实买卖节奏和仓位变化
3. 热帖里题材扩散、预期差发酵、情绪拐点的早期信号
4. 交割单截图里可复核的买点、卖点、错误和修正
