# 每日交易工作流总控

更新时间：2026-06-28

## 统一脚本

以后日常优先用这一条脚本：

```bash
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase premarket
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase postmarket
```

## 阶段

| 阶段 | 命令 | 用途 |
|---|---|---|
| 盘前 | `--phase premarket` | 生成/检查作战总控、AI上下文包、D+任务，并更新 current-ai-context |
| 只切上下文 | `--phase context` | 只刷新 `.system/current-ai-context.json` 和配置指针 |
| 盘后 | `--phase postmarket` | 生成/检查盘后复盘 RAW、统计训练回写、卖出后验证 |
| 全流程 | `--phase full` | 顺序执行盘前和盘后；只适合输入文件都已经存在时使用 |

## 每日推荐流程

### 盘前

```bash
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase premarket
```

然后打开：

- `wiki/07-作战室/YYYY-MM-DD-作战总控.md`
- `wiki/07-作战室/YYYY-MM-DD-AI上下文包.md`
- `raw/03-每日计划/YYYY-MM-DD-竞价监控清单.md`

### 盘中

只按作战总控、竞价监控清单和当前作战室执行。

禁止：

- 计划外冲动交易。
- 持仓未确认时扩大风险。
- 资料逻辑替代盘面。

### 盘后

```bash
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase postmarket
```

然后填写：

- `raw/02-每日复盘/YYYY-MM-DD-盘后复盘RAW.md`
- `wiki/09-统计与进化/YYYY-MM-DD-统计与AI训练回写.md`
- `wiki/09-统计与进化/YYYY-MM-DD-D+验证任务.md`
- `wiki/09-统计与进化/卖出后验证/YYYY-MM-DD-卖出后验证.md`

## 输入前提

盘前阶段必须已有：

- `wiki/07-作战室/YYYY-MM-DD-作战室候选票评分表.md`
- `raw/03-每日计划/YYYY-MM-DD-竞价监控清单.md`

盘后阶段必须已有：

- `wiki/07-作战室/YYYY-MM-DD-作战总控.md`
- `wiki/09-统计与进化/YYYY-MM-DD-D+验证任务.md`

## 安全规则

默认不覆盖已有文件。  
如确需重建，使用：

```bash
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase premarket --force
python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase postmarket --force
```

## 目的

这套工作流只做一件事：让 20万小资金账户每天形成闭环。

```text
盘前计划 -> 盘中执行 -> 盘后复盘 -> D+验证 -> 错误库/模式库 -> 明日计划
```
