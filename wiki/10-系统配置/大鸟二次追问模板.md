# 大鸟二次追问模板

更新时间：2026-06-18

## 用法

当大鸟读取当天 RAW 后，发现还缺关键主观信息，就用这份模板在飞书里继续追问你。

你的补充再继续写回 RAW。

## 标准模板

```text
【需要补充】
日期：____
1. 股票 / 主题：____
   缺什么：____
   为什么要补：____

2. 股票 / 主题：____
   缺什么：____
   为什么要补：____

3. 股票 / 主题：____
   缺什么：____
   为什么要补：____
```

## 简短版

```text
【需要补充】
日期：____
1. 股票 / 主题：____
   请补：____
2. 股票 / 主题：____
   请补：____
3. 股票 / 主题：____
   请补：____
```

## 示例

```text
【需要补充】
日期：2026-06-18
1. 某只交易
   缺什么：卖出原因
   为什么要补：需要判断这笔是纪律卖、看弱卖、换仓卖还是情绪卖

2. 某条消息
   缺什么：你为什么觉得它最重要
   为什么要补：需要判断它是夜间催化，还是普通资讯
```

## 大鸟执行要求

1. 只问明确缺的内容。
2. 不自己补脑。
3. 不替你回答。
4. 你的补充必须追加到：

```text
raw/02-每日复盘/YYYY-MM-DD-口述补充.md
```
