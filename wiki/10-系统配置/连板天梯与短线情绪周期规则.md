# 连板天梯、市场预期与短线情绪周期规则

更新时间：2026-06-18

## 目的

这里的“情绪”不是大鸟自己的情绪，而是市场参与者对第二天交易的乐观、悲观、分歧、恐慌、贪婪和风险偏好。

连板天梯、涨跌停数据、竞价、通达信热榜 Top100、高手复盘、实盘赛仓位、公众号文章、财联社和淘股吧讨论，必须共同服务一个目标：

```text
判断明天市场是否愿意继续进攻，应该用什么模式，允许多大仓位。
```

补充口径：通达信热榜 Top100、首板涨停催化日报、剔除 ST 后的 2板及以上连板天梯是短线选手每日必看结构，复盘正文必须保留全量表或全量附录。天梯只包含 2板及以上，必须包含主板10cm、创业板20cm、科创板20cm、北交所30cm；首板单独写 RAW/JSON，不进入天梯；ST 直接剔除，不写明细、不单列、不补全。不能为了正文变短而丢掉核心市场结构。

## 总原则

1. 情绪不是一句主观看法，而是多源证据合成判断。
2. 文本观点只能代表市场参与者表达，不能直接等于事实。
3. 最终结论必须服务第二天交易，而不是停留在描述层。
4. 没有足够证据时，可以标注 `数据缺口`，但不能假装完整。

## L1 市场环境的组成

L1 市场环境不是单一数据，而是四层合成判断：

| 层级 | 内容 | 作用 |
|---|---|---|
| 盘面硬数据 | 涨跌停、炸板率、连板高度、晋级率、成交额、指数、上涨下跌家数 | 判断真实赚钱效应和亏钱效应 |
| 主线结构 | 题材强度、龙头、中军、补涨、梯队、扩散、分歧修复 | 判断是否有可持续进攻方向 |
| 人类预期 | 高手复盘、实盘赛仓位、公众号观点、淘股吧热帖、人气榜变化 | 判断市场对明天是乐观、悲观还是分歧 |
| 用户适配 | 用户持仓、近期有效模式、亏损模式、重仓条件 | 判断你能不能做、该不该重仓 |

## 市场预期采集

每天盘后到次日盘前，都要采集市场对第二天的预期。

### 公开预期来源

| 来源 | 重点看什么 | 默认可信度 |
|---|---|---|
| 淘股吧高手复盘 | 对明天主线、情绪、龙头、高标、风险的判断 | S3 |
| 淘股吧实盘赛 | 高手仓位变化、买入方向、减仓空仓、风格切换 | S2/S3 |
| 游资公众号 | 高潮、冰点、分歧、修复、退潮、主升、抱团等关键词和模式解释 | S3 |
| 知识星球 / 研报 | 产业逻辑、催化、预期差、机构关注方向 | S2/S3 |
| 人气热榜 | 同花顺、开盘啦、通达信 Top100、东方财富、淘股吧的关注度变化 | S2/S3 |
| 盘面数据 | 涨跌停、连板、竞价、龙虎榜、成交额、板块强弱 | S1/S2 |

### 预期字段

每天作战室必须输出：

```yaml
market_expectation:
  direction: 乐观/谨慎乐观/分歧/谨慎悲观/悲观
  reason:
  strongest_consensus:
  biggest_disagreement:
  possible_trap:
  tomorrow_strategy:
```

### 文本预期关键词

```text
乐观类：主升、继续干、赚钱效应、超预期、抱团、龙头、扩散、加速、修复、强更强
谨慎类：分歧、轮动、兑现、高潮、后排掉队、缩量、冲高回落、接力难
悲观类：退潮、冰点、亏钱效应、核按钮、A杀、补跌、天地板、杀高标、空仓
机会类：恐慌低吸、尾盘抄底、冰点修复、弱转强、反包、出监管、绕异动、并购重组
```

注意：文本关键词只能代表表达，不能直接等于事实。必须和盘面硬数据、热门榜、竞价和后续走势互相验证。

## 市场状态

| 状态 | 特征 | 仓位倾向 | 重点方法 |
|---|---|---|---|
| 题材主升行情 | 大题材持续强化，核心股不断新高，板块有高度也有宽度，分歧后能修复，市场预期继续乐观 | 可重仓，但必须是 A 类模式 | 主线龙头、容量中军、换手核心、补涨前排 |
| 连板接力行情 | 高度板打开，晋级率高，断板反馈不差，情绪票赚钱效应强 | 标准仓到重仓，按情绪强度决定 | 最高标、换手龙、补涨龙、空间板 |
| 趋势抱团行情 | 连板不强，但容量核心趋势持续，资金抱团清晰 | 标准仓，重仓需更高确定性 | 趋势核心、题材中军、机构抱团票 |
| 题材轮动行情 | 热点切换快，持续性差，日内强但隔日不稳，市场预期分歧大 | 轻仓或标准仓 | 日内前排、低位套利、隔日兑现 |
| 冰点修复行情 | 前期亏钱效应释放后，核心票止跌，市场从悲观转向谨慎 | 小仓试错，确认后加仓 | 恐慌低吸、尾盘抄底、核心反包、弱转强 |
| 牛市急跌行情 | 大环境仍强，偶发大跌释放恐慌，核心方向没有被证伪 | 可做逆向机会，但必须等恐慌释放 | 尾盘抄底、核心低吸、次日修复 |
| 混沌退潮行情 | 高标 A 杀，炸板率高，跌停扩散，主线不清晰，高手降仓或空仓 | 空仓或极轻仓 | 风险锚，不主动进攻 |

## 牛市急跌与尾盘抄底规则

牛市或题材主升中的偶发大跌，可能是机会，不应机械归类为退潮。

允许研究尾盘抄底时，必须满足多数条件：

```text
1. 大级别市场或主线仍未被证伪。
2. 跌幅来自恐慌、指数拖累或短期获利盘，而不是核心逻辑崩塌。
3. 核心票没有批量 A 杀，龙头或中军仍有承接。
4. 成交额没有极端萎缩，说明仍有资金愿意交易。
5. 高手复盘和实盘赛行为不是一致空仓，而是出现分歧或有人低吸。
6. 尾盘出现恐慌释放后的承接，而不是一路无承接下跌。
7. 次日有明确修复触发条件和止损条件。
```

禁止：

```text
1. 把退潮期大跌误判成牛市急跌。
2. 在核心票集体 A 杀时盲目抄底。
3. 没有次日退出计划就尾盘买入。
4. 只因为跌得多就认为便宜。
```

## 次日涨停预判与验证

每天盘后必须基于完整 2板及以上连板天梯、首板涨停催化日报、通达信 Top100、涨停原因 6 维度、财联社和公众号，形成次日涨停候选评分表：

```text
wiki/09-统计与进化/YYYY-MM-DD-次日涨停候选评分表.md
```

次日收盘后，必须用实际涨停池、连板晋级、炸板和 Top100 变化验证前一日预判：

```text
wiki/09-统计与进化/YYYY-MM-DD-涨停预判验证与权重修正.md
```

预判失败不是总结一句“弱于预期”，而是要追问：

1. 是消息催化误判，还是题材共振不足。
2. 是热榜注意力太一致，还是没有盘口确认。
3. 是连板梯队退潮，还是个股地位不是核心。
4. 是作战室过滤不足，还是模式权限给高了。
5. 下一次同类信号该加分、降分还是禁止。

## 大鸟判断原则

大鸟不能说“我感受到情绪”，只能说：

```text
根据公开复盘、实盘赛行为、热门榜变化、盘面数据和竞价反馈，市场参与者当前更偏乐观 / 悲观 / 分歧。
```

## 一句话

连板天梯和短线情绪判断，最后都要落到三件事：

1. 明天能不能做
2. 做什么模式
3. 允许多大仓位
