# GPT Pro 持续工作规则

更新时间：2026-07-05

## 定位

GPT Pro 的 token 优先用于让 73神话持续进化，不用于无边界读资料。

## 优先级

1. 交易安全：持仓、仓位、回撤、禁止动作。
2. 明日胜率：作战室、竞价、主线、候选票。
3. D+验证：把预测变成反馈。
4. 错误修正：把亏损和失控动作写成限制。
5. 模式升级：只升级被真实市场验证过的模式。
6. 高手学习：淘股吧千万组、名人堂、大游资只作为样本和规则候选。
7. 资料整理：只保留能服务交易和训练的资料。

## 自动推进口径

当用户说“继续”“你自己推进”“全部执行到位”时，Codex 默认按以下顺序执行：

```text
每日总控驾驶舱
-> D+债务清零
-> 当前作战室新鲜度
-> 持仓证据检查
-> 三榜/淘股吧/公告/消息验证
-> 模式权限修正
-> 错误库和统计回写
-> Wiki资料瘦身
```

## 禁止浪费 token

- 不为“看起来有用”的资料长篇总结，除非能写出验证点。
- 不把名人堂观点直接写成交易规则。
- 不在 D+验证未完成时宣称系统进化。
- 不覆盖用户交易事实。
- 不用历史后视镜倒填盘前计划。
