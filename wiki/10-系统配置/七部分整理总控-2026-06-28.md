```bash

python3 raw/07-系统脚本/codex_daily_workflow.py --date YYYY-MM-DD --phase postmarket
```

```bash

```

该脚本会生成/检查当天作战总控、AI上下文包、D+验证任务，并更新 `.system/current-ai-context.json`。

如果只需要更新 current-ai-context，使用：

```bash
python3 raw/07-系统脚本/codex_update_ai_context.py --date YYYY-MM-DD
```

盘后收口使用：

```bash
python3 raw/07-系统脚本/codex_prepare_postmarket.py --date YYYY-MM-DD
```

前提是当天的 AI 上下文包、作战总控、候选评分表、竞价监控清单、D+验证任务已存在。
