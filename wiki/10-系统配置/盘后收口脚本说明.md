# 盘后收口脚本说明

更新时间：2026-06-28

## 脚本

```bash
python3 raw/07-系统脚本/codex_prepare_postmarket.py --date YYYY-MM-DD
```

示例：

```bash
python3 raw/07-系统脚本/codex_prepare_postmarket.py --date 2026-06-29
```

## 作用

该脚本用于收盘后生成复盘和训练骨架，不自动编写行情结论。

它会读取：

- `wiki/07-作战室/YYYY-MM-DD-作战总控.md`
- `wiki/07-作战室/YYYY-MM-DD-作战室候选票评分表.md`
- `wiki/09-统计与进化/YYYY-MM-DD-D+验证任务.md`

然后生成：

- `raw/02-每日复盘/YYYY-MM-DD-盘后复盘RAW.md`
- `wiki/09-统计与进化/YYYY-MM-DD-统计与AI训练回写.md`
- `wiki/09-统计与进化/卖出后验证/YYYY-MM-DD-卖出后验证.md`

## 安全规则

默认不覆盖已有文件。

如果要重新生成，必须显式加：

```bash
python3 raw/07-系统脚本/codex_prepare_postmarket.py --date YYYY-MM-DD --force
```

## 盘后推荐顺序

```text
导入交割单到 raw/01-交割单
  -> 运行盘后收口脚本
  -> 填写 raw/02-每日复盘/YYYY-MM-DD-盘后复盘RAW.md
  -> 更新 D+验证任务
  -> 更新统计与AI训练回写
  -> 写入错误库/模式库
```

## 关键原则

盘后不是写行情作文，而是训练交易系统。

必须回答：

1. 今天有没有按计划交易。
2. 赚钱/亏钱来自模式、环境、运气还是违规。
3. D+验证有没有证伪自动判断。
4. 哪条规则需要升级或降级。
