# 飞书任务模板

更新时间：2026-06-18

## 总模板

```text
大鸟，按73交易大脑现行制度开工。
任务：____
目标：____
范围：____
重点：____
禁止：____
输入：____
输出：____
```

## 常用短句

### 盘前

```text
大鸟，按73交易大脑现行制度，做明日作战室。重点只留 1-3 只最值得做的票。
```

### 盘中

```text
大鸟，按73交易大脑现行制度，看现在能不能继续做，重点看持仓风险和计划是否失效。
```

### 盘后

```text
大鸟，按73交易大脑现行制度，复盘今天交割单，重点看我是不是计划外交易。
```

```text
大鸟，按73交易大脑现行制度，做完今天复盘后，把复盘全文通过飞书发我审核；我确认后，你再写入 WIKI。
```

### 学习

```text
大鸟，按73交易大脑现行制度，把这篇文章当学习样本处理，只提炼可验证规则，不要直接升级模式。
```

### 历史复盘

```text
大鸟，按73交易大脑现行制度，做 YYYY-MM-DD 的历史复盘。只准使用该日及之前已经存在的数据、消息、截图、交割单、计划和聊天记录，不准开后视镜。需要的话可调用腾讯财经、AKShare、热榜、iWenCai、iFinD补证，但必须标 reviewTradeDate、evidenceTradeDate、evidenceCutoffDate，并把后验内容只放事后验证层。
```

```text
大鸟，按73交易大脑现行制度，做完 YYYY-MM-DD 的历史复盘后，把复盘全文通过飞书发我审核；我确认后，你再写入 WIKI。
```

### 历史复盘 + 交割单

```text
大鸟，按73交易大脑现行制度，先把 YYYY-MM-DD 的交割单、口述、截图落 RAW，再做该日历史复盘。只准用当天及之前信息重建当时事实和当时判断，不准用后面走势反推。重点看：我为什么买、为什么卖、计划是怎么失效的、当时有没有及时纠错、当天龙头为什么能走出来。
```

### 复盘送审固定补句

```text
复盘完成后，先把全文发我飞书审核。未经我明确确认，不准写入 WIKI。
```
