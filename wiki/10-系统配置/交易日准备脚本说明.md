# 交易日准备脚本说明

更新时间：2026-06-28

## 脚本

```bash
python3 raw/07-系统脚本/codex_prepare_trading_day.py --date YYYY-MM-DD
```

示例：

```bash
python3 raw/07-系统脚本/codex_prepare_trading_day.py --date 2026-06-29
```

## 作用

该脚本用于每天盘前准备 AI 和作战室入口。

它会读取：

- `wiki/07-作战室/YYYY-MM-DD-作战室候选票评分表.md`
- `raw/03-每日计划/YYYY-MM-DD-竞价监控清单.md`
- `wiki/09-统计与进化/核心候选个股D+验证队列.md`

然后生成或检查：

- `wiki/07-作战室/YYYY-MM-DD-作战总控.md`
- `wiki/07-作战室/YYYY-MM-DD-AI上下文包.md`
- `wiki/09-统计与进化/YYYY-MM-DD-D+验证任务.md`
- `.system/current-ai-context.json`

并同步更新：

- `73wiki.system.json -> aiContext`
- `73wiki.config.json -> tradingBrain.aiContext`

## 安全规则

默认不覆盖已有文件。

如果当天作战总控、AI上下文包、D+验证任务已经存在，脚本会跳过这些文件，只更新 current-ai-context。

如需覆盖，必须显式加：

```bash
python3 raw/07-系统脚本/codex_prepare_trading_day.py --date YYYY-MM-DD --force
```

## 前提

运行前必须已有：

1. 当天候选票评分表。
2. 当天竞价监控清单。
3. 核心候选个股 D+验证队列。

缺少候选票评分表或竞价监控清单时，脚本会退出，不生成半成品。

## 每日推荐顺序

```text
生成候选票评分表
  -> 生成竞价监控清单
  -> 运行交易日准备脚本
  -> 打开 Wiki 对话框
  -> AI 自动先读 current-ai-context
```
