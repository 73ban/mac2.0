# 交易总规则（2026-05-31 终版）

> **操作系统总纲**：市场情绪决定是否做，仓位管理决定做多少，交易模式决定怎么做，纪律决定能否做到，错误库决定如何纠偏。

---

## 一、交易基石（4条哲学原则）

1. **确定性、盈亏比、稳定盈利三者统一**
   - 三者不是递进的大小关系，而是一个统一概念的不同维度。确定性决定是否做，盈亏比决定怎么做，稳定盈利是二者统一后的自然结果。缺一不可。

2. **仓位与确定性挂钩**
   - 单票仓位由个股确定性决定
   - 总仓位由市场情绪和个股确定性共同决定
   - **🆕 核心修正**：情绪分不是大盘的衍生品——5000多只票，只选1-2只最强标的。最强标的的确定性和盈亏比，本身就是情绪分的核心构成。

3. **交易模式、纪律、执行力三者统一**
   - 模式解决"怎么做"，纪律解决"能否做到"，执行力解决"能否坚持做到"。

4. **预判与盘面统一**
   - 尊重盘面实际走势，不执念。预判是假设，盘面是答案。

## 二、市场情绪量化模型（每日必看）

### 2.1 情绪分数的本质 🆕

情绪0-10分，衡量的是**超短线赚钱效应**。它与大盘涨跌有关，但最主要取决于：

| 构成因子 | 权重 | 说明 |
|:---|:---|:---|
| **短线赚钱效应的强度** | 高 | 涨停家数、连板高度、炸板率、跌停数 |
| **确定性个股的表现** | 高 | 你实际选择的1-2只最强标的的走势与封板质量 |
| 大盘整体走势 | 低 | 仅作为背景，不是情绪分的直接决定因素 |

> **情绪分既是大环境的温度计，也是你选股能力的成绩单。** 当你选到确定性最高、盈亏比最优的标的时，市场情绪往往就是高分。也因此，情绪分直接决定仓位上限——**但只有当你的持股是核心标的时，这个上限才对你有意义。**

### 2.2 0-10分情绪温度计

| 情绪分 | 赚钱效应 | 典型盘面 |
|:---|:---|:---|
| **0分** | 无赚钱效应，千股跌停 | 流动性枯竭，百股跌停，必须空仓 |
| **3-4分** | 显著亏钱效应 | 高位崩塌，连板断板A杀，负反馈连续出现 |
| **5分** | 一日游，随机拉砸 | 各板块随机拉砸，盈亏比随机——**多数亏损行情的温床** |
| **6分** | 轮动行情，容错率尚可 | 板块轮动，有结构性机会，可适度参与 |
| **7-8分** | 明确主线+强赚钱效应 | 主线持续，梯队完整，赚钱效应扩散 |
| **9-10分** | 超级大行情（年3-4次） | 百股涨停，前排大单一字，踏空资金溢出 |

**关键阈值**：
- ≤5分：总仓位≤3成
- ≥6分：可满仓满融（需满足选股质量过滤器，见第三章）
- 退潮日：清仓，且退潮当天买入的票次日无条件清仓

## 三、仓位管理（二层控制体系）🆕

仓位管理由两层联合决定：**客观情绪分（第一层）× 主观选股质量（第二层）**。

### 3.1 第一层：情绪分基准仓位上限（客观输入）

| 情绪分 | 仓位上限（基准） | 说明 |
|:---|:---|:---|
| 0-2 | 空仓 | 无条件空仓，不参与 |
| 3-4 | ≤20% | 退潮期，仅保留最强逻辑的底仓或空仓 |
| 5-6 | ≤50% | 结构性机会，轻仓参与 |
| 7-8 | ≤80% | 主线明确，积极进攻 |
| 9-10 | **满仓满融** | 需满足超级题材认定标准（见3.3） |

### 3.2 第二层：选股质量过滤器（主观输出）🆕

**核心原则**：5000多只票，只选1-2只最强标的。这1-2只票的**确定性和盈亏比**，决定了你**当下真实的赚钱效应**。

仓位上限的最终确定，必须在客观情绪分的基础上，通过以下自我提问：

| 问题 | 如果答案是"是" | 如果答案是"否" |
|:---|:---|:---|
| **我当前持仓（或计划买入）的是不是方向内角度最陡的标的？** | 仓位可参考客观情绪分上限 | **仓位立即降一级**（如基准80%→降为50%，基准50%→降为≤20%） |
| **有没有更陡的角度出现而我未持有？** | 考虑切换，持有期间仓位不扩 | 维持 |
| **我的持仓是否属于资金正在流出的方向？** | **立即降至≤20%或清仓** | — |
| **我是否"选不出"最强标的？** | **按基准仓位降一级，优先解决选股问题** | — |

**🎯 执行口诀**：
> **非龙勿重仓，无确定不加仓。情绪分高≠你的赚钱效应高——只有你手里的票是核心，情绪分才属于你。**

**选股-情绪联动法则**：
```
你手里的票 = 最强龙一 → 情绪分对该笔操作有效性极高 → 按情绪分上限执行
你手里的票 = 次强或跟风 → 情绪分对该笔操作有效性打折 → 仓位上限自动降一档
你手里的票 = 无票/边缘标的 → 即使百股涨停也不构成你满仓的理由 → 先解决选股
```

### 3.3 超级行情例外条款：满仓满融条件

**条件（必须同时满足）**：

| 条件 | 标准 |
|:---|:---|
| 题材级别 | **超级大题材**（央行/政治局/国家级产业政策引爆，板块涨停≥15只） |
| 市场情绪 | **情绪≥9分**，百股涨停，涨跌比≥5:1 |
| 前排状态 | 方向内最强标的**大单一字封死**，封单≥10亿，无开板迹象 |
| 入场时机 | **引爆初始日**（题材第一天发酵或第二天的确认加速日），非行情末尾 |
| 标的类型 | 方向内**角度最陡的标的**（前排龙头、20cm/30cm弹性票），非后排跟风 |

**仓位规则**：

| 阶段 | 仓位上限 | 说明 |
|:---|:---|:---|
| 日常行情 | 按情绪分+选股质量联合决定 | 见3.1+3.2 |
| **满足上述全部条件时** | **满仓满融** | 超级赚钱效应下，踏空是最大风险 |

**退出机制**：
- 前排开板分歧 → 弹性票仓位降至≤30%，腾出资金切前排龙头
- 弹性票日涨幅≥15%+ → 不再加仓，设移动止盈（回撤-8%即出）
- 市场情绪从10分降至8分以下 → 弹性票全部清仓或降至≤20%
- 弹性票出现[[概念/筹码松动]]中度以上信号 → 立即清仓

## 五、交易纪律（11条操作底线）

1. **每天复盘，只做龙头** — 非龙头不参与
2. **退潮信号必须竞价减仓** — 总龙断板→不开新仓；连续高潮→只卖不买
3. **低于预期，格局有限度** — 次日低于预期，竞价减半仓
4. **高开非龙头分时直线向下清仓** — 强势票不会走这种形态
5. **低开不能快速翻红减仓** — 弱势已确认
6. **高位不做T，更不做梦** — 高位T = 给主力送流动性
7. **退潮日当天买入的票，次日无条件清仓** — 退潮惯性不可抱有幻想
8. **退潮/冰点期不做除试错外的任何开仓** — 等待下一个周期
9. **高开非龙头、分时向下，清仓**
10. **套利票次日必须走** — 套利的本质是短持，不能留恋
11. 被套个股第三天不强势反包必须清仓 — 防止小亏变大亏

## 六、执行力（3条心理锚点）

1. **交易计划执行 → 复盘检查** — 盘后逐条核对
2. **忍受踏空和卖飞** — 承认这不是你能赚到的钱
3. **每天都有机会，不赌博** — 等待下一个系统内机会
