# 股票严重异动计算规则

## 结论

开盘啦 `10天100%`、`30天200%` 的严重异动计算，核心不是“拿窗口第一天收盘做基准”，而是：

1. 用 **有效窗口内最低收盘价** 作为基准价
2. 同时扣除 **对应指数同期涨幅**
3. 界面显示 `N日偏离值` 时，实际计算使用 **N+1 个收盘价**，其中包含 `Day0`

## 核心公式

设：

- `b` = 有效窗口内最低收盘价所在日
- `P0` = 个股在 `b` 日的收盘价
- `I0` = 对应指数在 `b` 日的收盘价
- `Pt` = 当前价格或当前收盘价
- `It` = 当前指数或当前收盘指数

则：

```text
偏离值 = ((Pt / P0 - 1) - (It / I0 - 1)) × 100%
```

## 窗口规则

### 1. 显示天数和实际收盘数不是一回事

| 界面显示 | 实际计算收盘数 |
|------|------|
| 3日异动 | 4个收盘 |
| 7日偏离值 | 8个收盘 |
| 10日严重异动 | 11个收盘 |
| 21日偏离值 | 22个收盘 |
| 24日偏离值 | 25个收盘 |
| 30日严重异动 | 31个收盘 |

### 2. Day0 定义

- `Day0` = 第1个有效交易日前一交易日
- 开盘啦显示的 `21日/24日/30日`，是不含 `Day0` 的
- 但实际回算必须把 `Day0` 的收盘纳入窗口

### 3. 窗口滚动

- 每收盘一天，窗口向前滚一天
- 若最低收盘价被滚出窗口，基准价抬升
- 若严重异动触发并公告，窗口从下一交易日起重置

## 触发条件

| 规则 | 阈值 |
|------|------|
| 10天严重异动 | 偏离值达到 100% |
| 30天严重异动 | 偏离值达到 200% |

参照指数：

| 板块 | 指数 |
|------|------|
| 沪主板 | 上证指数 000001 |
| 深主板 | 深证成指 399001 |
| 创业板 | 创业板指 399006 |
| 科创板 | 科创50 000688 |

## 触发价公式

```text
触发收盘价 = 基准价 × (1 + 阈值 + 指数同期涨幅)
指数同期涨幅 = It / I0 - 1
```

## 开盘啦两种提示的计算口径

### 当日提示

```text
[当日] 收盘涨幅达到 X% 将触发
X = 触发收盘价 / 昨收 - 1
```

### 次日提示

```text
次日 涨幅达到 X% 将触发
X = 触发收盘价 / 今日收盘 - 1
```

## 已验证样本

### 中船特气 688146

- 开盘啦显示：`21日偏离值 166.05%`
- 正确口径：`21日 = 22个收盘`
- 回算结果：`166.04%`
- 当日触发：`18.48%`
- 次日触发：`12.59%`

与样本 `166.05% / 18.48% / 12.58%` 一致。

### 索辰科技 688507

- 开盘啦显示：`24日偏离值 156.24%`
- 正确口径：`24日 = 25个收盘`
- 回算结果：`156.25%`
- 当日触发：`14.67%`
- 次日触发：`16.64%`

与样本一致。

### 风华高科、泰坦股份、逸豪新材

- 收盘回算与样本接近
- 偏差主要来自开盘啦盘中使用 **实时股价 + 实时指数**

## 最终确定

以后关于严重异动，统一按下面这套口径：

```text
先取有效窗口内最低收盘价
再扣指数同期涨幅
显示N日时，实际计算用N+1个收盘
当日提示除以昨收
次日提示除以今日收盘
```
