# 交易总规则（2026-05-31 终版）

> **操作系统总纲**：市场情绪决定是否做，仓位管理决定做多少，交易模式决定怎么做，纪律决定能否做到，错误库决定如何纠偏。

---

## 一、交易基石（4条哲学原则）

1. **确定性 > 盈亏比 > 稳定盈利**
2. **仓位与确定性挂钩**
3. **交易模式、纪律、执行力三者统一**
4. **预判与盘面统一**

---

## 二、市场情绪量化模型（每日必看）

### 情绪分数的本质 🆕

情绪0-10分，衡量的是市场**超短线赚钱效应**：

| 构成因子 | 权重 | 说明 |
|:---|:---|:---|
| 短线赚钱效应的强度 | 高 | 涨停家数、连板高度、炸板率、跌停数 |
| 确定性个股的表现 | 高 | 你实际选择的1-2只最强标的的走势与封板质量 |
| 大盘整体走势 | 低 | 仅作为背景 |

> **情绪分既是大环境的温度计，也是你选股能力的成绩单。**

### 0-10分情绪温度计

| 情绪分 | 赚钱效应 | 典型盘面 |
|:---|:---|:---|
| 0分 | 无，千股跌停 | 流动性枯竭，必须空仓 |
| 3-4分 | 显著亏钱效应 | 高位崩塌，连板断板A杀 |
| 5分 | 一日游，随机拉砸 | 各板块随机拉砸——多数亏损温床 |
| 6分 | 轮动行情，容错率尚可 | 结构性机会 |
| 7-8分 | 明确主线+强赚钱效应 | 梯队完整，赚钱效应扩散 |
| 9-10分 | 超级大行情（年3-4次） | 百股涨停，大单一字，踏空资金溢出 |

**关键阈值**：≤5分总仓位≤3成；≥6分可满仓满融（需通过选股质量过滤器）。

## 三、仓位管理（二层控制体系）🆕

仓位管理由两层联合决定：**客观情绪分（第一层）× 主观选股质量（第二层）**。

### 3.1 第一层：情绪分基准仓位上限

| 情绪分 | 仓位上限（基准） |
|:---|:---|
| 0-2 | 空仓 |
| 3-4 | ≤20% |
| 5-6 | ≤50% |
| 7-8 | ≤80% |
| 9-10 | 满仓满融（需满足超级题材条件） |

### 3.2 第二层：选股质量过滤器 🆕

**核心原则**：5000多只票，只选1-2只最强标的。

| 问题 | 是 | 否 |
|:---|:---|:---|
| 我持仓是不是方向内角度最陡的标的？ | 按情绪分上限执行 | **仓位立即降一级** |
| 有没有更陡的角度出现而我未持有？ | 考虑切换，仓位不扩 | 维持 |
| 持仓是否属于资金正在流出的方向？ | **降至≤20%或清仓** | — |
| 我是否"选不出"最强标的？ | **降一级，优先解决选股** | — |

**🎯 执行口诀**：
> **非龙勿重仓，无确定不加仓。情绪分高≠你的赚钱效应高——只有你手里的票是核心，情绪分才属于你。**

**选股-情绪联动法则**：
```
持有最强龙一 → 按情绪分上限执行
持有次强/跟风 → 仓位上限自动降一档
无票/边缘标的 → 百股涨停也不满仓，先解决选股
```

**🆕 方向集中度法则**：
```
超级行情中，≥70%仓位必须集中在单一最强主线的角度最陡标的
持有多个方向龙头 = 变相的筹码松动 = 主动放弃主力仓位
```

### 3.3 超级行情例外条款：满仓满融条件

**必须同时满足**：
- 超级大题材（国家级产业政策，板块涨停≥15只）
- 情绪≥9分，百股涨停
- 方向内最强标的大单一字封死，封单≥10亿
- 引爆初始日，非行情末尾
- 角度最陡标的，非后排跟风
- **🆕 方向集中度要求**：≥70%仓位集中在最强主线的角度最陡标的——非最强方向的持仓合计<30%

**退出机制**：前排开板→弹性票降至≤30%；弹性票日涨幅≥15%→设移动止盈-8%；情绪降至8分以下→弹性票清仓或降至≤20%。

## 五、交易纪律（11条操作底线）

1. 每天复盘，只做龙头
2. 退潮信号必须竞价减仓
3. 低于预期，格局有限度
4. 高开非龙头分时直线向下清仓
5. 低开不能快速翻红减仓
6. 高位不做T
7. 退潮日当天买入的票，次日无条件清仓
8. 退潮/冰点期不做除试错外的任何开仓
9. 高开非龙头、分时向下，清仓
10. 套利票次日必须走
11. 被套第三天不强势反包必须清仓
