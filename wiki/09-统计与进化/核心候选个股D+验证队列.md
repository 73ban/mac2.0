# 核心候选个股 D+验证队列

更新时间：2026-06-28

## 定位

本页承接 `RAW增量个股核心池复核` 中的“正式核心候选”。

这里不是买入清单，而是验证清单。所有核心候选必须用 D+1、D+3、D+5、D+10、D+20、D+30 回看，验证自动打分是否真的能转化为超短线交易优势。

## 验证口径

| 验证点 | 看什么 |
|---|---|
| D+1 | 是否继续强于板块，是否有承接，是否骗炮 |
| D+3 | 是否仍在主线内，是否出现第一次有效分歧修复 |
| D+5 | 是否走出一段持续性，还是一日游/两日游 |
| D+10 | 是否仍有题材地位，是否进入趋势抱团或退潮 |
| D+20 | 是否被业绩、公告、价格数据验证 |
| D+30 | 是否形成可复用模式，或证伪自动核心池规则 |
