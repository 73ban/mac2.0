# 刺激强度与龙头生命周期

这份文档记录超短选股里最重要的一层：消息刺激有多强，市场环境是否匹配，个股是否处在龙头生命周期的关键阶段。

## 目标

不是复述消息，而是判断它会不会转化成次日或盘中的资金行为。

## 刺激因子

每条消息都要拆成：

```text
消息类型 -> 权威度 -> 具体性 -> 首次披露 -> 是否改变认知 -> 是否符合当前风格 -> 刺激强度
```

优先级通常从高到低：

1. 并购重组
2. 官方公告直指个股
3. 权威媒体、研报、官方消息点名
4. 订单、涨价、技术突破、产业链验证
5. 热榜和盘面确认

## 市场环境

同一条消息，在不同市场环境下效果不同。必须同时看：

1. 情绪强弱
2. 主线集中度
3. 连板生态
4. 异动压制是否有效
5. 资金是否抱团龙头

## 龙头生命周期

高位票不能只看题材，还要看它处于哪个阶段：

1. 首板启动
2. 二板三板加速
3. 断板
4. 反包
5. 高位震荡
6. 趋势新高
7. 是否被新龙头替代

## 规则落点

这类判断不允许只留在对话里，必须写进：

1. `data/brain`
2. `wiki/09-统计与进化`
3. 盘前候选和作战室排序

这样关掉窗口后，系统还能继续沿用这套判断。
