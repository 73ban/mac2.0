# 豫能控股天地板：三次卖点错过的完整拆解

## 错误定性

**类型：** 系统性执行失败（非技术判断失误）
**严重程度：** ⭐⭐⭐⭐⭐（最高级）
**可避免性：** 100%（三个卖点窗口各有清晰逻辑，非事后诸葛亮）
**损失：** 浮盈+26,638 → 浮亏-1,853，单日回撤 **2.85万元**

## 背景

- 股票：豫能控股（001896）
- 题材：电力+电算协同龙头，算力收购重组
- 前日状态：2连板，用户将其定位为"新龙头"
- 持仓：12,900股，成本20.01，总市值约25.6万
- 市场环境：情绪4分中等退潮日，电力板块集体崩盘前兆

## 三次卖点完整时间线

### 卖点一：9:35 — +8~9% 区间的"该强不强"

| 项目 | 详情 |
|------|------|
| **盘面** | 高开+4.2%，开盘后急拉至+9%（24.04） |
| **信号** | 开盘5分钟内未封涨停板 |
| **逻辑** | 作为电力板块+电算协同的新龙头，如果真正强势，开盘5分钟应快速封涨停。5分钟没封住 = 资金不认可 = 弱 |
| **应有动作** | 8-9个点减仓50%，锁定浮盈约1.3万 |
| **实际动作** | ❌ 没有卖出 |
| **心理原因** | "分歧之后还会拉升涨停"的幻想 |

### 卖点二：9:35 — 从+8%砸至零轴附近的"龙头被砸"

| 项目 | 详情 |
|------|------|
| **盘面** | 瞬间从+8%急砸至零轴附近，振幅约8个点 |
| **信号** | 龙头被砸，引发整个电力板块深度调整 |
| **逻辑** | 豫能是电力板块龙头，龙头被砸 = 板块走弱确认信号。不需要等反弹，立即清仓 |
| **应有动作** | 无条件清仓全部豫能控股 |
| **实际动作** | ❌ 没有卖出 |
| **心理原因** | 大脑短路，不愿承认走弱，"舍不得"减仓 |

### 卖点三：14:20 — 跌停板上的"幻想翘板"

| 项目 | 详情 |
|------|------|
| **盘面** | 14:20封死跌停板19.88 |
| **信号** | 跌停封死 = 最后逃命窗口 |
| **逻辑** | 跌停板封死后，次日大概率低开或继续杀跌。跌停价卖出是保命 |
| **应有动作** | 第一时间挂跌停价卖出 |
| **实际动作** | ❌ 故意晚了一点点挂单，想等翘板资金。结果封死到收盘，未成交 |
| **心理原因** | "幻想有其他资金翘板"+"想卖又不想卖"的矛盾 |

## 三重心理陷阱诊断

### 陷阱1：锚定效应
- **表现：** 锚定在"豫能是龙头"的认知上，忽视盘面已经走弱的客观信号
- **关联：** [[锚定效应-1]]
- **治疗：** "龙头也会死，而且龙头死的时候最惨。"盘面信号 > 任何认知锚定

### 陷阱2：亏损厌恶
- **表现：** 从浮盈2.6万跌到浮亏，心理无法接受利润消失，越跌越舍不得卖
- **关联：** [[亏损厌恶在交易中的表现]]
- **治疗：** 浮盈不是你的钱——没落袋的都是纸。从+9%到-10%，不是"丢了利润"，是"亏了本金"

### 陷阱3：幻想翘板
- **表现：** 跌停板上故意晚卖，等待其他资金翘板救援
- **关联：** [[幻想翘板——跌停板上的致命犹豫]]
- **治疗：** 翘板是小概率事件。活下来比多赚几个点重要一万倍。跌停即卖，不幻想。

## 根因总结

**这不是技术问题，是态度问题——不尊重盘面，只看自己想要的剧本。**

三次卖点，每一次都有清晰的技术逻辑支撑卖出。但"幻想"贯穿了整个操作过程：
1. 幻想分歧后继续拉升 → 错过+9%卖点
2. 幻想龙头不会死 → 错过零轴逃命窗口
3. 幻想翘板资金救援 → 错过跌停板最后窗口

**技术判断力在，执行力却归零**——这是典型的"知行断裂"。

## 系统性教训

1. **卖点窗口稍纵即逝。** 从+9%到跌停，三个窗口加起来可能只有几分钟的有效操作时间。没有预案=必死。
2. **退潮日必须降仓位。** 如果开盘按[[退潮日仓位管理规则]]减至半仓，亏损将减半。满仓满融是放大损失的根本原因。
3. **盘前预期管理缺失。** 用户自述"只看了想要的剧本"。应在盘前写下多种情景预案（强势封板/冲高回落/直接走弱）。
4. **"利好出尽"即卖点。** 算力收购消息落地当天就是Sell the news窗口。参见[[豫能控股Sell-the-News案例]]和[[Sell the news机制]]。
