## 偏离值计算

### 1. 确定性偏离

```
事前确定性：9
实际结果：天地板跌停 → 映射分 = 0
确定性偏离 = 0 - 9 = -9
```

**等级：🔴 严重高估**

> 判断为"几乎必赚"（9分），实际遭遇天地板跌停。差距极大。

### 2. 盈亏比偏离

```
预期涨幅：+30%
实际最大涨幅：+20.1%（24.04 / 20.01 - 1）
盈亏比偏离 = +20.1% - +30% = -9.9%
```

**等级：🟡 偏高估**

> 实际最高冲到24.04，距预期26仅差约8%。方向正确，但没卖在最高点。偏离不算灾难级。

### 3. 最大回撤偏离

```
无事前止损线 → 无法计算偏离
实际最大回撤：-17.3%（24.04→19.88）
```

**等级：🔴 无止损纪律**

> 如果有止损线（如-5%），实际回撤远超止损线。没设止损+没执行=最大问题。

1. **确定性9分以上必须同时满足**：板块持续强势 + 催化未兑现 + 盘面无分歧信号 + 无监管风险。豫能6/5盘前已不满足（监管降温+催化即将兑现）。
