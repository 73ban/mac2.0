# 中国长城与飞腾CPU深度分析：六大国产CPU全景对比

> 📅 2026年5月21日 | 数据来源：太平洋证券研报、财联社、证券时报、国金证券、龙芯中科/海光信息公告

## 一、中国长城与飞腾：不是"控股"，是"第一大股东+命运共同体"

### 股权关系

| 维度 | 数据 |
|------|------|
| 中国长城持飞腾股权 | **28.04%**（第一大股东） |
| 飞腾对长城的意义 | 长城所有计算终端（台式机/笔记本/服务器）以飞腾芯片为算力底座 |
| 飞腾对长城的收入贡献 | 不直接并表，但长城是飞腾芯片最大的整机出货渠道 |
| 飞腾芯片供应商 | 飞腾信息（未独立上市） |

### 商业模式

```
飞腾信息（芯片设计公司）
    ↓ 供应CPU芯片
中国长城（整机制造商 + 第一大股东）
    ↓ 制造服务器/台式机/笔记本/AI训推一体机
党政/金融/央企/关键行业客户
```

> **中国长城是飞腾芯片变现的"主力渠道"。** 飞腾设计芯片→长城造整机→卖给党政信创客户。飞腾芯片卖得越多，长城整机出得越多——这是一个正反馈闭环。

### PKS体系：飞腾的护城河

| 字母 | 代表 | 角色 |
|------|------|------|
| **P** | Phytium（飞腾） | CPU芯片 |
| **K** | Kylin（麒麟） | 操作系统 |
| **S** | Security（安全） | 内生安全+过程安全+供应链安全 |

> PKS比鲲鹏路线的关键差异：**安全属性更强**——飞腾路线为信创工程提供从芯片到系统的全流程安全服务。这就是为什么党政信创首选飞腾而非鲲鹏。

## 三、飞腾 vs 海光 — 最关键的对比

| 维度 | 飞腾 | 海光信息 |
|------|------|------|
| **架构** | ARM v8 | x86（AMD Zen1） |
| **自主可控** | 🟢 ARM永久授权，可持续演进 | 🟡 AMD停供新授权，仅Zen1，需自研迭代 |
| **生态兼容** | 🟡 Linux生态，不能跑Windows | 🟢 **完全兼容x86/Windows生态** |
| **性能水平** | 🟡 80核比肩业内高端 | 🟢 浮点性能达Zen1的10倍，接近国际水平 |
| **主战场** | 🥇 **党政信创绝对王者** | 🥇 服务器/运营商/金融 |
| **第一大股东** | 中国长城（28.04%） | 中科曙光 |
| **营收规模** | 未独立上市 | 51亿+（2022年）|
| **生态成熟度** | PKS体系，安全性强 | x86成熟生态，迁移成本低 |

### 关键差异一句话

> **海光靠性能+生态抢服务器市场，飞腾靠安全+信创抢党政市场。** 不是直接竞争关系——它们是国产CPU的两条腿：飞腾走党政信创（安全优先），海光走商业服务器（性能优先）。

## 四、飞腾在中国CPU市场的地位

### 四个维度综合评分

| 维度 | 评分 | 说明 |
|------|------|------|
| 党政信创市场份额 | 🥇 第一 | 128部委+32省市，不可撼动 |
| 技术自主可控 | 🟢 高 | ARM v8永久授权，不受美国制裁影响 |
| 生态成熟度 | 🟡 中 | PKS体系完整但不如x86生态宽广 |
| 商业市场竞争力 | 🟡 中 | 信创以外市场不如鲲鹏/海光 |
| 性能水平 | 🟡 中上 | 追赶中，S5000C-E 80核接近高端 |
| AI算力布局 | 🟢 加速 | DeepSeek大模型已与飞腾算力深度耦合 |

### 核心结论

```
飞腾 = 党政信创的"Intel"
海光 = 商业服务器的"AMD"
鲲鹏 = 云计算的"自研芯片"

三者不是零和博弈：
- 信创市场 → 飞腾优先
- 商业市场 → 海光/鲲鹏优先
- 超算/特殊 → 申威/龙芯

国产CPU总共要替代的是一个万亿级市场，三家都能吃饱。
```
