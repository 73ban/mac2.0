# RAW增量个股候选池复核

更新时间：2026-06-28

## 定位

本页承接 `RAW增量个股候选池清单`。候选池不等于买入池，只能进入作战室观察，必须通过 L1/L2/L4 过滤。

## 使用规则

1. 候选池只负责提醒，不负责下结论。
2. 只有同日主线确认、竞价确认、量能确认，才允许进入明日作战计划。
3. 未通过 D+验证前，不升级正式个股档案。
4. 来源偏少、乱码催化、低信号来源，一律降权。

## 分层口径

| 条件 | 处理 |
|---|---|
| 来源数 >= 8 且催化字段正常 | 作战室观察，等待主线和竞价确认 |
| 来源数 5-7 | 中等观察，必须有板块确认 |
| 来源数 < 5 | 低权重观察 |
| 催化字段乱码/质量差 | 先核验证据，不进入买入结论 |
