# 重点个股作战档案模板

更新时间：2026-06-13

## 用途

这个模板用于把重点个股从“资料堆积”整理成“能不能交易、怎么交易、错了怎么卖”的作战档案。

## 基本信息

```yaml
stock:
code:
theme:
role: 龙头/容量中军/趋势核心/弹性前排/补涨/套利/风险锚
last_update:
data_sources:
truth_grade: S1/S2/S3/S4
```

## 仓位权限

| 条件 | 仓位 |
|---|---|
| 只有资料逻辑，无盘口确认 | 不买或观察 |
| 题材确认，个股非核心 | 轻仓 |
| 题材确认，个股核心，竞价符合 | 标准仓 |
| 主升行情，A 类模式，核心票，竞价超预期 | 可讨论重仓 |

## 错误警报

1. 把后排当核心。
2. 把研报逻辑当短线买点。
3. 把高开诱多当弱转强。
4. 把退潮反抽当主升。
5. 没有退出条件就买。
