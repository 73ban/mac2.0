# 退潮结束识别

## 定义

退潮结束识别是判断市场从持续下跌/恐慌状态（退潮期）转入修复/新周期（主升期）的信号系统。它是[[概念/退潮日]]的对称知识——不仅要知道退潮来了怎么防守，更要知道退潮走了怎么进攻。

## 三维度确认框架

本框架提炼自2024年11月28日退潮正式结束的判断逻辑。

### 维度一：跌停家数降至阶段性最低

| 信号 | 说明 |
|------|------|
| 非ST跌停降至单日个位数 | 2024-11-28仅6家，创本轮退潮最低 |
| 连续3日无扩散 | 跌停不再蔓延至中位股，恐慌链断裂 |
| 跌停封单规模萎缩 | 剩余跌停票封单极小（如三维化学仅0.2亿），即将开板 |

这是**客观量化指标**，不受主观情绪干扰。

### 维度二：情绪指数跳升

| 信号 | 说明 |
|------|------|
| 情绪分从6→8 | 单日跳升2分，从"恐慌"区进入"积极"区 |
| 涨停家数保持三位数 | 110家涨停，市场韧性强 |
| 赚钱效应广泛 | 非单一妖股独立行情，而是多板块共振 |

这是**主观判断但可校验**的指标。

### 维度三：新主线确立且龙头明确

| 信号 | 说明 |
|------|------|
| 新题材爆发 | 谷子经济/IP经济全产业链扩散 |
| 龙头封单强度 | 广博股份7连板一字10亿封单，全场最大 |
| 梯队完整 | 7→6→5→4→3→2→首板完美衔接 |
| 多主线助攻 | 消费+物流+深圳国资共振，非孤军深入 |

这是**结构性指标**——退潮能否真正结束，取决于市场是否找到了新的叙事方向。没有新主线的反弹只是技术修复，有新主线的反弹才是周期切换。

## 退潮→修复→新周期的完整信号链

```
退潮开始 → 高标断板 + 跌停骤增 + 中位股崩塌
退潮持续 → 跌停维持两位数 + 连板高度压缩 + 核按钮频发
退潮尾声 → 跌停开始减少 + 旧周期票补跌完毕
退潮结束 → 跌停降至个位数 + 情绪跳升 + 新主线确立 ← YOU ARE HERE
修复确认 → 新主线持续 + 跌停归零 + 成交放量
新周期启动 → 主线龙头连板延展 + 梯队补涨 + 情绪创新高
```

## 案例：2024-11-28

| 维度 | 状态 |
|------|------|
| 跌停家数 | 6家（本轮最低）✅ |
| 情绪指数 | 8分（昨日6→今日8）✅ |
| 新主线 | 谷子经济，广博7连板10亿一字 ✅ |
| 梯队 | 7→6→5→4→3→2→首板完整 ✅ |
| 多主线 | 消费+物流+深圳国资共振 ✅ |

**结论**：退潮正式结束。

## 退潮结束后的仓位管理（关键）

退潮结束≠可以全仓。修复期存在"假结束"风险：

1. **首日轻仓试探**（≤30%）：确认新主线持续性
2. **次日加仓**（30-50%）：新主线龙头封单不减且跌停无回升
3. **次日满仓**（仅当跌停归零 + 新主线梯队完整 + 3天无回潮时）

**错误案例**：2024-11-28用户重仓粤桂75%集中度，粤桂是旧周期票，与谷子经济新主线无关。——退潮结束判断正确，但仓位投向错误。
