# 市场环境扫描清单

## 目的

在每日/每周复盘前，系统性地扫描可能影响A股的外部变量和内部市场状态，避免“只埋头个股而忽视系统性风险”。本清单属于**L1市场环境层**的核心工具。

## 外部宏观扫描（每日开盘前）

### 美联储与利率
- [ ] 隔夜美债收益率变动（2Y / 10Y）
- [ ] 联邦基金利率期货隐含的加息/降息概率
- [ ] FOMC议息会议时间表（下一次什么时候）
- [ ] 美联储官员讲话/表态

### 关键经济数据
- [ ] 本周/下周待公布数据日历（CPI、非农、PPI、GDP等）
- [ ] 最新数据与预期的偏离度

### 美股表现
- [ ] 隔夜三大指数涨跌幅（纳指、标普、道指）
- [ ] 费城半导体指数（SOX）——对A股芯片/科技映射最强
- [ ] VIX恐慌指数——是否>20、是否单日跳升>20%
- [ ] 板块轮动信号（等权重指数 vs 市值加权指数）

### 地缘与商品
- [ ] 油价走势（WTI/布伦特）
- [ ] 重大地缘事件

## A股内部市场状态（每日收盘后）

### 情绪指标
- [ ] 连板高度、涨停/跌停数量
- [ ] 主线题材是否出现分歧/退潮信号
- [ ] 资金从哪个板块流出、流向哪个板块

### 量价结构
- [ ] 成交量变化（放量/缩量、相对前期均值）
- [ ] 指数是否处于关键支撑/压力位

### 北向资金
- [ ] 当日净流入/流出
- [ ] 外资重仓股表现

## 风险信号汇总（三色评级）

| 颜色 | 条件 | 行动 |
|------|------|------|
| 🟢 绿色 | 无显著外部冲击，内部市场正常 | 正常按体系操作 |
| 🟡 黄色 | 外部有冲击但A股可能已部分price in，或内部出现分歧信号 | 降低仓位、收紧止损 |
| 🔴 红色 | 外部三重共振级别冲击，或内部明确退潮 | 大幅减仓/空仓，只做防御 |

## 使用频率

- **开盘前**：快速扫描外部宏观（5-10分钟）
- **收盘后**：结合当日盘面更新内部状态
- **周度**：汇总梳理，识别趋势性变化
