# 2026-07-02 大退潮情绪2分样本

## 核心结论

2026-07-02 是一次标准大退潮样本。用户给市场情绪打分 **2/10**，这个分数应作为系统训练锚点写入 L1。

它不是普通下跌，而是：

1. 外部冲击引发 AI/算力/半导体链集体杀跌。
2. 前期强主线从“能涨”切换成“批量证伪”。
3. 热榜 Top100 中跌停和深跌票密集出现。
4. 个股局部仍有涨停活口，但系统风险大于个股机会。
5. 正确动作不是找最强涨停，而是先执行清仓、降风险、等待冰点修复。

| 项目 | 事实 | 含义 |
|---|---|---|
| 用户情绪分 | 2/10 | 明确进入冰点/退潮执行框架 |

## 这一天要训练出的认识

### 1. 退潮不是没有涨停

大退潮日仍然会有涨停、连板、地天板和活口。

如果只盯涨停，系统会误以为还有机会；如果看热榜跌停扩散、核心主线崩塌、D+验证批量证伪，就能识别真实环境。

正确判断：

```text
退潮日的核心不是“有没有票涨停”，而是“亏钱效应是否压过赚钱效应”。
```

### 2. 主线退潮时，文本催化自动降权

7月2日很多科技、半导体、先进封装、存储方向仍有公告、研报、产业逻辑，但盘面资金选择了跌停和深跌。

规则：

```text
退潮日里，文本逻辑不能对抗资金撤退。
没有盘口承接和板块止跌，消息越硬也只能观察，不能追。
```

### 3. 热榜 Top100 是退潮识别入口

热榜不是只用来找机会，也用来识别风险。

7月2日热榜中大量高人气科技票深跌，说明散户注意力集中在亏钱效应上。高热度不再代表进攻，而代表踩踏、被套和恐慌传播。

以后 Top100 必须同时回答：

```text
今天热榜是在扩散赚钱效应，还是在扩散亏钱效应？
```

### 4. 清仓正确性优先于卖点完美

多氟多第一次反抽未卖，是可以优化的卖点问题；但二次确认退潮不可逆后清仓，避免从“卖慢”演化为“硬扛”。

退潮日评价交易，不能只评价价格高低，要先评价是否完成风险切断。

```text
退潮日卖得不完美，不等于错。
退潮日不卖、补仓、追高，才是系统性错误。
```

### 5. 逆势强票不等于可交易

先导基电逆势涨停，但背景是半导体大面积跌停。用户排板后撤单，收盘看像错过，按环境看是纪律。

退潮日逆势票的真实问题在 D+1：

```text
强退潮日逆势涨停，必须验证次日是继续晋级，还是补跌。
没有 D+1 验证前，不能把“逆势涨停”直接升级成买入模式。
```

## 情绪2分执行规则补充

当用户明确判定市场情绪为 2 分，并且出现核心主线批量证伪时，执行以下权限：

| 项目 | 权限 |
|---|---|
| 新开仓 | 禁止追高，原则上不开 |
| 连板接力 | 禁止早盘追；只观察 D+1 承接 |
| 深水低吸 | 只允许核心票、小仓、确认承接后执行 |
| 持仓处理 | 优先清仓或降到最低风险敞口 |
| 作战室 | 从“进攻候选”切换为“风险锚 + 修复验证锚” |
| 消息权重 | 文本催化降权，资金验证优先 |
