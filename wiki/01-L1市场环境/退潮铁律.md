# 退潮铁律

## 核心定义

退潮 ≠ 所有股票都跌。退潮分为两个层级：

### 市场退潮（必须空仓）
> 满足以下**全部三个条件**，判定为市场退潮：

1. **市场情绪分 ≤ 4 分**（参考[[市场情绪指数]]评分体系）
2. **市场最核心的几支票均低于预期**，出现较大的负反馈（龙头断板/炸板 + 其他核心票大跌）
3. **负反馈扩散到多个板块**，不是单一板块的问题

→ 触发市场退潮：**立即空仓，不开新仓，不做任何操作**

### 板块退潮（可以换板块）

> 当市场处于**增量资金环境**（如日均成交额持续在 3 万亿左右），某单一板块退潮，但：

- 市场整体情绪分 > 4 分
- 其他板块不受影响
- 资金从退潮板块撤出，重新流入没有负反馈的新板块

→ 这种情况**不属于全局退潮**，可以参与其他板块的龙头

## 退潮信号

| 信号 | 市场退潮 | 板块退潮 |
|------|:---:|:---:|
| 市场情绪分 ≤ 4 | ✅ 必须满足 | ❌ 不适用 |
| 龙头断板/炸板 | ✅ | ✅（仅该板块龙头） |
| 多只核心票同时负反馈 | ✅ | ❌（仅限该板块内） |
| 连板高度骤降（7板→3板） | 🔴 高危 | 🟡 关注 |
| 跌停家数 ＞ 涨停家数 | 🔴 高危 | ❌ 不适用 |
| 炸板率 ＞ 40% | 🔴 高危 | ❌ 不适用 |
| 资金流出到防御板块 | 🟡 预警 | ❌ — 正常轮动 |
| 成交额萎缩+多个板块走弱 | 🔴 高危 | — |

## 强制规则

### 市场退潮
1. 退潮确认后，当日不再开新仓
2. 持仓中非核心票全部清仓
3. 核心票降至半仓以下
4. 退潮票早盘必须清仓 — 不等到"反弹再卖"
5. 融资账户降杠杆至 0

### 板块退潮
1. 清仓该板块所有持仓
2. 其他板块正常操作
3. 观察资金从退潮板块流出的方向，识别新主线

## 退潮票早盘清仓触发条件

- 昨日跌停 → 今日开盘竞价直接挂跌停卖出
- 昨日断板 → 今日 9:25 竞价低于 5% → 直接清
- 昨日冲高回落 → 今日开盘低于前收盘 → 清
- 任何退潮期开仓的票 → 不恋战，次日有赚就走

## 判断流程

```
今日市场情绪分？
  ├── ≤ 4 分 → 继续往下看
  │   ├── 核心票是否集体负反馈？→ 是 → 🚨 市场退潮，空仓！
  │   └── 否 → 观察，可能是震荡
  └── ＞ 4 分 → 增量资金环境
      ├── 某个板块龙头断板 → 板块退潮，清该板块
      └── 参与其他未退潮板块的龙头
```

## 违规代价

- 退潮期追高 → 次日跌停 → 锁死流动性
- 退潮期加仓 → 亏损放大 → 情绪崩盘
- 退潮期抄底 → "抄在半山腰"
