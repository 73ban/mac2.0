# 混沌期老龙头情绪承接

## 一句话

大盘弱势、主线不清晰时，短线资金需要一个情绪承接口。近期最高辨识度的老龙头，在连续杀跌洗盘后，早盘出现反核封板——打板买入，赚的是**封板确定性和次日情绪溢价**。

## 核心逻辑

**"弱市炒老妖"的现代版本。**

- 主线退潮或真空 → 短线资金无处可去
- 但资金不会完全离场，必须找一个"大家都认识"的标的承接情绪
- 近期最高板/最大妖股（如18天11板、6板稀缺高度）就是全市场的共同记忆
- 连续杀跌（跌停/大阴线）把浮筹洗干净后，反核阻力最小
- **打板买入的核心**：板上确认资金合力，赚的不是当天差价，是第二天的高开溢价

不是赌基本面，是赌**市场注意力的流向 + 次日溢价**。

## 触发条件

### 必须同时满足：

| 条件 | 说明 |
|------|------|
| **L1 混沌或弱势** | 大盘下跌或横盘，主线不清晰，无明显板块效应 |
| **标的近期最高辨识度** | 近1个月内达到过稀缺高度板（如6板+），或曾是大妖股（如10板+），全市场短线玩家都认识 |
| **充分洗盘** | 断板后至少1-2个跌停（或连续大阴线），浮筹大幅出清，筹码结构干净 |
| **竞价反核信号** | 竞价高开（或尾盘竞价拉升），表明资金进场承接 |
| **早盘快速封板** | 开盘后快速拉板，必须是市场合力封板（有换手），非独庄无量一字 |

### 加分项：

- 股性历史好（之前连板干净利落，非庄股对倒）
- 早盘辨识度在同期断板票中最突出
- 封板时换手充分（非一字板无量封）

## 买入规则

1. **仓位**：小仓位试错（≤10%总仓位），确定性来自市场合力而非基本面
2. **买点优先级**：**打板为主 → 半路为辅 → 低吸不适合。** 
   - **打板（主）**：封板那一刻是资金合力的确认信号，确定性最高。板上排板，争取回封时成交
   - **半路（辅）**：竞价反核信号极强 + 拉升斜率陡峭时，可以在封板前小仓位先手，但不能替代打板的确定性确认
   - **低吸（不适用）**：这个模式赚的是次日溢价，不是当天差价。低吸赌反核是左侧博弈，确定性太低，不符合模式逻辑
3. **为什么打板为主**：不是追高，是买确定性。封板 = 市场用真金白银投票选了这只票。第二天高开溢价比板上那一点差价大得多

## 卖点规则

| 情景 | 动作 |
|------|------|
| 次日封板（涨停封住） | 🟢 持有，反包成功 |
| 次日炸板或冲高回落 | 🔴 无条件清仓，反核失败 = 逻辑证伪 |
| 次日低开无承接 | 🔴 竞价即走 |
| 次日高开溢价（+3~5%） | 🟢 分批止盈，赚溢价才是这个模式的目的 |
| 次日高开但快速转弱 | 🔴 破分时均线即走 |

**持股周期：1-2天。** 这不是趋势票，是情绪脉冲。核心收益来自隔日溢价，不是连板预期。

## 禁区

- 退潮初期做反核（退潮第一天的"抄底" = A杀起点）
- 标的辨识度不够（非近期最高板/最大妖股，辨识度打折）
- 封板无量（庄股对倒，非市场合力，第二天没人接）
- 大盘环境极度恶劣（千股跌停级别，第二天没人接溢价）

## 与「断板低吸」的区别

| | 断板低吸 | 混沌期老龙头情绪承接 |
|------|------|------|
| 环境 | 主升期可用 | 混沌期/弱市专属 |
| 逻辑 | 技术修复反弹 | 资金情绪出口 + 隔日溢价 |
| 催化 | 板块联动 | 无板块，纯辨识度 |
| 买点 | 分歧低吸 | **板上确认（打板）** |
| 收益来源 | 反弹涨幅 | **次日高开溢价** |
| 持仓周期 | 可稍长 | 1-2天，脉冲即走 |

## 案例

### 案例 1：金螳螂（002081）2026-05-29 ✅ 成功

- **辨识度**：18天11板，A股最大妖股之一，全市场公认
- **洗盘**：妖股退潮后持续回落，浮筹大幅出清
- **买入**：5/29 混沌期打板买入 5000股，成本 ~6.17
- **卖出**：6/1 卖出 5000股 @6.34，盈利 **+873元（+2.83%）**
- **逻辑验证**：混沌期打板老龙头 → 次日溢价兑现 → 隔日止盈

### 案例 2：大有能源（600403）2026-06-11 ⏳ 待验证

- **辨识度**：前期6板（近1月稀缺高度），市场记忆深刻
- **洗盘**：7板断板 → 双跌停，浮筹出清
- **买入**：6/11 竞价高开 +2%，快速封板（涨停 8.42），打板买入 3200股
- **当天状态**：盘中炸板回落至 8.11（+6.01%），封板质量存疑
- **待观察**：明日（6/12）能否封板或给出溢价
