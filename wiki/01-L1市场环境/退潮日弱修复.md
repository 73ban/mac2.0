# 退潮日弱修复

## 定义

退潮日弱修复是指**在退潮大周期中出现的短暂反抽型修复**：指数仍绿、总龙头断板、成交量萎缩，但个股涨多跌少。这是一种"指数-情绪背离"的**退潮中继**信号，不是反转。

## 弱修复 vs 真修复的判别

| 维度 | 弱修复（退潮中继） | 真修复（转主升） |
|------|-------------------|-------------------|
| 指数 | 全绿或窄幅震荡 | 收阳或至少一阳一星 |
| 成交量 | 缩量（较前一日-10%以上） | 放量（至少回到退潮前水平） |
| 龙头状态 | 断板/大烂板 | 续板或强势反包 |
| 板块梯队 | 断层（例如5板断→4板烂） | 完整（各层有代表） |
| 赚钱效应 | 上涨家数多但全是小票，没有引领方向 | 主线明确，有板块效应 |
| 跌停家数 | 仍偏高（>15家） | 明显收敛（<10家） |
| 情绪指数 | 2-4分 | 5+分 |

## 实战案例：2026-06-01

| 指标 | 数值 | 判定 |
|------|------|------|
| 上证指数 | -0.27% | ❄️ 弱势 |
| 深证成指 | -1.51% | ❄️ 弱势 |
| 科创50 | -5.00% | 🔴 暴跌 |
| 成交量 | 2880亿（缩量439亿） | ❄️ 缩量 |
| 上涨vs下跌 | 2013:875 | 🔀 背离 |
| 跌停家数 | 21家 | ❄️ 偏高 |
| 电力龙头 | 华电能源5板断板 | 🔴 退潮信号 |
| 情绪指数 | 3分 | ❄️ 退潮区 |

**结论**：所有实质量指标指向退潮延续。仅上涨家数一维看似修复，但这正是因为权重股在拖累指数，小票在反抽——典型的"指数-情绪背离"弱修复。

## 操作规则

1. **弱修复日不开新仓** — 宁可错过，不能做错
2. **弱修复日的最优操作是全部清仓** — 利用流动性窗口逃命
3. **弱修复≠抄底机会** — 次日往往是真正的杀跌
4. **弱修复日唯一例外**：有明确的新主线+新题材第一天爆发+成交量不萎缩 — 但这种情况在退潮大周期里几乎不存在

## 弱点

弱修复最容易触发**确认偏差**——交易者会挑出对自己有利的数据（上涨家数多），忽略不利的信号（龙头断板、缩量、跌停偏多）。这就是为什么需要一个明确的判别框架来防止自我欺骗。
