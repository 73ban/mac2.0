# L1市场环境分类标准

更新时间：2026-06-13

## 用途

本文件给老虎、龙虾、大鸟和统计看板统一 L1 市场环境标签。补历史交割单时，忘记买卖理由可以写 `unknown`，但当天市场环境必须尽量客观复原。

L1 环境决定能不能做、做什么模式、给多少仓位。不能把连板行情、机构趋势行情、抱团行情、题材轮动、退潮行情混在一起统计。

从 2024-09-01 起回填历史交割单时，L1 不只打一个“涨跌好坏”标签，还要识别当时的主流赚钱结构：

```text
连板赚钱结构
机构趋势赚钱结构
抱团赚钱结构
题材主升赚钱结构
轮动套利结构
防守/退潮结构
```

只有交易落在当时主流赚钱结构上，确定性和收益上限才最高。行情不好时，也可能有局部主流结构，但仓位权限必须下降。

## 一级分类

| L1标签 | 核心特征 | 优先模式 | 禁用/降权 | 仓位权限 |
|---|---|---|---|---|
| 连板行情 | 连板高度打开，晋级率高，空间板/换手龙有溢价，断板反馈不差 | 连板空间破局、换手龙回封、竞价弱转强、前排确认打板 | 趋势慢票后排、无辨识度低吸 | 标准仓；强情绪可提高 |
| 机构趋势行情 | 成交额核心强，容量中军持续新高或回踩承接，连板弱但趋势赚钱效应强 | 强趋容量抱团、趋势回踩均线、主线核心低吸、容量票强弱锚 | 追连板后排、情绪小票乱打 | 标准仓 |
| 抱团行情 | 资金高度集中在少数核心票或少数方向，龙头/容量核心持续强，后排和非主线明显失血 | 强趋容量抱团、题材主升核心持股、主线核心低吸、核心抗跌穿越 | 非主线、后排补涨、分散持仓 | 标准仓；只限核心 |
| 题材主升行情 | 主线清晰，核心/中军/前排/补涨梯队完整，分歧后能修复 | 题材主升核心持股、主线核心低吸、强板块前排半路 | 后排不碰、高潮后追补涨 | 标准仓到可重仓 |
| 题材轮动行情 | 多题材快速切换，日内龙频繁换，隔日持续性弱 | 首板新题材试错、日内龙套利、缩量轮动套利 | 重仓、格局后排、隔日幻想 | 轻仓到标准仓 |
| 冰点行情 | 涨停少、跌停/大面多、亏钱效应集中，资金恐慌 | 计划外交易禁止、风险锚减仓；新东西站出来只观察 | 打板、接力、重仓低吸 | 空仓到极轻仓 |
| 修复行情 | 冰点后指数/情绪修复，核心止跌，部分前排反包 | 冰点恐慌修复、良性分歧修复、竞价弱转强 | 弱修复追后排、无主线重仓 | 轻仓到标准仓 |
| 退潮行情 | 高标 A 杀，连板高度压缩，后排大面，核心补跌 | 风险锚减仓、计划外交易禁止、亏损后降频 | 连板接力、打板、后排半路、补仓 | 空仓到极轻仓 |
| 混沌行情 | 指数和题材都无方向，赚钱效应不稳定，日内冲高回落多 | 观察、轻仓试错、计划外交易禁止 | 重仓、追涨、频繁换仓 | 空仓到轻仓 |
| 指数逼空行情 | 指数连续强，券商/权重/容量票推动，个股分化可能大 | 趋势抱团、容量中军、主线核心低吸 | 小票后排、无量情绪票 | 标准仓，避免乱追 |
| 系统性风险行情 | 外围暴跌、指数破位、跌停扩散、流动性不足 | 防守、减仓、空仓 | 所有进攻模式 | 空仓到极轻仓 |

## 关键分歧：连板行情 vs 机构趋势行情 vs 抱团行情

这三个必须分开，否则历史统计会失真。

### 连板行情

重点看：

```text
最高连板高度
连板家数
2板/3板/4板/5板+梯队
晋级率
断板反馈
炸板率
空间板是否有溢价
中位股是否亏钱
一字板和换手板质量
```

适合：

```text
连板空间破局
换手龙回封
竞价弱转强
前排确认打板
一字定方向扩散
```

### 机构趋势行情

重点看：

```text
成交额前50
成交额50亿/100亿以上强势股
20日/60日新高数量
容量中军是否补跌
趋势核心是否放量滞涨
板块成交额是否扩张
机构方向是否持续：AI算力/半导体/PCB/机器人/有色/券商等
```

适合：

```text
强趋容量抱团
趋势回踩均线
主线核心低吸
容量票强弱锚
核心抗跌穿越
```

### 抱团行情

重点看：

```text
成交额是否向少数方向集中
成交额前20/前50 是否被同一主线占据
核心票是否持续强于指数和板块
后排是否明显失血
非主线是否没有赚钱效应
ETF/机构/北向/大成交票是否持续集中
龙头是否从逻辑定价切到流动性定价
抱团是否出现拥挤：高估值、放量滞涨、全民化叙事
```

适合：

```text
强趋容量抱团
主线核心低吸
题材主升核心持股
核心抗跌穿越
风险锚减仓
```

抱团不是简单的机构趋势。机构趋势强调“容量中军趋势赚钱”；抱团强调“资金没有更好选择，集中抱少数核心”。抱团行情里，小资金可以跟随核心，但不能买后排，因为后排可能持续失血。

## 历史交割单补标规则

1. 当时买卖理由忘记，就写 `original_reason: unknown`。
2. L1 环境用当日客观数据复原，不用回忆。
3. 事后推断模式必须写 `inferred_confidence`。
4. 低置信度推断不能用于 A 类模式升级。
5. 所有交易都可用于错误分析，但不是所有交易都可用于模式验证。

## 统计用途

历史交易补齐后，统计看板应回答：

```text
连板行情里，我赚不赚钱？
机构趋势行情里，我赚不赚钱？
抱团行情里，我是否只买核心，还是买了后排？
退潮行情里，我是否仍在重仓？
题材轮动行情里，我是否追后排？
计划外交易在不同 L1 环境下损失多少？
哪类 L1 环境最适合小资金快速进攻？
哪类 L1 环境必须空仓保命？
```

## 禁止

1. 不能用主观体感替代客观数据。
2. 不能把单只牛股行情标成题材主升。
3. 不能把指数红盘但个股大面标成好行情。
4. 不能把机构趋势行情误判为连板行情。
5. 不能把抱团行情误判为普涨行情。
6. 不能在抱团行情里分散买后排。
7. 不能把退潮反抽当新主升。
