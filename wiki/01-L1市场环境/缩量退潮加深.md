# 缩量退潮加深

## 定义
在退潮市场中，成交量连续萎缩是市场深度恶化的核心量化指标。缩量 → 承接资金不足 → 赚钱效应进一步收窄 → 确定性持续下降。

## 信号链
```
连续缩量 → 资金离场加速
       ↓
   涨停溢价率下降 → 打板/追高胜率骤降
       ↓
   跌停家数增加 → 恐慌蔓延
       ↓
   市场确定性螺旋下降 → 应降仓/空仓
```

## 5/18 数据锚点
| 指标 | 数值 | 信号强度 |
|------|------|----------|
| 成交额变化 | -4,532亿（连续2日缩量） | 🟢强 |
| 跌停家数 | 53家 | 🟢强 |
| 下跌家数 | 3,000+ | 🟢强 |
| 涨停家数 | 102家 | 🟡中（仍有局部活跃，但持续性低） |

## 操作纪律
- **连续缩量日**（阈值为单日缩量超3,000亿或连续2日缩量）：不开新仓
- **跌停>50家+缩量**：减仓至≤30%
- **赚钱效应收窄+缩量**：不做追高/半路，只做低吸且有明确止损
