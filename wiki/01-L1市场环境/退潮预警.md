# 退潮预警

## 定义

**退潮预警**是指市场仍处于高潮期时出现的、预示市场情绪即将转折的前兆信号。它与[[概念/退潮日|退潮日]]不同——退潮日是已发生的衰退，退潮预警是**可识别但尚未兑现的风险提示**。

## 核心信号：龙头断板

最可靠的退潮预警信号之一是**市场中最高辨识度龙头的首次断板**。

### 2024-10-28 案例：双成药业断板

- 双成药业26天23板，是924行情以来的标志性龙头
- 2024-10-28当日未出现在涨停榜
- 复盘明确指出：“这是924以来最重要的信号之一——双成药业可能终于见顶了。”

### 信号特征

| 要素 | 描述 |
|------|------|
| 标的身份 | 市场公认的最高辨识度龙头 |
| 信号类型 | 首次不在涨停榜（非跌停，只是断板） |
| 市场背景 | 当日涨停282家，情绪仍在高潮 |
| 预警时效 | 退潮通常在1-3天内确认 |

## 其他辅助预警信号

同一份复盘中出现的其他弱信号：
- **大唐电信22次炸板**：高连板票极度不稳定
- **华立股份断板后回封**：虽然后来成功回封，但断板本身已暴露分歧

## 与[[概念/退潮日|退潮日]]的关系

退潮预警是退潮日的前置概念。识别退潮预警的价值在于：
- **在高潮期提前防御**：不必等退潮确认才减仓
- **审视持仓质量**：预警期间检验持仓是否有独立逻辑支撑
- **控制仓位激进程度**：从9分满仓降到7分仓位

## 识别框架

```
龙头断板 + 高连板炸板率上升 + 量能未继续放大
→ 退潮预警
→ 检查持仓质量、降低仓位激进程度
→ 等待1-3天确认是否进入退潮期
```
