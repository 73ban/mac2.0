# 退潮假高潮

## 定义

退潮假高潮是退潮期的一种表面繁荣现象：**涨停家数多，但跌停家数也多，且涨停中以杂毛跟风板为主，核心主线仅集中在极少数方向上**。它制造假象让人误以为市场走强，实则是退潮期的核心陷阱。

## 识别标准

| 信号 | 退潮假高潮 | 真正的市场高潮 |
|------|-----------|--------------|
| 涨停家数 | 多（100+） | 多 |
| 跌停家数 | **也多（50+）** | 极少（<10） |
| 主线集中度 | 仅1-2条主线，其余杂毛 | 多主线共振，赚钱效应扩散 |
| 封单质量 | 杂毛封单小，主线封单大 | 普遍封单合理 |
| 成交量 | 缩量或平量 | 放量 |
| 赚钱效应 | 仅限于主线核心票 | 普涨，连杂毛也能赚钱 |

## 2026-05-18 经典案例

```
涨停 102 家 ← 表面繁荣
跌停 53 家  ← 退潮真相

主线：存储芯片（14只首板）+ Token工厂（6只）
其余：大量杂毛涨停，多为一日游
```

核心撕裂：退潮不是所有票都跌，而是**资金从弱票撤退，集中涌入最强逻辑**。这是[[概念/内部缩圈]]的极端体现。

## 最大的陷阱

交易者看到102家涨停，容易产生"市场好了"的错觉 → 追跟风板 → 次日低开割肉。

> ⚠️ 退潮期涨停家数多不是买入信号，是离场信号。它意味着退潮还未结束，资金在做最后的缩圈挣扎。

## 应对纪律

1. **只做主线核心票**：退潮期任何跟风板次日都可能直接低开
2. **看跌停数先行**：跌停数不降至个位数，不轻易做多
3. **等背离收敛**：涨停数/跌停数的背离缩小，才是退潮接近尾声的信号
4. **宁可错过杂毛**：退潮期最大的亏损来源不是空仓，而是在假高潮中追杂毛

## 与[[概念/退潮日]]的关系

退潮假高潮是退潮日的**中期特征**：
- 退潮初期：普跌，容易识别
- 退潮中期：出现假高潮，最容易亏钱 ← 当前阶段
- 退潮末期：缩圈完毕，主线开始独立走强，酝酿下一轮周期
