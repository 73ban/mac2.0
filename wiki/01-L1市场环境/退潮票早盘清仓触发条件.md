# 退潮票早盘清仓触发条件细化

## 竞价阶段（9:15-9:25）

| 竞价表现 | 操作 |
|------|------|
| 跌停竞价 | 9:20前挂跌停价卖出 |
| 竞价跌幅>5% | 9:24挂当前价-2%卖出 |
| 竞价跌幅3-5% | 9:25开盘即清一半 |
| 竞价跌幅<3%但成交量萎缩 | 观察至9:35 |
| 竞价翻红 | 可以等到开盘 |

## 开盘阶段（9:30-10:00）

| 开盘表现 | 操作 |
|------|------|
| 开盘直接跳水>3% | 市价清仓 |
| 开盘横盘不涨 | 10:00前清仓 |
| 开盘反弹但无量 | 反弹就是卖点 |
| 开盘放量拉升>3% | 可以持有观察 |

## 午盘阶段（11:00-11:30）

| 表现 | 操作 |
|------|------|
| 上午未翻红 | 午盘收盘前清仓 |
| 上午翻红但回落 | 清仓 |

## 特殊情况

- 一字跌停：收盘挂跌停价排队，次日继续挂
- 板块集体退潮：不等到个股信号，直接清
- 大盘暴跌（>3%）：叠加退潮直接清

> 核心原则：退潮期不持有任何退潮票过夜
