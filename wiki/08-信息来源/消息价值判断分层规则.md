# 消息价值判断分层规则

更新时间：2026-07-06

## 信息源角色

| 信息源 | 主要作用 | 不能替代 |
|---|---|---|
| 公告 | 最高等级事实确认 | 不能替代市场是否认可 |
| 互动易 | 公司确认/否认市场炒作逻辑 | 普通股东户数、分红、股价问答不入个股卡 |
| 财联社 | 快速新闻和产业催化 | 不能单独证明利润弹性 |
| 韭研公社 | 产业链逻辑、散户研究、预期差发现 | 需要公告/互动易/价格/订单验证 |
| 淘股吧 | 情绪周期、短线模式、游资关注点 | 不能证明公司长期价值 |
| 同花顺/通达信/淘股吧热榜 | 市场关注度和资金拥挤度 | 不能替代公司影响判断 |
| 涨停原因 | 当日资金给出的交易解释 | 不能替代真实业务证据 |

## 处理顺序

1. 保真写 RAW。
2. 判断消息对公司的影响分类。
3. 判断证据等级。
4. 判断短线资金是否可能认可。
5. 判断机构/价值投资是否可能认可。
6. 对持仓票、作战室票、热榜核心票、题材核心票更新个股卡/题材卡。
7. D+1/D+3/D+5 验证短线结果。
8. 中长期跟踪订单、量产、价格、收入和利润兑现。

## 降权规则

- 只有热度、没有公司证据：按短炒型处理。
- 只有观点、没有数据：保留 RAW，不升 wiki 结论。
- 互动易回答与当下炒作逻辑无关：不写入个股卡。
- 题材已经充分发酵但无新增证据：不重复升权。
- 公司明确否认核心逻辑：进入长期风险或反向样本。
