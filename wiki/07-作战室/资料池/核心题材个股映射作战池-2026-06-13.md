# 核心题材个股映射作战池-2026-06-13

## 用途

本页把自动知识卡和高价值个股池压缩成作战室可用的题材-个股映射。

原则：

1. 资料多只说明 WIKI 有沉淀，不等于可以买。
2. 作战室只使用“题材强度 + 个股辨识度 + 盘口确认 + 模式匹配”的组合。
3. 每个题材必须区分核心票、中军、弹性票、补涨票、风险锚。
4. 进入作战室前必须再检查最新市场状态、热门榜、竞价、成交额、板块梯队。

## 作战室使用流程

```text
突发新闻/研报/公告
→ 先查本页对应题材
→ 再查高价值个股资料池
→ 再查独立知识卡和原始RAW
→ 判断 L1 市场环境
→ 判断对应 L4 模式
→ 写入作战室候选
→ 竞价 9:15/9:20/9:25 确认
→ D+1/D+3/D+5 跟踪
```

## 禁止

1. 资料多不等于逻辑硬。
2. 逻辑硬不等于明天能涨。
3. 题材强不等于后排能买。
4. 研报看好不等于短线资金认可。
5. 没有竞价和盘口确认，不允许重仓。
