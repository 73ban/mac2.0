# 盘前盘中盘后检查清单

更新时间：2026-06-18

## 盘前检查

必须完成后才能交易：

```text
[ ] 今日市场状态已判断
[ ] 今日市场预期已判断
[ ] 今日主线候选已列出
[ ] 今日主模式已选择
[ ] 候选票已评分
[ ] 主计划只有 1 个
[ ] 备选计划不超过 2 个
[ ] 每只票都有退出条件
[ ] 禁做清单已写
[ ] 仓位权限已写
[ ] 持仓处理已写
[ ] 竞价确认条件已写
[ ] 数据缺口已标明
```

## 9:15 检查

```text
[ ] 大单一字方向是否符合预期
[ ] 大单一字跌停是否有风险
[ ] 作战室票是否符合预期
[ ] 核心票是否强于板块
[ ] 风险锚是否恶化
[ ] 是否需要降仓
[ ] 若无辅助竞价数据，是否已标注数据缺口
```

## 竞价总检查

```text
[ ] 竞价反馈是否支持主计划继续
[ ] 竞价反馈是否要求计划降级
[ ] 风险锚是否恶化到需要暂停
[ ] 持仓票竞价是否低于底线预期
[ ] 是否需要主动汇报重大变化
```

## 买入前检查

```text
[ ] 是否计划内
[ ] 是否有主模式
[ ] 是否符合模式买入条件
[ ] 是否没有触发禁用条件
[ ] 是否有明确止损或退出条件
[ ] 仓位是否符合权限
[ ] 如果买错，亏损能否接受
```

## 持仓中检查

```text
[ ] 持仓逻辑是否仍成立
[ ] 题材是否仍强
[ ] 个股是否仍是核心
[ ] 风险锚是否恶化
[ ] 是否触发减仓条件
[ ] 是否触发清仓条件
```

## 盘后检查

```text
[ ] 交割单是否已落 RAW
[ ] 今日复盘 RAW 是否完整
[ ] 当前作战室工作页是否按收盘后最小回填清单更新
[ ] 当前持仓决策页是否按收盘后最小回填清单更新
[ ] 今日错误是否记入错误库候选
[ ] 今日样本是否进入跟踪池
[ ] 到期 D+验证项是否已更新
[ ] 明日作战室输入是否已准备
```
