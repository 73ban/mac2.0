# 每日作战室候选票评分自动生成模板

更新时间：2026-07-01

## 用途

本文件用于每天盘前生成：

```text
wiki/07-作战室/YYYY-MM-DD-作战室候选票评分表.md
```

目标不是把所有资料里的股票都塞进来，而是把当天真正可能进入交易计划的 3-8 只票拿出来横向比较，最终只保留 1 只主计划、1-2 只备选、若干观察和明确禁买票。

每日还必须额外生成 15 个训练样本：

```text
raw/03-每日计划/YYYY-MM-DD-市场预期差与重大催化15票.md
```

这 15 票用于训练消息权重判断和市场总结能力，不等同于作战室买入候选。

## 输入来源

每天按以下顺序读取：

1. `wiki/00-总纲/2026-06-18-73交易大脑协同与执行总制度.md`
2. `wiki/10-系统配置/2026-06-18-现行制度总目录.md`
3. `wiki/10-系统配置/作战室生成规则.md`
4. `wiki/04-L4交易模式与执行/交易模式选择器.md`
5. 相关题材与个股档案
6. 当天河马、龙虾写入的 RAW
7. 当天人气热榜、连板天梯、成交额、龙虎榜、竞价辅助快照

没有当天辅助行情和竞价数据时，必须写：

```text
存在数据缺口，候选可先评估，但不能假装完整。
```

## 候选池筛选规则

进入候选池至少满足一项：

1. 当天主线题材的核心票、中军、前排
2. 多平台人气热榜交集
3. 与持仓或主计划直接相关
4. 具备明确模式和退出条件

15 票训练池至少满足一项：

1. 一字板、大单一致、强封板或连板晋级，需要倒查背后逻辑。
2. 重大新闻催化，且能映射到具体个股和板块。
3. 低预期但资金强验证，存在市场最大预期差。
4. 热榜 Top100 快速跃迁，注意力明显扩散。
5. 题材龙头带动后出现补涨、20cm 弹性或容量中军机会。

## 输出要求

最终必须明确：

1. 主计划
2. 备选
3. 观察
4. 删除

并写明：

1. 触发条件
2. 禁止条件
3. 退出条件
4. 是否存在数据缺口

15 票训练池必须写明：

1. 消息级别：重大/一般/弱。
2. 资金验证：一字板/大单一致/涨停/连板/板块扩散/热榜跃迁/暂无。
3. 基本面纯度和短线情绪权重分开打。
4. 预期差来自哪里。
5. 次日验证点是什么。
