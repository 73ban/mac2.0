# 作战室目录说明

更新时间：2026-07-05

本目录用于区分“现行作战室工作页”和“历史作战室资料”。

## 现行入口

左侧导航和日常打开顺序优先使用以下入口：

1. `00-每日工作台.md`
2. `当前作战室工作页.md`
3. 当日 `YYYY-MM-DD-作战总控.md`
4. 当日 `YYYY-MM-DD-作战室候选票评分表.md`
5. 当日 `YYYY-MM-DD-作战室候选验证回看.md`
6. 当日 `YYYY-MM-DD-消息催化统一评分.md`
7. `../06-持仓与资金管理/当前持仓决策.md`
8. `../09-统计与进化/D+验证待回填总览.md`
9. `../05-错误库/`
10. `../04-L4交易模式与执行/20万小资金超短模式总控.md`
11. `../09-统计与进化/YYYY-MM-DD-数据接口运行总控报告.md`

制度和模板入口：

1. `2026-06-18-作战室制度入口.md`
2. `明日作战室模板.md`
3. `每日作战室候选票评分自动生成模板.md`

## 历史资料口径

- 带具体日期的旧盘前、午盘、候选池、旧评分表，默认视为历史参考。
- 这些文件保留样本价值，但不作为当前主计划入口。
- 当前是否能做、做什么、怎么做，以现行制度入口和当前工作页为准。

## 一句话

作战室目录里，先看入口和当前工作页，日期旧的页默认先当历史资料。
