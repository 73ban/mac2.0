# 断板纪律

## 定义

涨停板股票次日未能继续涨停（断板）时的卖出纪律和决策框架。

## 触发条件

1. 昨日涨停，今日开盘未能一字/大幅高开
2. 盘中炸板（涨停打开）
3. 尾盘未能封回涨停

## 纪律规则

| 情形 | 操作 |
|------|------|
| 一字板次日低开 | 竞价/开盘即清 |
| 炸板后15分钟未回封 | 分批卖出 |
| 断板但成交量未放大 | 可观察至午盘 |
| 断板+巨量换手 | 立即清仓 |
| 断板+板块退潮 | 无条件清仓 |

## 纪律背后逻辑

- 涨停是情绪极致，断板意味着情绪转弱
- 连板股的本质是接力游戏，断板意味着接力断裂
- 越早断板，越弱

## 常见违规

- 断板后幻想"洗盘"
- 因为是龙头所以"多拿一天"
- 前一天盈利好所以"给点回撤空间"
