# 快节奏观察模式

> 2级意外扰动触发的标准化操作流程：停止、挂损、等待、限价、平仓——五步循环，日内了结。

## 触发条件

当[[概念/突发事件三级响应框架]]确认进入**2级意外扰动**时启动：
- 突发政策讲话、局部冲突、监管变化等
- ATR(14)突破5日均值2倍 + 5分钟成交量>20日均量300% + 关键价位突破

## 标准操作清单

### 第一步：停止（立即执行）
- [ ] **停止所有新开仓位**——趋势仓位和短线仓位均不得新增
- [ ] 取消所有未成交的市价挂单
- [ ] 暂停执行任何非紧急的交易计划

### 第二步：保护（立即执行）
- [ ] **所有已持短线仓位挂止损单**，止损设在冲击前日内最低点下方1-2个ATR
- [ ] 趋势底仓评估：利润垫是否>5%？是→不移损；否→移损至成本价
- [ ] **趋势底仓不得主动卖出**——除非后续崩塌条件触发

### 第三步：等待（15-30分钟）
- [ ] 离开屏幕或最小化交易界面——防止情绪操作
- [ ] 等待**至少15分钟**让市场消化第一波算法冲击
- [ ] 此期间禁止：刷新闻标题、看社交媒体讨论、修改挂单

### 第四步：确认（流动性恢复后）
- [ ] 检查：价格是否在窄幅区间（如5分钟布林带）盘整至少5分钟？
- [ ] 检查：成交量是否回落至冲击前的70%以下？
- [ ] 检查：ATR是否开始回落（不再加速扩张）？
- [ ] 以上三项全部满足 → 视为可交易窗口打开
- [ ] 任一项不满足 → 继续等待或放弃当日操作

### 第五步：交易（如有信号）
- [ ] **订单类型**：限价单 + 可接受滑点范围，禁止市价单
- [ ] **仓位限制**：T+0交易次数≤平时计划的1/3，单笔风险资本≤1%净资产
- [ ] **入场方式**：在关键支撑/阻力位挂预订单，不追市价
- [ ] **仓位性质**：所有开立仓位标记为"应急短线"，不得加入趋势持仓池

### 第六步：了结（尾盘前）
- [ ] **当日14:50前**，所有应急短线仓位必须全部平仓
- [ ] 不得以任何理由将应急短线仓位"转正"为趋势持仓
- [ ] 检查：是否还有未平仓的应急短线？→ 强制平仓

## 关键禁忌

| 禁忌 | 原因 | 关联认知陷阱 |
|------|------|-------------|
| **禁止追市价** | 流动性枯竭时滑点可高达数个百分点 | [[概念/补涨标的流动性差异]] |
| **禁止"再等等"** | 等待期被压缩=被算法带走的概率大增 | [[概念/修复速度麻醉效应]] |
| **禁止"只是小仓位"** | 小仓累积→尾盘忘平→污染趋势池 | [[概念/可用资金幻觉]] |
| **禁止"今天是特殊的"** | 每次意外都感觉特殊，但模式是反脆弱的 | [[概念/全胜交易日]]前置激励 |
| **禁止夜盘开仓** | 非交易时间决策质量最差 | [[概念/深夜纪律脆弱窗口]] |

## 退出条件

满足任一即退出快节奏观察模式：

- [ ] ATR回落至5日均值1.5倍以下
- [ ] 5分钟成交量回到20日均量150%以下
- [ ] 价格回到冲击前区间内
- [ ] 距离冲击发生已超过2小时且未出现新冲击

退出后：恢复正常交易规则，但当日不可再触发第2次快节奏模式（一周上限2次）。

## 频率上限

一周内触发快节奏观察模式超过2次 → **不是模式问题，是市场本身极度不明** → 缩仓至正常仓位的50%，直到下周重新评估。
