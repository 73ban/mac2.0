# 一致预期目标价

## 定义

**一致预期目标价**：多家券商分析师给出的12个月目标价的均值（或中位数），代表市场共识对一只股票合理价值的集体判断。数据通常由Bloomberg、FactSet或国内通达信TQLEX等平台聚合。

## 核心用途：动态压力位

一致目标价最重要的交易含义不是"股票会涨到这个价格"，而是**当股价逼近目标价时，上行空间被显著压缩**——这构成一个动态压力位/天花板信号。

### 卖点信号：空间耗尽

当剩余空间<5%时，新进场资金面临极低的风险收益比。

**典型案例：天孚通信（2026-05-28）**

| 指标 | 数值 |
|------|------|
| 当前价 | 447.56 |
| 一致目标价 | 464.65 |
| 剩余空间 | **+3.8%** |
| PE (2026E) | 106x |

仅剩3.8%的含义：
- 分析师需要**上调目标价**才能打开新的上行空间
- 在当前目标价框架下，这是**卖点信号，而非买点信号**
- 除非出现新催化迫使分析师集体上调，否则性价比极低

### 中性：空间适中

剩余空间10-20%时，需要结合PE/PEG和催化判断。

**案例：中兴通讯（2026-05-28）** — 38.30→43.48，+13.5%，5家覆盖

### 买点条件：空间充裕

剩余空间>20%、逻辑未被破坏、一致预期在上调趋势中——提供安全边际。

## 局限性（必须认知的偏差）

| 偏差类型 | 描述 | 影响 |
|----------|------|------|
| **滞后性** | 目标价通常基于前次财报，股价可能早已超越 | 可能低估真实价值 |
| **乐观偏差** | 分析师天然存在乐观倾向，目标价系统性偏高 | 实际可达性打折 |
| **静态性** | 不反映突发催化（政策、行业事件） | 上行空间可能被低估 |
| **覆盖不均** | 无机构覆盖的标的=没有目标价 | 无法使用此工具 |
| **群体思维** | 分析师互相参考，目标价趋同 | 一致预期可能集体错误 |

## 进阶用法：关注变动方向

一致预期的**边际变化**往往比绝对值更有信息量：
- **上调**：分析师在追认积极变化 → 动能向上
- **下调**：分析师在追认消极变化 → 动能向下
- **胜宏科技案例**（2026-05-28）：一致预期-7.28%，即使逻辑最硬（英伟达核心供应商），短期业绩压力明确

## 实战规则建议

1. **剩余空间<5%**：高度警惕，默认不新增仓位，考虑减仓
2. **剩余空间5-15%**：需要催化配合才考虑买入
3. **剩余空间>20%**：安全边际存在，可纳入候选
4. **一致预期在下调**：无论空间多大，优先了解下调原因
5. **无机构覆盖**：放弃使用此工具，归入情绪博弈框架
