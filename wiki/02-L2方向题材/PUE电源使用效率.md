# PUE（电源使用效率）

## 1. 什么是PUE

**PUE（Power Usage Effectiveness，电源使用效率）** 是衡量数据中心能效的核心国际指标，定义为**数据中心总能耗与IT设备能耗的比值**。

```
PUE = 数据中心总能耗 / IT设备能耗
```

- **IT设备能耗**：服务器、存储、网络设备等实际承载计算任务的设备所消耗的电力
- **总能耗**：IT设备能耗 + 制冷系统 + 供配电系统 + 照明及其他辅助设施的能耗总和

PUE数值越低，表示用于IT设备之外的能耗越少，能效越高。理论上，PUE最理想值为1.0（所有电力全部用于IT设备），但在现实中，总会有制冷和配电损耗。

### 1.1 关键特征

- **2014年**中国数据中心平均PUE约为**1.65**
- **2023年**中国数据中心平均PUE降至**1.48-1.58**，较2022年的1.54有所下降
- 全球数据中心平均PUE从2007年的**2.5**降至2023年的**1.58**
- 传统风冷数据中心的PUE通常在**1.4-1.5**之间

### 1.2 能耗结构

以PUE=1.5的数据中心为例，能耗分布为：
- IT设备总能耗：**67%**（其中服务器33.5%、存储系统23.5%、网络通信设备10%）
- 制冷系统：**27%**（降PUE的关键突破口）
- 供配电系统：5%
- 照明及其他：1%

**制冷系统是IT设备之外最大的能耗来源**，这是液冷技术能够大幅降低PUE的核心逻辑所在。

## 2. PUE与液冷的关系

### 2.1 物理基础

液冷相较风冷具有根本性的物理优势：
- **导热能力是空气的1000倍以上**，热量传递速度极快
- **比热容是空气的3000倍**，同样体积的液体能吸收远超空气的热量

### 2.2 不同散热技术的PUE范围

| 散热技术 | PUE范围 | 制冷能耗占比 | 备注 |
|---------|--------|-------------|------|
| 传统风冷 | 1.4-1.5 | 30%-40% | 技术上PUE极限值约为1.25 |
| 冷板式液冷 | 1.1-1.2 | 15%-20% | 当前市场主流，兼容性好、改造成本低 |
| 浸没单相液冷 | ~1.1 | ~10% | 散热能力强 |
| 浸没相变液冷（两相） | **1.04-1.1** | <10% | 散热能力最强，适合超高密度 |

**结论**：液冷可以将PUE从风冷的**1.4-1.5**压到**1.04-1.2**，制冷能耗占比从**30-40%**压缩到**10%以内**。液冷替代风冷是AI算力爆发时代数据中心的必然选择。

> **PUE与单机柜功率密度的关系**：单机柜功率在**20kW以下**可采用风冷；超过**20kW**需要液冷技术；**7.5-18kW**区间为风冷/液冷均可的中高密度；**30kW以上**则必须采用液冷。

### 2.3 经济和环境效益

以**10兆瓦**数据中心为例，采用液冷替代风冷：
- **年省电费2000-3000万元**（PUE从1.5降至1.2，电价0.8元/度）
- 回本周期约**1.5-2年**，AI高密场景（GPU集群）仅需**1年左右**
- 全生命周期（20年）可累计节省**近亿元电费**

冷板液冷**初投资已低于风冷系统**，浸没液冷运行约4.5年后TCO低于风冷。在10kW/机架密度下，浸没式液冷与传统风冷的初始投资大致相等；当密度升至20kW或40kW/机架时，液冷分别可节省10%和14%的投资成本。

## 3. 中国PUE政策要求

PUE是**东数西算**战略和绿色数据中心建设的核心考核指标，国家层面已形成**分阶段收紧**的政策体系。

### 3.1 国家层面政策演进

| 时间节点 | 政策要求 | 适用对象 |
|---------|---------|---------|
| **2021年底** | PUE < 1.35 | 新建大型及以上数据中心 |
| **2023年底** | PUE < 1.3 | 新建大型及以上数据中心 |
| **2025年** | PUE 普遍 ≤ 1.5 | 所有存量数据中心 |
| **2025年+** | PUE ≤ 1.3，枢纽节点 ≤ 1.25 | 八大国算枢纽节点新建数据中心 |
| **2026年** | 持续收紧，一线城市更严 | 全国34个省级行政区全覆盖 |

### 3.2 重点地区PUE要求（2026年）

**一线城市（最严）**：
- **北京**：PUE＞1.35开征差别电价，超过1.35按**0.2元/度**加价，＞1.7按**0.5元/度**加价。目标2027年全市年均PUE降至1.35以下
- **上海**：新建数据中心PUE力争≤1.25，智算中心以1.25为硬门槛
- **深圳**：新建大型项目PUE≤1.25

**枢纽节点**：
- **八大算力枢纽节点**要求PUE≤1.3，**十大集群节点**要求≤1.25
- **内蒙古**（和林格尔等）：严控在**1.2以下**
- 实际运行案例：中国移动呼和浩特数据中心PUE<1.3，浩云长盛宁夏基地PUE 1.1-1.2

**其他省份典型**：
- **安徽**（芜湖集群）：PUE<1.25；全省平均2026年底≤1.35
- **山东**：新建大型≤1.3，鼓励浸没液冷
- **河南**：PUE≤1.2补贴0.04元/度，连续2年

### 3.3 政策的硬约束性质

PUE合规已从"鼓励"升级为"强制"：
- 一线城市对**不达标项目限制扩容、征收惩罚性电价**
- **东数西算**将算力从东部热区向西部冷区分流，东部承载推理任务、西部承载训练与存储
- 2026年起，**北京对PUE超标数据中心开征差别电价**，是首个明确执行惩罚性电价的一线城市

## 4. 降PUE的技术路径

### 4.1 核心突破口：制冷系统

制冷系统占数据中心总能耗的**27-43%**，是降低PUE的首要目标。三大招数：

1. **干掉"电老虎"**：用液冷完全取代或大幅减少传统精密空调，将制冷能耗从30-40%压缩到10%以内
2. **让服务器风扇"下岗"**：风冷服务器内部风扇消耗IT设备功耗的10-15%，两相液冷可大幅降速甚至完全关闭服务器风扇
3. **让芯片更"冷静"**：液冷直接冷却CPU/GPU等发热元件，散热效率较风冷提升3-5倍

> 单颗芯片功耗正向**2000瓦**迈进，万卡集群常态化，风冷已触达物理天花板。

### 4.2 全链路节能

除制冷外，PUE优化还涉及：
- **供配电系统**：高压直流供电（HVDC）、模块化UPS、高效率变压器
- **电气损耗**：提升全电力链路设备效率、降低电能传输损耗
- **可再生能源**：绿电交易+绿证+风光储能，台达电子通过氮化镓(GaN)技术实现服务器电源单级效率高达99.2%
- **AI节能**：中国电信通过AI节能系统年均节电**8亿度**，节省电费**5.2亿元**

### 4.3 实际落地案例

| 项目 | PUE | 关键技术 | 节能效果 |
|-----|-----|---------|---------|
| 中国电信临港智算谷 | 1.08（部分区域） | 弹性供电+供冷+优化气流组织 | 接近理想水平 |
| 中国移动呼和浩特智算中心 | <1.15 | 冷板式液冷+余热回收 | 年节电1719万度 |
| 宁畅B7000液冷服务器 | 1.05 | 浸没式液冷 | 相比风冷降耗42% |
| 中国电信青海大数据中心 | <1.2 | 100%清洁能源+自然冷却 | 零碳数据中心 |
| 中兴通讯G5液冷服务器 | 1.1 | 冷板式液冷，液冷占比超80% | 整机功耗降80W |

## 5. 在本知识库中的角色

PUE是理解**AI算力产业链**中[[概念/液冷散热赛道|液冷散热赛道]]价值的核心量化锚点：

### 5.1 作为量化桥梁
PUE连接 **[[概念/风格撕裂|政策驱动]]→[[概念/液冷散热赛道|液冷必然性]]**，为"液冷替代风冷是物理必然"提供数字表达。

### 5.2 作为预期验证信号
[[概念/预期先行业绩滞后]]聚焦于业绩时间差，PUE政策信号补充**需求侧的观测指标**：
- 一线城市惩罚性电价执行力度 = 液冷改造需求加速信号
- 西部枢纽PUE≤1.2落实进度 = 新建液冷渗透率验证信号

### 5.3 作为技术路线分化锚点
PUE要求的收紧速度可能影响液冷内部技术路线选择：
- PUE<1.25 → 冷板式液冷足以达标
- PUE<1.15 → 浸没式液冷相对优势扩大

## 6. 补充研究建议

- **PUE实际运行数据的行业追踪**：工信部、中国信通院每年发布的全国数据中心PUE评估报告，可定期校准液冷市场的渗透速度
- **冷板式vs浸没式液冷的PUE-成本-可靠性三元对比**：英维克/高澜的冷板式方案与曙光数创等浸没式方案，在PUE指标上的实际差距和成本差异需要更详细的量化
- **PUE差别电价的经济冲击分析**：北京2026年起执行的差别电价政策，对存量风冷数据中心的运营成本冲击有多大，对液冷改造市场需求的拉动效应是否需要量化
- **英维克/高澜方案的典型PUE**：这两个液冷标的的方案在实际部署中达到的PUE水平分别是多少？
- **PUE政策执行力度的时间节点**：北京2026年差别电价是否有过渡期？存量改造是否有补贴？
