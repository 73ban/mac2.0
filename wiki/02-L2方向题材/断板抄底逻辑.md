# 断板抄底逻辑

## 定义

**断板抄底逻辑**指在连板股断板当天或次日买入，赌其不会A杀（连续跌停），会有修复反弹的交易行为。其核心假设是：连板股积累的市场辨识度和人气不会在一天内消散，总有资金会来“抄底”，从而推动反弹。

## 核心前提（很少成立）

断板抄底要成功，需要同时满足以下条件：

| 条件 | 说明 |
|------|------|
| 市场处于主升期 | 情绪向上，资金充裕，容错率高 |
| 标的仍是核心主线 | 板块未退潮，龙头地位未失效 |
| 断板原因是市场系统回调 | 非个股基本面或筹码问题 |
| 获利盘已充分释放 | 断板前已有充分的换手和洗盘 |

**罕见的是：这四个条件很少同时成立。更常见的是：市场退潮+标的不在主线+获利盘集中释放。**

## 为什么它会诱惑你

1. **锚定效应**：你被之前的连板高度所锚定——“9板的票怎么可能直接A杀？”——而忽略了断板本身就宣告了那段历史的终结。
2. **高赔率幻觉**：如果修复，可能赚10-20%；如果A杀，可能亏10-20%。看起来是1:1的赔率，但实际上A杀的概率远大于修复。**真实赔率并不是你想象的那样**。
3. **正反馈循环的自我破坏**：之前一次正确止盈（如国民技术+10%）让你高估了自己的判断力，误以为断板抄底也是“有把握的交易”。

## 10/15中粮资本案例

```yaml
标的: 中粮资本（002423）
前状态: 9连板T字（封单6.15亿）
断板日: 2024-10-15
操作: 分三笔买入3,300股，均价约19.40元，约64,000元
市场环境: 反弹一日游，情绪从6分降至4分，五连阴→一阳→再阴
主线状态: 中粮已不在任何主线（新主线为化工/军工/化债）
```

这是教科书级的“不该抄的断板”：
- ❌ 市场环境：退潮
- ❌ 主线状态：被缩出
- ❌ 获利盘压力：9板积累，未充分释放
- ❌ 连板高度：9板后跌回3板，梯队极度压缩

> **判断**：中粮不是不会修复——而是即使修复，你也大概率扛不过A杀的初期。

## 什么时候（几乎）可行？

断板抄底仅在一种情况下具有正期望值：**市场主升期中，龙头因系统性恐慌（非自身问题）短暂断板，且次日迅速修复**。这需要满足：

- 市场整体仍在主升
- 该股仍是最高辨识度
- 断板日成交量没有放大到极端
- 次日早盘有资金进场迹象

**双成药业（002693）是一个对比案例**：14连板后断板2天→重封2连板，显示了“主升期断板修复”的成功模式。但中粮的断板发生在市场从9→3连板压缩的过程中，环境完全不同。

## 操作铁律

```
断板日不买，这是铁律。
如果认定修复逻辑成立，等放量阳线确认后再进场——宁可错过第一根阳线，绝不在断板日接飞刀。
```
