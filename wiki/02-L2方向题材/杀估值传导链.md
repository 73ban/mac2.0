# 杀估值传导链

## 定义

**杀估值传导链**，是指美债收益率上行→DCF折现率提高→高估值成长股股价重定价→通过映射关系传导至A股对应板块的四层因果链条。这是理解"经济数据好→股市反而跌"这一反直觉现象的核心框架。

## 四层传导模型

```
Layer 1: 利率锚变动
  非农超预期/通胀高企 → 加息预期上升 → 10年期美债收益率跳升
  ↓
Layer 2: 估值重定价（DCF机制）
  折现率↑ → 未来现金流现值↓ → 高久期资产（科技成长股）首当其冲
  杀估值≠杀业绩：公司基本面未变，但市场愿意为其未来利润支付的价格降低了
  ↓
Layer 3: 拥挤交易去杠杆
  高估值+高拥挤+叙事翻转 → 所有人同时冲向出口 → 踩踏
  费城半导体-10.26%是这一层级的典型表现
  ↓
Layer 4: A股映射传导
  美股芯片崩塌 → A股半导体/AI低开 → 情绪传导
  关键区分：真冲击（产业逻辑受损） vs 假冲击（情绪性估值压缩）
```

## 真冲击 vs 假冲击

这是本概念最核心的区分，直接决定操作应对：

| 维度 | 真冲击 | 假冲击 |
|------|--------|--------|
| 触发源 | 产业逻辑本身受损（需求崩塌/技术路线被颠覆） | 情绪性估值压缩（利率上行/风险偏好下降） |
| A股影响 | 产业趋势终结，应清仓离场 | 短期波动，低开可能是加仓窗口 |
| 修复速度 | 持续恶化，不修复 | 1-2日内修复≥50%低开幅度 |
| 经典案例 | 2000年互联网泡沫（需求证伪） | 6/5美股暴跌→Rubin产业链（产业趋势完好） |
| 判别标准 | 产业链核心订单/资本开支是否改变 | 台积电路线图是否推迟？英伟达GPU是否滞销？ |

## 6/5 黑色星期五的完整传导链

```
非农17.2万(预期8.5万) ［Layer 1: 利率锚变动］
  → 年底加息概率60%→98%
  → 10Y美债收益率→4.54%
  ↓
芯片股年涨40%+，前瞻PE透支严重 ［Layer 2: 估值重定价］
  → DCF折现率上升 → 高估值无法维持
  → 博通Q3指引不及whisper number → 叙事裂痕
  ↓
机构多头集中在科技7巨头，9周连涨 ［Layer 3: 拥挤去杠杆］
  → 费半-10.26%，单日蒸发1.2万亿美元
  ↓
周一A股半导体/AI大概率低开 ［Layer 4: A股映射］
  待验证：Rubin/MLCC/玻璃基板是"真冲击"还是"假冲击"？
```

## 核心洞察

**AI叙事本身未破**（博通积压订单超900亿美元），但**估值已经透支了太多未来**。

这意味着：
- 对美股的冲击是真实的——它需要时间来消化过高的估值
- 对A股Rubin产业链的冲击可能是虚假的——台积电的路线图不会因为英伟达股价跌6%而改变
- MLCC的逻辑（VR200用量）独立于英伟达股价，英伟达股价下跌不等于VR200用量减少

## 判别SOP

面对外部暴跌，对每个A股持仓方向问三个问题：

1. **美股对应标的跌的是什么？** — 杀估值（股价贵了）还是杀逻辑（产业需求没了）？
2. **A股方向的产业逻辑是否依赖美股估值？** — Rubin进度 vs 英伟达股价，是不同的东西
3. **修复窗口判断** — 如果周一低开后周二修复≥50%，基本确认为假冲击

## 与现有框架的关系

- **vs [[概念/强预期弱现实]]**：杀估值属于"强预期（AI叙事）遭遇利率现实（折现率上升）"，是预期与现实的另一种背离形式
- **vs [[概念/风格撕裂]]**：杀估值过程中，资金从拥挤科技向价值/防御切换（道指-1.35% vs 纳指-4.18%），形成风格撕裂
- **vs [[概念/结构性牛市]]**：外部杀估值传导是对[[结构性牛市]]"外部冲击抵抗力"维度的终极压力测试
