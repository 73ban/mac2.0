# 内部缩圈

## 定义

在一个板块/主题内部，资金从"宽泛受益"的后排标向"核心收益"的前排标集中的过程。是主线分化的核心机制。

## 识别信号

| 阶段 | 特征 |
|------|------|
| 全面爆发 | 板块所有票一起涨，涨停家数>10 |
| 开始分化 | 龙头继续涨停，后排冲高回落 |
| 内部缩圈 | 只有1-2只龙头在涨，后排跌 |
| 龙头独舞 | 龙头涨停但板块已无跟风 |
| 退潮 | 龙头也撑不住了 |

## 操作策略

| 阶段 | 操作 |
|------|------|
| 全面爆发 | 可以追后排（但要快进快出）|
| 开始分化 | 只买龙头和次龙头 |
| 内部缩圈 | 只持有龙头，清后排 |
| 龙头独舞 | 减仓龙头，准备清仓 |
| 退潮 | 空仓 |

## 核心逻辑

- 资金总量有限 → 先涨所有的 → 发现龙头 → 资金集中到龙头 → 后排被抽血
- 缩圈是市场效率的体现：资金只给最确定的
- 缩圈后期是最后一个安全卖点

## 案例

- CPO/光通信缩圈：中际旭创（龙头）vs 后排光模块
- MLCC 缩圈：风华高科/火炬电子 分化
