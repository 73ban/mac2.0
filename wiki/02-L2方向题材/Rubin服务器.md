# Rubin服务器

## 定义

NVIDIA继Blackwell Ultra之后的下一代AI服务器平台，采用台积电N3P+N5B芯粒和NVL144互联架构，2026年8月全面量产。**PCB板价值量同比增幅233%**，是本轮AI服务器迭代中增量最大的赛道。

## 核心参数

| 参数 | 详情 |
|------|------|
| 制程 | 台积电N3P + N5B芯粒 |
| 互联架构 | NVL144（联结144块GPU） |
| 性能提升 | 较Blackwell Ultra提升3.3倍 |
| 推理成本 | 单Token降至1/10 |
| PCB价值量 | 同比+233%（最大增量） |
| 量产时间 | 2026年8月全面量产 |

Rubin PCB是"强预期弱现实"的经典案例：
- **预期极强**：价值量+233%、性能提升3.3倍、NVIDIA路线图确定性高

- **典型模式**：第一波预期炒作（5/18-5/26 +50%）→ 利好兑现/筹码分散 → 回调 → 等待量产催化
