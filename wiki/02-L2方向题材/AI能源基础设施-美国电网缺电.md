# AI能源基础设施：美国电网缺电与A股映射

## 核心逻辑

```
AI数据中心 → 电力需求暴增 → 美国电网不堪重负 → 2026年出现电力缺口
→ 燃气轮机订单暴增(GEV排期4.91年) → 涡轮叶片(最瓶颈环节) → 应流股份
→ 电网扩容投资2027年达434亿美元 → 电网设备出口 → 思源电气/特变电工
→ 核电/SMR/储能 → 长期替代方案 → 东方电气/宁德时代
```
