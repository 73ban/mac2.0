# 震荡无方向一日游

## 定义

市场处于震荡期、无明确主线时，任何热点题材的持续性不超过一天的现象。

## 特征

- 大盘指数横盘震荡（日振幅<1.5%）
- 成交量萎缩（两市成交<8000亿）
- 无明确主线板块（没有一个板块连续3天涨幅居前）
- 热点轮动速度快（今天涨的明天就跌）

## 一日游机制

- 无方向 → 资金不敢长期持有 → 只能做一日游
- 题材多但都不新鲜 → 每个题材只能炒半天
- 没有增量资金 → 存量博弈互割

## 应对策略

| 策略 | 操作 |
|------|------|
| 空仓 | 最佳策略，等主线出现 |
| 轻仓试错 | 单票≤5%，总仓≤15% |
| 只做龙头 | 不做跟风（跟风一日游更短）|
| 当日进出 | 不隔夜，尾盘必清 |
| T+0 | 有底仓的做T，不开新仓 |

## 典型一日游陷阱

- 上午涨5%追进去 → 下午跌回 → 次日低开
- 看到消息追热点 → 追在高点 → 当天套牢
- "这次不一样" → 每次都一样
