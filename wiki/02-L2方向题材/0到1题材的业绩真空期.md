# 0→1题材的业绩真空期

## 定义

**0→1题材的业绩真空期**：当一个产业/技术从概念阶段进入研发试产阶段时，距离真正的业绩贡献可能长达数季度甚至跨年度。在这段"真空期"里，股价完全由预期和情绪驱动，没有业绩作为锚定——炒的是空间想象，不是业绩兑现。

## 核心特征

| 特征 | 说明 |
|------|------|
| 时间跨度长 | 从研发试点到批量供货通常需12-18个月 |
| 纯预期驱动 | 没有EPS、PE、订单数据可参照 |
| 波动极大 | 情绪来去无常，极易暴涨暴跌 |
| 证伪风险 | 技术路线可能变更，量产可能延迟 |
| 适合做波段 | 不适合死扛，需要根据催化节奏灵活进退 |

## 案例：CoWoS新技术（玻璃基板+SiC）

| 时间节点 | 事件 | 状态 |
|------|------|------|
| 2026年6月 | 嘉义产线建成 | 研发试点阶段 |
| 2026下半年 | 研发试点 | **业绩零贡献——纯炒预期** |
| 2027年 | 渗透率提升至20-30% | 才开始有业绩贡献 |

**判断**：至少还有12个月以上的业绩真空期。期间A股相关标的（沃格光电、天岳先进、帝尔激光等）的任何上涨都是预期驱动，随时可能因情绪逆转而大幅回撤。

## 与[[概念/强预期弱现实]]的关系

0→1题材的业绩真空期是**强预期弱现实**在时间维度上的极端形态：
- 强预期弱现实：预期与现实的差距（可以缩小）
- 0→1真空期：现实为零（尚无业绩可兑现），预期是全部驱动力
- 两者叠加时，风险呈指数级放大

## 与[[概念/事件驱动时间衰减]]的关系

在0→1真空期里，事件驱动时间衰减曲线更陡峭——因为没有业绩锚定，一次催化事件的推动力消退后，股价更容易跌回原点。

## 交易启示

1. **仓位控制**：0→1真空期题材的仓位应远低于有业绩支撑的标的
2. **止损更紧**：没有基本面锚定，技术破位必须止损，不能"死扛等业绩兑现"
3. **催化驱动**：买入和卖出都应基于催化事件的时间窗口，而非估值判断
4. **波段思维**：在催化前布局，利好兑现时出货，而非长期持有
5. **区分阶段**：Rubin PCB（8月量产后开始有业绩）vs CoWoS（2027年才有业绩）——两者风险等级完全不同

## 与用户持仓的关系

用户当前持仓中，鹏鼎控股属于Rubin链（8月量产→业绩可期），但同有科技/华能蒙电/东百集团/金螳螂与三大产业事件无关（关联度⭐），这些持仓缺乏产业框架支撑，处于更被动的"无催化剂等待"状态——这比0→1真空期更被动，因为连炒作的想象空间都没有。
