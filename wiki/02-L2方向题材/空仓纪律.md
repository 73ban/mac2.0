# 空仓纪律

## 定义

**空仓不是不做交易，是把仓位留给确定性更高的方向。**

空仓是仓位管理的最高形式——它意味着：
- 识别当前市场不存在高确定性机会
- 拒绝为"出手而出手"的心理补偿
- 将资金和能力保留到真正值得出击的时刻
- 这是纪律，不是恐惧，更不是懒惰

## 为什么空仓难

| 障碍 | 来源 | 后果 |
|------|------|------|
| **怕错过** | [[概念/贪婪驱动的纪律撤销]] | 无方向也硬上 |
| **出手压力** | 一日空仓后"必须出手"的心理补偿 | 降低标准买入 |
| **持仓惯性** | 习惯重仓→空仓的"不适感" | 为保持仓位而买错票 |
| **证明自己** | 用交易数量证明能力 | 从交易员退化为赌徒 |

## 6/2实战：空仓不是没能力

### 证据

- 6/1持有两个小仓位试错票（粤电力A、豫能控股）
- 6/2全部纪律清仓：
  - 09:31 粤电力A：弱转强失败→纪律止损
  - 14:44 豫能控股：非龙一→主动放弃
- 收盘空仓（0%），净盈亏-261
- 用户自评：**"想把仓位放在明天"**

### 定性

这不是被市场打出来（被动空仓），而是主动清仓后选择等待（主动空仓）。与6/1的恐惧驱动卖出有本质区别：
- 6/1：恐慌→急于脱手→不加判断地卖
- 6/2：判断弱转强/龙头地位→冷静清仓→空仓等待确定性

## 触发空仓的判定条件

| 条件 | 说明 |
|------|------|
| 混沌期/退潮期 | 无明确主线方向 |
| 持仓票丧失龙头地位 | 不是最主动→清仓 |
| 弱转强失败 | 烂板票次日开盘走弱→纪律止损 |
| 四个潜在方向都不满足开仓条件 | 没有一个PK胜出的方向 |
| **确定性不高时** | 唯一需要判断的核心指标 |

## 明日出手压力管理（开放警惕）

> "6/3如果四个方向都不满足条件，还能空仓吗？"

**空仓一日后最容易犯的错**：产生"必须出手"的心理补偿 → 降低标准买入 → 回到[[概念/割肉-追涨循环]]。

**纪律防线**：
1. 四个方向有且仅有一个满足条件 → 出手
2. 多个方向同时满足 → PK选择最强
3. **零个方向满足 → 继续空仓** ← 这是对纪律的真正考验
