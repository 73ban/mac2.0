# 因果链重建

## 定义

指通过补全缺失的关键时间点数据，将被中断的交易事件序列重新连接为可理解的因果顺序。在复盘系统中，这是降低“数据断裂”严重性的核心行动。

## 在 3 月初的应用

### 原始断裂

3 月初的复盘链存在严重缺失：

```text
3/1 → [空白] → 3/2 → [空白] → ... → 3/6（切换至存储）
```

整个离场→切换的动机和节奏无法被连贯解释。

### 3/2 补缺后的部分重建

```text
3/1 → [仍空白] → 📍3/2 → 减持中钨+清仓北方稀土 → ... → 3/6 → 清仓中钨+买入兆易创新
```

### 恢复的部分

- 离场是分步进行的（3/2 已开始减仓，非 3/6 突然切换）
- 从有色+稀土向存储的腾挪是渐进决策

### 仍断裂的部分

- 北方稀土何时买入、为何买入 —— 3/1 仍然空白
- 从 3/1 → 3/2 的持仓变动全貌仍不完全

## 方法论价值

因果链重建的层级：

1. **事实层**（客观交易记录） — 3/2 已补全 ✅
2. **动机层**（为什么买/卖） — 仍部分缺失 ⚠️
3. **环境层**（当时大盘/板块状态） — 可后验推断

只有当三层都完整时，因果链才真正闭环。当前 3 月初仅修复了事实层的关键拼图，动机层和环境层仍需要 3/1 买入记录作为锚。
