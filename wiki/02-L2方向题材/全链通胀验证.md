# 全链通胀验证

## 定义

通过跨环节供应链的产能/价格/订单信号**交叉验证**全行业需求景气度的框架。当上游关键供应商产能持续满载且积极扩产时，即使中下游环节暂未完全受益，也能确认需求端真实强劲——"全链通胀"意味着**需求是真实的，不是局部炒作**。

## 液冷赛道的具体映射

### 上游锚点：AVC（冷板制造商）
- **产能状态**：供不应求，越南扩产中
- **信号意义**：冷板订单爆满 → AI 液冷需求真实 → GPU 出货在加速

### 中游交叉验证：英维克/高澜（系统集成商）
- **合同负债变化**：若上升，则集成订单在跟进
- **毛利率变化**：若下降，则验证冷板利润被上游锁定

### 下游交叉验证：算力中心
- **PUE 要求**：政策驱动 + 液冷刚需
- **上架率**：算力中心上架率高 → 液冷系统持续需求

## 验证逻辑

```
上游（冷板）满产扩产
    +
中游（集成）合同负债上升
    +
下游（算力中心）上架率高
    ↓
结论：液冷赛道景气度确认，不是炒作
```

但需要注意：
- 上游满产 ≠ 中游利润好（冷板利润可能被上游锁定）
- 景气度确认的是**行业需求端**，不是**具体标的利润端**

## 操作意义

1. **AVC 产能利用率** = 液冷需求景气度的最直接先行指标
2. 当 AVC 满产扩产时 → 持有英维克的行业景气度支撑成立
3. 当 AVC 产能利用率下滑 → 液冷需求预警，重新评估持仓
4. **景气度 ≠ 利润确定性**：英维克 Q2 需单独验证毛利率变化

## 可迁移性

这个框架可推广至 AI 算力链其他环节：
- **PCB 环节**：上游覆铜板厂（生益科技）产能 + 中游 PCB 厂（深南电路）订单
- **光模块环节**：上游光芯片（Lumentum）产能 + 中游模块厂（中际旭创/光迅）订单
- **MLCC 环节**：上游粉体（村田/国瓷）产能 + 中游元器件（三环）订单
