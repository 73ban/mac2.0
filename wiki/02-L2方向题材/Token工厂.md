# Token 工厂

> 为 AI 模型提供推理算力、按 Token 产出量计费的算力基础设施。

## 定义
Token 工厂是指部署大规模 AI 算力集群（如 GPU/昇腾超节点），专门承接 AI 模型的推理任务，按照输出的 Token 数量向客户收费的商业模式。它不同于训练算力（卖 GPU 机时）或数据中心托管，核心差异在于计费单位是 **“Token 产出”**，而非硬件占用时长。

## 与 DRAM 的区别
Token 工厂做的是 **AI 推理算力**（模型推理产生 Token），而长鑫存储等做的是 **DRAM 存储芯片**。两者是完全不同的赛道，不应混淆。这也是 [[概念/题材无实质映射]] 的典型应用场景。

## 典型标的
- **弘信电子**：市场公认的 Token 工厂龙头，已签约 51.6 亿，华为昇腾 384 超节点集群，政府签约。详见 [[股票/弘信电子]]
- **平治信息**：跟风者，仅口头披露算力规模，无实质背书。详见 [[股票/平治信息]]

## 辨识真龙头
真正的 Token 工厂标的应具备 **“政府签约 + 生态巨头硬件 + 运营商渠道”三重背书**。仅有口头算力规划或 GPU 采购计划、缺乏生态验证的，应视为题材跟风，不应给予龙头溢价。详见 [[概念/龙头确认的三重背书]]。

## 风险
- **订单兑现率**：签约金额不等于营业收入，需跟踪实际结算进度。弘信的 24.8% 兑现率就是典型例子。
- **技术迭代**：国产算力芯片（昇腾/燧原）与英伟达 CUDA 生态的差距可能影响长期竞争力。
- **主题持续性**：Token 工厂是否具备持续性题材特征，还是短期事件驱动，需持续验证。
