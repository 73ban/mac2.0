# MLCC涨价映射

## 概念定义

MLCC涨价映射，是指**全球MLCC龙头村田宣布2026年7月1日涨价**后，通过产业链传导和资金对标，对A股MLCC板块产生的系统性影响。核心逻辑链：

```
村田涨价(7/1生效) → 全球MLCC供给趋紧 → 三星电机/国巨跟进涨价
    → A股MLCC估值对标抬升 → 风华高科/洁美科技受益
```

与普通概念不同，此概念带有**明确的时效窗口**（涨价生效前仍有3周博弈期）和**强外围依赖**（村田/三星电机走势直接影响A股MLCC方向）。
