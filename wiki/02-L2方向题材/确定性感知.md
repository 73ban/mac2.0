# 确定性感知

## 定义
在盘中能清晰区分标的确定性强弱、预判板块和个股走势、知道何时追高/低吸的综合认知能力。是交易从"炒股"升级到"概率博弈"的核心标志。

## 六个能力维度

| 能力维度 | 状态 |
|---------|------|
| 感知确定性强弱 | ✅ 能区分 |
| 板块走势预判 | ✅ 有框架 |
| 龙头个股走势预判 | ✅ 有框架 |
| 何时追高 vs 何时低吸 | ✅ 清晰 |
| 何时追高会亏 | ✅ 有认知 |
| 何时低吸安全边际大 | ✅ 有认知 |

**六个问题能同时清晰回答 → 已进入概率博弈层面。**

## 发展阶段定位

| 阶段 | 状态 | 特征 |
|------|------|------|
| ① | 小白 | 什么都不懂 |
| ② | 迷茫 | 懂一点，模棱两可 |
| ③ | 事后看懂 | 高手解释→能懂；盘中→确定性不足 |
| ④ | **盘中看懂** | 有明确买卖理由 + 确定性衡量标准 |
| ⑤ | 本能反应 | 观察者与交易者分离（[[概念/观察者交易者分离]]）|

**当前：④级初期** — 从③到④的跃迁已发生。

## 从③到④的跃迁标志
- **③级**：别人说了你能懂，但盘中无法独立做出高确定性判断
- **④级**：自己盘中能看到，有明确买卖理由 + 确定性衡量标准
- **核心突破**：从"事后解释"到"盘中独立判断"的质变

## 与交易纪律的关系
- 确定性感知成形 → 用规则替代恐惧驱动（与[[概念/恐惧驱动的纪律执行]]互补）
- 但需警惕：确定性感知 ≠ 过度自信 → 仍需刚性规则约束

## 验证建议
本评估基于6/3单日数据，建议后续3-5个交易日持续追踪六个能力维度的稳定性。
