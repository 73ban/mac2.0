# 结构化止损

## 定义

"现金→填仓→不涨→割肉"的标准化四步操作链。不是一笔大亏，而是多笔工业化的小亏（每笔2-8%本金），循环周期约24小时。连续多日全部清仓亏损，使亏损呈现系统化、结构化的特征。同时，被界定为[[概念/注意力税]]的微观支付载体——每次操作都是一次注意力税的缴纳。

## 核心特征

| 特征 | 描述 |
|------|------|
| 循环周期 | ≈24小时（今天买→明天割） |
| 单笔亏损 | 2-13%本金 |
| 触发条件 | 账户有可用现金 |
| 操作序列 | 卖出→立即买入（无冷却期） |
| 累积效应 | 多笔小亏合计吞噬全部已实现损益 |

**这不是偶然——这是标准化流程。** 你把现金投入新票→新票不涨→你割肉→你用新现金买更新的票。标准工时：24小时。

## 作为注意力税的支付载体

5天38次操作 = 38次注意力税缴纳。

每次买入→持有焦虑→不涨→割肉的循环，不仅是资金在损耗，注意力也在持续被消耗。结构化止损的每一笔操作都是注意力税的一次缴纳——越是频繁操作，缴纳的注意力税越多，用于核心持仓管理的注意力越少。这直接通向[[概念/注意义务分散]]。

## 防范规则

1. **买入前预写持有逻辑**：如果说不清为什么持有超过1天，就不买
2. **卖出后30分钟冷却**：任何卖出后30分钟内禁止新开仓
3. **每日开仓上限**：单日新开仓不超过2笔
4. **连续亏损熔断**：连续2笔清仓亏损后，当日禁止新开仓
