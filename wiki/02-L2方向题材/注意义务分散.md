# 注意义务分散

## 定义

当账户持有核心龙头（确定性强、仓位重、浮盈大）时，交易者未能围绕核心持仓建立"集中持仓+减少操作"的纪律框架，而是在其他随机标的上进行高频交易，导致注意力被非核心标的消耗、核心持仓管理被忽视，且非核心操作全面亏损的现象。

## 核心特征

| 特征 | 描述 |
|------|------|
| 持有核心龙头 | 如华映科技10连板，浮盈+8.6万 |
| 高频非核心操作 | 持有期间29次其他买卖操作 |
| 非核心操作全亏 | 所有非核心操作的合计结果为亏损 |
| 核心被忽视 | 没有为核心持仓建立操作纪律和风控框架 |
| 浮盈掩盖 | 核心的浮盈在心理上"覆盖"了非核心的亏损 |

## 定义性案例：华映科技持有期（11/4-11/7）

| 数据点 | 数值 |
|--------|------|
| 核心持仓 | 华映科技8,300股 |
| 持有天数 | 8天（11/4至今） |
| 期间非核心操作次数 | 29次 |
| 非核心操作结果 | 合计亏损 |
| 核心浮盈 | +86,320元（心理锚） |
| 实亏被掩盖 | 11月累计已实现亏损-21,260元 |

**诊断原文**："你不是在'以华映为核心持仓'——你是在'拥有华映的同时，用其余资金进行高频率的随机交易'。华映是你偶然踩到的正确节奏，但你并没有围绕它构建一个'集中持仓+减少操作'的纪律框架。"

## 与[[概念/注意力税]]的关系

注意义务分散是[[概念/注意力税]]的一种具体表现形式：
- **注意力税**：衡量非主线标的占用时间/资金比例的隐性成本
- **注意义务分散**：注意力税在核心持仓场景下的特化——核心存在时，注意力应更集中，但实际更分散

## 产生机制

1. **浮盈麻醉**：核心持仓的浮盈制造"已经赚了，随便玩玩"的错觉
2. **机会饥渴**：看到其他方向涨了想参与，害怕错过
3. **现金不耐受**：卖出后有现金就必须立刻投入某处
4. **缺乏核心持仓纪律**：没有"持有核心期间减少操作"的硬性规则

## 防范措施

1. **核心持仓模式**：当某标的占账户浮盈>30%时，触发"核心持仓模式"
   - 非核心标的单日操作≤1次
   - 非核心标的仓位上限≤总仓位10%
   - 新开仓必须有独立逻辑且不与核心方向重叠
2. **每周注意力审计**：统计核心vs非核心花费的时间/操作次数/盈亏
3. **收盘自问**：今天为核心持仓的管理做了什么？非核心操作是否在消耗注意力？
