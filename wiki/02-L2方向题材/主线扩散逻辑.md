# 主线扩散逻辑

## 定义

**主线扩散逻辑**指一个主线题材（如 Rubin-VR200 → PCB）从核心标的向外围标的扩散过程。用户在执行时的核心信念是：既然逻辑强，板块内的其他标的也会被资金发掘。

## 逻辑本身并非错误

题材扩散是市场运行的正常现象：
- **第一阶段**：核心标的上涨（受益最直接、逻辑最硬的龙头）
- **第二阶段**：扩散至相关配套标的（供应链的延伸）
- **第三阶段**：沾边标的补涨（逻辑最弱、最边缘）

问题在于：**同一逻辑在不同时机执行，结果天差地别。**

## 有效扩散 vs 高潮后追涨

| 维度 | 有效扩散阶段 | 高潮后追涨（陷阱） |
|------|------------|-----------------|
| **市场认知度** | 板块首次被市场认知，仍有分歧 | 集体涨停潮，预期高度一致 |
| **情绪温度** | 贪婪初期，FOMO 刚萌芽 | 贪婪极致，FOMO 顶点 |
| **核心标的状态** | 高位横盘或温和调整 | 已连续大涨，有超买迹象 |
| **外围标的起点** | 低位/未启动 | 已跟涨但未涨停 |
| **参与方式** | 低吸/回调买入 | 追涨/追高买入 |
| **安全边际** | 有，因为位置低 | 无，因为已在高位 |
| **结果概率** | 可能吃到第二轮 | 大概率站岗吃回调 |

## 有效扩散的判断条件

| 条件 | 满足？ |
|------|--------|
| 板块首次被市场认知（非二次/三次高潮） | ✅ |
| 核心标的未出现明显分歧/天量换手 | ✅ |
| 外围标的处于相对低位 | ✅ |
| 市场情绪尚未过热（无集体涨停潮） | ✅ |
| 参与方式为回调/低吸，而非追涨停 | ✅ |

## 何时不能执行扩散逻辑

| 条件（任一触发即禁止） | 原因 |
|------------------------|------|
| 板块已出现≥3只涨停 | 高潮信号，次日分歧概率大 |
| 板块指数单日涨幅 >5% | 情绪过热 |
| 核心标的已连板/天量换手 | 龙头分歧在即 |
| 成交量异常放大 | 高潮筹码分散 |

## 本案例的教训

[[2026-05-22 当日委托]] 中，用户的核心驱动即是“主线扩散逻辑”——Rubin-VR200 → PCB → 多只 PCB 标的。但执行在 **5/21 高潮次日**，属于典型的高潮后追涨。

> **同一个逻辑，延迟一天执行 = 完全不同的结果。**
