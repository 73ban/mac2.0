# 龙头替代

## 定义

在同一题材中，出现一个更强的新标的（更早封板、更高封单、更强分时），**抢走了原龙头的市场关注度和资金**，导致原龙头迅速失去情绪溢价和龙头溢价。这是[[概念/退潮日]]的微观版本——不是题材退潮，而是**个股地位被换**。

## 触发信号

| 信号 | 含义 |
|------|------|
| 新标的走出更强分时（反包板/一字板） | 资金用脚投票，选择新欢 |
| 原龙头多次拉升但无法封板 | 市场不愿意给溢价——资金已被新标的锁死 |
| 原龙头量能萎缩或炸板无力回封 | 持币者转向新标的，持筹者急于兑现 |

## 经典案例：合百集团 vs 达实智能（2026-05-25）

### 竞争背景
- 题材：长鑫存储IPO概念
- 合百集团：5/22打板进，是当时资金选出的龙头
- 达实智能：5/25早盘冲涨停、走出**反包2板** —— 更强、更坚决

### 合百集团的死亡信号
1. 早盘低开 → 开局不利
2. 红盘震荡期间，多次尝试拉升但始终没有形成市场合力
3. 资金不愿意封板 → 因为更强的达实智能已经把情绪资金吸走
4. **达实智能封板那一刻，合百集团的龙头地位正式丧失**

### 用户的操作：清仓兑现25万

用户正确识别了信号，果断执行。这是**龙头替代 → 立即认输**的教科书级正面案例。

### 反面教训（如果没走）

如果继续持有合百集团：
- 资金继续流向达实智能 → 合百缩量阴跌
- 龙头溢价消失 → 跌回题材平均水平
- 从盈利变亏损的概率极高

## 与相关概念的区别

| 概念 | 范围 | 本质 |
|------|------|------|
| **龙头替代** | 同一题材内的个股竞争 | 地位被换，资金搬家 |
| [[概念/退潮日]] | 整个题材/市场的情绪退潮 | 水退了，所有船都降 |
| [[概念/内部缩圈]] | 大题材内部分化，资金向最核心标的集中 | 范围内的优胜劣汰 |

## 操作规则（建议）

1. 同一题材中出现更强的新标的 → 立即评估原龙头的相对地位
2. 原龙头出现“多次拉升无法封板” → 大概率已被替代，执行减仓/清仓
3. 新标的封板确认 → 原龙头不再有任何持有理由，除非是核心逻辑票（非情绪票）
4. **认输要快** —— 龙头溢价的衰减速度远超想象
