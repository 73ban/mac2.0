# 风格切换先行预警信号体系

连接信号判断与仓位转换的操作清单。当以下四层信号出现共振时，风格切换的概率将极大升高。

## 综合信号体系

**【顶层】主导资金切换**
- [ ] 北向资金连续多日净流入标的属性转变（如：从食品饮料转至电子）。
- [ ] 公募发行结构逆转（如：价值型基金发行占比反超成长型）。
- [ ] 成长ETF vs 价值ETF的净申购剪刀差持续扩大/收窄。

**【中观】多维量化共振**
- [ ] **高PE/低PE比值** 连续3月同向运行。
- [ ] **ROE预期** 出现方向性转变（参考[[概念/ROE-TTM高位拐点]]）。
- [ ] **信用利差** 走阔，配合货币活化指标下降，预示防御型大盘价值占优。

**【微观】涨停板结构恶化**
- [ ] 当日涨停股中，市值50亿以下的微盘股占比持续**低于 30%**，标志着资金审美从小盘弹性向大盘稳定性转移。
- [ ] 散户新开户数连续环比下滑。

**【事件驱动】催化确认**
- [ ] 重大政策、技术突破或外部宏观冲击发生，并与上述资金、量化流向形成同一方向合力。

## 操作转化
当上述四层中，顶层资金流且中观量化指标（至少一个）发出警示，叠加微观结构异化时，应将此信号作为削减旧风格仓位并转向新风格的决策触发点。
