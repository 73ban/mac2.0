# 外部冲击型风险

## 定义

**外部冲击型风险**，是指由境外市场暴跌、地缘政治事件、政策突变等**A股体系外因素**引发的市场急跌风险。与内生性退潮不同，外部冲击的触发源不在A股自身的情绪周期内，但会通过传导机制快速影响A股开盘和盘中走势。

## 传导机制

```
境外利空事件（美股暴跌/地缘/政策突变）
  → 市场情绪恐慌传导（VIX飙升/北向资金预期流出）
  → A股开盘低开（幅度与事件烈度正相关）
  → 情绪打分骤降（可能触发退潮铁律）
  → 结构性分化：真冲击（产业逻辑受损）vs 假冲击（情绪性估值压缩）
  → 修复观察期：次日修复力度决定是否演化为持续退潮
```

## 两档强度案例

### 弱档：日韩熔断（2026-06-05周四）
- 触发源：日韩股市熔断
- A股反应：结构性低开，但Rubin产业链方向快速修复
- 结果：验证[[概念/结构性牛市]]的"外部冲击抵抗力"维度
- 定性：**假冲击**——外部波动未破坏A股独立主线的产业逻辑

### 强档：美股黑色星期五（2026-06-05周五）
- 触发源：费城半导体-10.26%，纳斯达克-4.18%
- A股预期：周一大概率低开，半导体/AI方向直接承压
- 待验证：Rubin产业链/MLCC/玻璃基板能否再次展现独立韧性？
- 关键区分：英伟达跌6%是股价波动，台积电路线图/Rubin进度未受影响

## 识别三问

1. **触发源是否影响A股主线产业逻辑？**
   - 是 → 真冲击，需重新评估持仓
   - 否 → 假冲击，低开可能是加仓窗口

2. **北向资金反应烈度？**
   - 大幅流出（>50亿）→ 外部冲击正在内化为A股系统性压力
   - 小幅流出/流入 → 外资认为冲击可控

3. **独立主线次日修复速度？**
   - 日内修复≥50% → 结构性牛市框架依然有效
   - 连续两日不修复 → 可能触发[[策略/退潮期不开新仓]]

## 应对SOP

| 情境 | 操作 |
|------|------|
| 周一低开≤2%，独立主线抗跌 | 维持原计划，观察修复力度 |
| 周一低开2-4%，独立主线跟跌 | 执行退潮铁律，开盘30分钟减仓至半仓 |
| 周一低开≥4%，全线崩塌 | 竞价出/开盘即出，三层清仓SOP（见[[概念/清仓执行力]]） |
| 周二修复≥50%低开幅度 | 考虑重新建仓，优先独立逻辑方向 |
| 周二继续下跌 | 确认退潮，全面空仓等待 |

## 与现有框架的关系

- **vs [[概念/退潮日]]**：退潮日是内生情绪周期的自然衰竭，外部冲击是外生事件的突然扰动。但外部冲击可以**触发**退潮——关键看修复速度。
- **vs [[策略/退潮期不开新仓]]**：外部冲击导致的情绪急跌（≤4分），**是否等同自然退潮？** → 建议：若独立主线同步暴跌且次日不修复，则等同退潮；若独立主线抗跌，则不触发铁律。
- **vs [[概念/结构性牛市]]**：为"外部冲击抵抗力"维度提供了系统化的评估框架。详见该页面五维判定更新。
