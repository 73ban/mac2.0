# 异动价卖出规则

## 定义

当股票价格触及异动停牌阈值（涨停/跌停、偏离值阈值等）附近时，提前制定的卖出策略。

## 涨停板异动

| 情形 | 规则 |
|------|------|
| 连续涨停接近3日异动阈值 | 第3个涨停应减仓50% |
| 连续涨停接近10日严重异动 | 减仓至10%以下 |
| 封单骤缩（<原来20%） | 立即清仓 |
| 一字板转T字板 | 减仓50% |

## 跌停板异动

| 情形 | 规则 |
|------|------|
| 一字跌停无法卖出 | 次日竞价挂跌停价 |
| 跌停板反复打开 | 打开就是卖点 |
| 连续跌停 | 第三个跌停开盘清（如果可能） |

## 偏离值异动

| 阈值 | 操作 |
|------|------|
| 偏离值接近3日20%/30% | 减仓至30%以下 |
| 偏离值接近10日100% | 全部清仓 |
| 偏离值接近30日200% | 全部清仓 |
