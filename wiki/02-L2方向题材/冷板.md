# 冷板 (Cold Plate)

## 定义

冷板（Cold Plate/液冷板）是**冷板式液冷系统**中最核心的散热部件，直接贴合在GPU、CPU、ASIC等发热芯片表面，通过内部的精密微通道结构，将芯片产生的高密度热量高效传递给流经的冷却液，实现芯片级精准散热。

## 工作原理

1. **热传导**：芯片热量通过导热界面材料（TIM）传递至冷板底板
2. **对流散热**：冷却液在冷板内部的微通道（针翅/蛇形流道）中流动，通过对流带走热量
3. **系统级匹配**：冷板内部的流阻设计需与CDU泵力、Manifold分配、快接头口径形成精确匹配，任何环节的不匹配都会降低整体热管理效率

## 在液冷系统中的角色

冷板是[[概念/冷板式液冷系统架构]]中与芯片直接接触的**第一级散热界面**，其性能直接决定了整个系统的散热上限。它与其他组件的关系：

```
芯片 → 冷板 → 快接头 → Manifold → CDU → 外冷源
```

冷板的微通道设计（流阻、流量分配）决定了系统运行的总体流量需求和泵功耗。

## 英维克冷板产品线

[[股票/英维克-002837]]的冷板产品覆盖三大类：

- **GPU冷板模块**：独家供应英伟达GB300服务器，支持高功耗GPU定制化流道
- **ASIC冷板模块**：针对各类专用集成电路的全套方案
- **光模块冷板**：支持400G/800G/1.6T光模块冷却，全生命周期热性能保障

## 单品天花板与全链条破局

冷板本身属于[[概念/中间厂商流动性陷阱|中间厂商流动性陷阱]]的高风险环节：
- 单品市场规模相对有限
- 利润易受上游芯片设计方和下游服务器集成商挤压

破局路径：**全链条整合**——将冷板作为[[概念/全链条液冷解决方案]]的关键拼图，通过配套自家CDU、Manifold、快接头形成系统级锁定，从"卖部件"升级为"卖方案"。
