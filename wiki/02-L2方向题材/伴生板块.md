# 伴生板块

## 核心定义

**伴生板块**是与市场主线同时活跃但并非主线的平行板块，与主线交替轮动，具有独立逻辑和持续资金关注。

## 当前案例：电力/电算协同

5月下旬的观察：
- **主线**：AI算力链（英伟达Rubin架构驱动，PCB/MLCC/光模块）
- **伴生板块**：电力及电算协同（华电能源、华电辽能、豫能控股、大唐发电轮番创新高）
- **逻辑**：AI算力需求 → 电力/电算基础设施需求
- **特点**：非主线，但资金持续活跃，板块有独立空间

## 伴生板块的识别特征

| 特征 | 说明 |
|------|------|
| 与主线同期活跃 | 不是主线结束后才启动的补涨 |
| 有独立产业逻辑 | 非单纯的资金溢出，有自身催化 |
| 轮动节奏交错 | 主线强时不抢戏，主线分歧时表现 |
| 个股轮番创新高 | 非一日游，有持续性 |

## 操作注意事项

1. **不能只盯主线**：伴生板块可能提供独立交易机会
2. **节奏判断是关键**：主线分歧时伴生板块可能有表现窗口
3. **选股逻辑独立**：不因伴生而降低选股标准
4. **仓位分配**：主仓位在主线上，伴生板块作为补充
