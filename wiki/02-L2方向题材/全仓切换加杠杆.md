# 全仓切换 + 加杠杆

## 定义

在一个交易日内：
1. **清仓多只持仓**（回收流动性）
2. **同日将回收资金全部投入新标的**（全仓切换）
3. **额外使用融资**（加杠杆，超出回收金额）

三步合成的高风险操作模式。

## 2026-05-11 样本

```
卖出回收：圣阳+中超+麦格米特 = 560,132
买入投入：中国长城+通富微电 = 595,342（含融资35,210）
结果：全仓切换完成 + 净多3.5万杠杆
```

## 模式特征

| 特征 | 信号类型 | 解读 |
|------|----------|------|
| 一日三清 | 🟡 中性偏正 | 卖出果断，不拖泥带水 |
| 当日全打回 | 🔴 危险 | “钱不能闲着” → 急躁驱动 |
| 另加融资 | 🔴🔴 高危 | 回收不够用？不对——是贪婪放大 |
| 分笔追高建仓 | 🔴 危险 | 非计划性低吸，而是怕错过 |

## 判断框架

此模式的危险等级取决于两个核心变量：

| 变量 | 低危组合 | 高危组合 |
|------|----------|----------|
| 卖出盈亏 | 三笔均止盈 | 有止损（割肉） |
| 市场环境 | 主升/修复初期 | 退潮/高潮末期 |
| 买入计划性 | 预设目标，等待多日 | 盘中临时起意 |
