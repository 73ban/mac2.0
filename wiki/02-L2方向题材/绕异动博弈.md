# 绕异动博弈

## 定义
利用10个交易日内收盘价涨幅偏离值接近异动公告触发阈值（通常为100%或200%）时，提前布局博弈触发后市场的加速拉升，以触发即为卖点。

## 核心机制
- **计算公式**：异动偏离值 ≈ ∑（个股每日涨幅 - 对应指数每日涨幅），连续10个交易日累计。
- **触发阈值**：
  - 偏离值达到 ±100% → 深交所/上交所异动公告（第一次）
  - 偏离值达到 ±200% → 严重异动公告（可能停牌核查）

## 博弈逻辑
1. **接近触发**：当偏离值达到85%-95%区间时，市场预期即将触发异动。
2. **抢筹预期**：部分资金预期触发异动公告后吸引关注度、产生加速拉升。
3. **触发即卖点**：公告触发日（通常为次日），即为阶段性高点——"利好出尽是利空"。

## 典型案例：大唐发电（2026-05-18）

| 指标 | 数值 |
|------|------|
| 10日偏离值 | 89.83% |
| 触发阈值 | 100% |
| 触发所需涨幅 | +5.04%（达到8.33元） |
| 涨停价 | 8.72元 |

## 风险提示
- ⚠️ 偏离值高≠必然触发——需要股价上涨配合
- ⚠️ 触发后方向不确定——可能拉高出货而非继续拉升
- ⚠️ 退潮日下确定性降低——需有止损预案

## 计划情景（大唐发电）
| 情景 | 操作 |
|------|------|
| 早盘抢筹→触发异动 | 封板即卖点，不贪 |
| 水下震荡未触发 | 低吸做T降低成本 |
| 京能崩盘+板块崩溃 | 无条件止损 |
