# 题材生命周期跟踪模板

更新时间：2026-06-13

## 用途

这个模板用于判断一个题材处在启动、主升、分歧、修复、高潮还是退潮。题材状态决定仓位，不允许只因为资料多就重仓。


## 生命周期阶段

| 阶段 | 典型特征 | 可用模式 | 仓位 |
|---|---|---|---|
| 启动 | 新催化出现，首批个股异动，市场还未充分认知 | 首板新题材试错、新东西站出来、一字定方向扩散 | 轻仓试错 |
| 确认 | 龙头/中军/前排出现，多平台热榜共振，板块有宽度 | 强板块前排半路、竞价弱转强、主线核心低吸 | 标准仓 |
| 主升 | 核心不断新高，分歧后能修复，板块梯队完整 | 题材主升核心持股、主线核心低吸、强趋容量抱团 | 标准仓到可重仓 |
| 分歧 | 后排掉队，核心震荡，资金开始分化 | 良性分歧修复、趋势回踩均线、核心抗跌穿越 | 降仓观察 |
| 修复 | 核心率先修复，风险锚止跌，资金重新回流 | 冰点恐慌修复、竞价弱转强、旧主线回流 | 轻仓到标准仓 |
| 高潮 | 大面积一致，后排补涨，媒体和散户关注度极高 | 持股为主，少开新仓 | 禁止追后排 |
| 退潮 | 龙头补跌，高标 A 杀，后排大面，修复失败 | 风险锚减仓、计划外交易禁止 | 空仓到极轻仓 |

## 10 分制题材评分

| 维度 | 分数 | 判断 |
|---|---:|---|
| 催化强度 | 0-10 | 政策、产业、业绩、订单、价格、事件强度 |
| 可信度 | 0-10 | S1/S2 高，S3 中，S4 低 |
| 板块宽度 | 0-10 | 涨停数、上涨家数、成交额、热榜覆盖 |
| 龙头高度 | 0-10 | 连板高度、趋势新高、辨识度 |
| 中军强度 | 0-10 | 容量票是否持续走强 |
| 分歧修复 | 0-10 | 分歧后是否快速修复 |
| 风险状态 | 0-10 | 风险越高扣分越多 |

总分参考：
```text
8-10：主升或强确认，可进入作战室重点。
6-7：可参与但不重仓。
4-5：观察。
0-3：不做。
```

## 禁止

1. 题材资料多，但盘面不强，不能重仓。
2. 龙头补跌时，不做后排补涨。
3. 高潮后不能用启动期打法。
4. 退潮期不能把反抽当主升。
