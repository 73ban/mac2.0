# 知识星球高置信 A 股主线池

## 结论

这页只保留和你小资金交易有关的 A 股高置信标的。它不是买入清单，而是盘前作战池。买入仍必须满足盘面主动性、L4模式、仓位权限和退出条件。

## AI算力 / PCB / 覆铜板 / 光通信

这是当前资料密度最高、逻辑最硬的主线。优先级高于泛 AI 应用。

| 分层 | 标的 | 判断 |
|---|---|---|
| 龙头/中军候选 | 中际旭创、新易盛、天孚通信 | 光模块强度锚，决定光通信主线是否继续。 |
| 龙头/中军候选 | 胜宏科技、沪电股份、生益科技、深南电路 | AI PCB/覆铜板核心，知识星球资料密度高，逻辑硬度高。 |
| 中军验证 | 工业富联、浪潮信息、中科曙光 | AI服务器/算力基础设施容量锚。 |
| 核心跟踪 | 光迅科技、剑桥科技、太辰光、长飞光纤、亨通光电 | 光通信扩散方向，强度弱于三大光模块核心时不能当龙头。 |
| 上游材料 | 联瑞新材、东材科技、华正新材、圣泉集团、国瓷材料 | 受益高端材料升级，适合看预期差和低位补涨。 |
| 禁止 | 后排蹭 AI、无订单无验证的小票 | 只当跟风，不能重仓。 |

## 半导体 / 国产算力 / 材料设备

逻辑硬，但分支多，容易选错。必须用盘面强度筛选。

| 分层 | 标的 | 判断 |
|---|---|---|
| 国产算力核心 | 寒武纪、海光信息 | 弹性和辨识度高，但估值和波动风险大。 |
| 制造设备中军 | 中芯国际、北方华创、中微公司、拓荆科技、芯源微 | 国产替代中军，适合验证行业趋势。 |
| 封测/先进封装 | 长电科技、通富微电、华天科技、盛合晶微 | AI算力扩散链，需看订单和资金认可。 |
| 测试/材料 | 长川科技、斯达半导、国瓷材料、中船特气、和远气体 | 有催化，但需要更强盘面确认。 |
| 禁止 | 单纯“国产替代”口号票 | 没有涨价、订单、客户验证，不给重仓权限。 |

## 机器人

长期空间大，但短线硬度弱于 AI 硬件。只在新增量产/订单/客户验证时提高优先级。

| 分层 | 标的 | 判断 |
|---|---|---|
| 核心零部件 | 绿的谐波、鸣志电器、五洲新春、中大力德 | 机器人链条弹性核心。 |
| 容量/中军 | 三花智控、拓普集团、汇川技术 | 机构容量票，适合看趋势强度。 |
| 应用扩散 | 机器人、埃斯顿、柯力传感 | 有题材属性，但需看板块主动性。 |
| 禁止 | 没有量产节点的纯故事票 | 不能因为长期空间直接重仓。 |

## 商业航天 / 低空

事件弹性强，但映射容易发散。必须等事件催化和核心股主动。

| 分层 | 标的 | 判断 |
|---|---|---|
| 事件核心 | 中国卫星、航天电子、航天发展 | 商业航天/卫星互联网辨识度较高。 |
| 低空联动 | 宗申动力、万丰奥威、信维通信 | 低空经济/卫星链映射，必须看盘口强度。 |
| 弹性观察 | 通易航天、晨曦航空、中天火箭 | 弹性强，流动性和波动约束更高。 |
| 禁止 | SpaceX 传闻后排发散 | 只做观察，不给重仓权限。 |

## 资源链

涨价和供需缺口资料较多，但容易高潮。你适合等分歧，不适合一致追高。

| 分层 | 标的 | 判断 |
|---|---|---|
| 铜钼中军 | 洛阳钼业、金钼股份、铜陵有色、北方铜业 | 资源链强度锚。 |
| 锂链观察 | 天齐锂业、赣锋锂业、华盛锂电、永杉锂业 | 锂价反转资料较多，但需要价格与盘面双确认。 |
| 小金属 | 中钨高新、章源钨业、厦门钨业 | 涨价弹性强，注意高潮风险。 |
| 禁止 | 涨价高潮后追后排 | 只看核心，不做后排扩散。 |

## 电力液冷

属于 AI 算力二阶扩散。只有当 AI 主线继续强时，它才有持续性。

| 分层 | 标的 | 判断 |
|---|---|---|
| 液冷核心 | 英维克、高澜股份、申菱环境 | 数据中心液冷方向。 |
| 电源/电力 | 麦格米特、中恒电气、科士达、科华数据、阳光电源 | AI数据中心电力约束扩散。 |
| 禁止 | AI主线退潮时补涨追高 | 主线不强，二阶扩散更容易坑人。 |

## AI应用

资料不少，但交易优先级低于 AI 硬件。只做事件驱动和低位补涨。

| 分层 | 标的 | 判断 |
|---|---|---|
| 中军 | 科大讯飞、金山办公 | 商业化验证相对清晰。 |
| 弹性 | 昆仑万维、万兴科技、拓尔思、中文在线 | 事件弹性强，但持续性不稳定。 |
| 禁止 | 发布会后追高应用后排 | 没有收入验证，不给重仓权限。 |

## 明日使用规则

- 每天只从这页选 1 条主线，最多 3 个观察股。
- 如果 AI硬件、PCB、光通信同时强，优先看它们，不要分散到泛题材。
- 题材强但个股弱，不买。
- 个股强但题材弱，只能小仓观察，不能重仓。
- 一致高潮不追，分歧承接强才考虑。
