# 集中式买入

**定义**：在单一交易日或极短时间内，将账户的绝大部分资金（通常包括自有资金及融资额度）集中投向单一标的的行为。

## 核心特征
1. **高仓位比**：单一标的占当日总买入金额的80%以上。
2. **融资加杠杆**：经常伴随使用融资工具，以放大预期收益。
3. **多次交易**：通常不是一笔买入，而是通过多次、分批操作集中持仓。
4. **心理驱动**：背后往往是对该标的极强的主观信心（可能已超出客观分析范畴），或受[[概念/锚定效应]]影响。

## 实证案例
- **2026-05-22 合百集团**：当日11笔买入，占买入总额约89%，并使用了[[融资]]。

## 风险：集中暴击亏损的前置条件
[[概念/集中式买入]]是[[概念/集中暴击亏损模式]]的直接前奏。
- **关系链**：过度自信 → 集中式买入（含融资） → 走势不及预期 → 亏损金额因仓位和杠杆而放大 → 情绪恶化导致后续决策扭曲（如[[概念/割肉-追涨循环]]或[[概念/协同亏损]]）。

## 应对策略
- 强制仓位上限：单票不超过总资产_x_%。
- 禁止逆势加仓：若股价未按预期方向运行，禁止为摊薄成本而追加买入。
- 融资约束：对高集中度标的，禁止或严格限制使用融资额度。
