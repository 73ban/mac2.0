# 价格战与毛利率侵蚀

## 概念定义

**价格战与毛利率侵蚀**指当一个赛道被市场广泛认知为“高增长”后，新进入者涌入并以低价策略争夺份额，导致行业整体毛利率受压，即使龙头公司也难以独善其身的风险模式。

## 核心机制

```
市场共识"高壁垒" → 吸引新进入者 → 实际门槛低于预期
  → 新进入者以成本价抢单 → 龙头被迫跟进降价
  → 行业毛利率集体下滑 → 利润增速远低于营收增速
  → 估值杀（增速预期下修）
```

## 液冷赛道的具体信号

| 信号层级 | 具体表现 | 来源 |
|---|---|---|
| **红色预警（已触发）** | 英维克2026Q1净利骤降82%（汇兑+坏账+打样成本） | 时代财经 |
| **黄色预警** | 冷板依赖CNC加工、快接头靠精密车削，制造门槛低于市场预期 | 国际液冷产业大会 |
| **结构隐忧** | 新进入者以成本价抢单，行业可能从“高技术壁垒”滑向“同质化价格战” | 雪球深度 |

## 识别框架

| 识别信号 | 含义 | 应对 |
|---|---|---|
| **龙头营收增、净利降** | 收入结构恶化，定价权松动 | 警惕，降低仓位 |
| **行业渗透率加速，毛利率反而下滑** | 竞争加剧正在侵蚀红利 | 回避，减仓 |
| **新进入者数量激增** | 门槛低于预期，壁垒不成立 | 重新评估赛道逻辑 |
| **龙头被迫参战** | 价格战从边缘蔓延至核心 | 系统性卖出 |

## 对液冷赛道的意义

- **英维克Q1净利骤降是一个早到的预警信号**：虽然被定性为[[概念/单季扰动vs方向性转变|单季扰动]]（汇兑、坏账、打样），但深层次暴露了冷板制造门槛低于预期的结构性风险
- **如果价格战成真**：所有液冷标的的估值模型将面临系统性下修，尤其是当前107xPE的情绪溢价标的
- **与[[概念/抱团崩塌四条件]]的关联**：价格战是“内部结构性恶化”的微观表现，当龙头毛利率系统性下滑时，即使拥挤度、资金流向尚可，也应启动减仓预案

## 可复现性

该模式可迁移至：
- **MLCC赛道**（风华高科）：扩产潮后供给过剩风险
- **光模块赛道**：技术迭代加速，旧产品毛利率快速下滑
- **PCB赛道**：高端PCB壁垒较高，但中低端同样存在同质化竞争风险

## 防护策略

1. **毛利率是比营收更敏感的先行指标**：营收增长可被新订单粉饰，毛利率下滑是定价权松动的直接证据
2. **区分“量增利增”与“量增利不增”**：前者是高景气，后者是红海前兆
3. **龙头Q1信号≠方向性确认**：但必须纳入监控，等待Q2数据确认
4. **赛道“高壁垒”叙事需要持续验证**：警惕市场共识与产业现实之间的落差
