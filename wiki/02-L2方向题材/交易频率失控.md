# 交易频率失控

## 定义

单月内交易次数远超正常水平，日均交易>3次的状态。

## 11月量化数据

| 指标 | 数值 |
|------|------|
| 交易天数 | 19天 |
| 总交易次数 | **103次** |
| 日均交易 | **5.4次** |
| 月净盈亏 | -19,248元 |

→ 日均5.4次，意味着几乎每个小时都在买卖。

## 频率如何放大亏损

- **成本累积**：交易越频繁，印花税和佣金吞噬利润越多
- **决策疲劳**：高频交易导致每一笔的决策质量下降
- **冲淡盈利**：暴赚一天无法覆盖其余18天的持续性亏损
- **退潮日放大**：在93%个股暴跌日仍进行6次交易——频率在退潮日特别致命

## 核心问题

不是选股能力不够——粤桂股份是全场最高连板龙。根源在于：
- **无法在退潮日空仓**
- **无法控制交易频率**
- 将交易频率当作弥补亏损的工具（割肉→追涨→再割→再追），反而加速了亏损
