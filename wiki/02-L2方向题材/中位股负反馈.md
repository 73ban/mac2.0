# 中位股负反馈

## 定义

在退潮日或市场情绪急剧恶化时，**连板中断的中位股直接跌停**，将亏钱效应以断崖式方式向整个中位梯队乃至全市场扩散的现象。

## 核心特征

| 特征 | 描述 |
|------|------|
| 触发节点 | 情绪评分≤3分，退潮日确认 |
| 直接表现 | 昨日连板股今日断板后直奔跌停 |
| 扩散机制 | 一只跌停 → 同高度其他股恐慌 → 板块内跟风跌停 |
| 中位定义 | 通常指3-5板高度，既不是龙头也不是首板 |

## 2026-06-08 案例

| 个股 | 昨日状态 | 今日状态 | 反馈强度 |
|------|------|------|:---:|
| 天洋新材 | 4连板 | **跌停** | 高位断板 |
| 利通电子 | 连板 | **2连跌停**（封单13.4亿） | 极度恶劣 |
| 嘉美包装 | 连板 | **2连跌停** | 极度恶劣 |
| 金时科技 | 连板 | **2连跌停** | 极度恶劣 |
| 连板高度 | 4板 | **压缩至2板** | 系统性断档 |

## 危害机制

1. **流动性冻结**：跌停封单巨大（如利通电子13.4亿），持有者无法出逃
2. **情绪传染**：一只中位股跌停 → 资金恐慌 → 同梯队抛售 → 扩散至板块
3. **高度压制**：连板高度被压缩，短线投机生态崩溃

## 识别与应对

### 预警信号
- 连板晋级率突然降至30%以下
- 昨日涨停股今日大面积低开
- 中位股断板后无法拉回均线

### 应对策略
- **一旦确认中位股负反馈，立即停止中位股的接力交易**
- 降低整体仓位，优先处理持仓中的中位股
- 不因"跌多了"而试图抄底——负反馈环境下无底可言
