# 烂板龙头

## 定义

**烂板龙头** 指虽然最终封死涨停板，但盘中反复炸板、封板强度极弱的龙头股。

## 识别特征

| 特征 | 说明 |
|------|------|
| 最终封板 | 收盘时确实是涨停板 |
| 盘中炸板 | 打开涨停的次数多（可多达数十甚至上百次） |
| 封板时间 | 往往封板时间晚，尾盘才勉强封住 |
| 成交量 | 放巨量，换手率畸高 |
| 筹码结构 | 资金分歧极大，卖方力量与买方力量几乎均衡 |

## 案例：粤电力A（2026-06-01）

- 连板高度：4板（当时市场最高之一）
- 炸板次数：**181次**
- 成交金额：15.19亿
- 用户评价：勉强封住
- 次日预判：大概率大幅低开甚至核按钮

**核心矛盾**：粤电力A是电力板块的最后一面旗，名义上是“龙头”，但实际上已是强弩之末。买在4板、买在炸板181次的这一天，意味着如果次日低开，几乎没有翻盘空间。

## 烂板龙头的次日规律

- **低开概率极高**：炸板181次意味着多头力量已消耗殆尽，次日卖方占优
- **高开是极小概率**：需要隔夜重大利好刺激
- **操作建议**：持有烂板龙头的次日，竞价即清仓是最优解（参见 [[策略/退潮票早盘清仓]]）

## 烂板 vs 硬板

| 特征 | 硬板 | 烂板 |
|------|------|------|
| 封板时间 | 早盘快速封 | 尾盘勉强封 |
| 盘中炸板 | 几乎不打开 | 反复打开多次 |
| 成交量 | 缩量更好 | 放量巨量 |
| 次日预期 | 高开/一字概率大 | 低开/核按钮概率大 |
| 参与时机 | T日打板 | 不参与，或T+1深水低吸 |

## 在退潮期的特殊意义

退潮期的烂板龙头，往往是板块最后的挣扎。如华电能源5板断板、粤电力A 4板烂板同时出现，几乎可以确定电力主线正在退出舞台。此时持有烂板龙头，不是“握住了龙头”，而是“握住了最后一棒”。
