# 组合效应恶化

## 定义

**组合效应（Mix Effect）** 是PVM毛利率分解中的三大效应之一，衡量**产品/客户/业务结构变化对综合毛利率的影响**：

- **正效应**：高毛利产品占比上升 → 综合毛利率提升
- **负效应（恶化）**：高毛利产品占比下降、低毛利产品占比上升 → 综合毛利率被拖累

## 为什么是关键信号

组合效应持续为负，是**毛利率结构性下行最直接的量化指征**。原因在于：

1. **价格效应**和**成本效应**可能受宏观因素干扰（涨价/降价周期），难以区分周期性与结构性
2. **组合效应反映的是客户的选择**——客户在买什么、不买什么，这比成本变化更本质
3. 高毛利产品占比持续下降 = **产品竞争力的不可逆丧失**，是结构性变化的铁证

## 判断逻辑

```
组合效应状态          解读
─────────────────────────────────
正且持续             产品升级进行中，竞争力强
正但转为负            高毛利产品遇到天花板，需关注
负（偶发）            可能是新产品导入期，暂时性
负且持续（≥3季）     🔴 结构性下行信号确认
```

## 在AI产业链中的典型情景

| 环节 | 正常状态 | 组合效应恶化的表现 |
|------|---------|-----------------|
| 液冷散热 | 高毛利冷板方案占比提升 | 标准化微利组装业务占比扩大 |
| 光模块 | 1.6T新品放量（高毛利） | 800G老产品占比上升、1.6T未放量 |
| PCB | 高阶AI板（高毛利）占比提升 | 普通板占比提升，高阶板增长停滞 |
| CCL | 高端覆铜板放量 | 低端走量，两头受压 |
