# 亏损加仓（摊平陷阱）

## 定义

在持仓亏损时继续买入同一标的，试图摊低持仓成本的行为。是散户最常见、最致命的错误之一。

## 为什么是陷阱

| 错觉 | 现实 |
|------|------|
| 摊低成本 → 更容易回本 | 如果继续跌，亏更多 |
| 股票便宜了 → 应该多买 | 便宜可能因为基本面真出问题 |
| 我研究过 → 它最终会涨回来 | 研究≠确定性 |
| 不买就亏了 → 害怕错过反弹 | FOMO驱动的错误决策 |

## 唯一合法的加仓条件

1. 最初买入就是分批建仓计划的一部分（有预案）
2. 加仓价 > 首次买入价（趋势确认加仓，非摊平）
3. 仓位不超过上限
4. 基本面没有恶化

## 典型案例

- 韶能股份：下跌中不断加仓→重仓深套→最大单票亏损
- 5月灾难：多只股票亏损加仓→全线崩溃

## 替代方案

| 想做 | 实际应该做 |
|------|------|
| 摊低成本 | 止损等待更好买点 |
| 补仓 | 等它涨回来再买（确定性更高）|
| 不想亏 | 接受亏损，保护剩余资金 |
