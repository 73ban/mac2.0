# 胜利后全仓再押

## 定义

**胜利后全仓再押（Rebet-It-All Syndrome）** 是 [[概念/贪婪驱动的纪律撤销]] 的一个具体子模式，指交易者在单笔大幅盈利后，立即将全部资金（包括本金+利润）重新全仓投入新标的，不留任何现金缓冲区的行为。

## 触发条件

| 条件 | 说明 |
|------|------|
| 前置事件 | 单笔交易实现显著盈利（如+15%以上或绝对金额可观） |
| 心理状态 | 过度自信、对市场风险的感知降低 |
| 行为表现 | 清仓后立即（同一天内）以同等或更大仓位追入新标的 |
| 仓位特征 | 无现金保留，100%资金再投入 |
| 环境特征 | 常见于退潮初期——盈利来自退潮前的持仓，再投入时环境已恶化 |

## 经典案例：2024-11-08

### 时间线

| 步骤 | 操作 | 金额 |
|------|------|------|
| 1 | 华映科技清仓（+14,443元） | +70,052 |
| 2 | 花园生物清仓（+567元） | +41,349 |
| 3 | 中信证券清仓（+2,645元） | +60,768 |
| 现金状态 | 收回现金 | **约172,125元** |
| 4 | 融资买入中国软件 | **-84,998元（8.5万）** |
| 5 | 证券买入晨曦航空 | **-60,840元（6.1万）** |
| 6 | 证券买入晟楠科技 | **-29,173元（2.9万）** |
| 结果 | 总仓位 | **约17.5万（≈收回现金的101%）** |

### 问题分析

1. **全仓再押**：收回17.2万，立即投入17.5万（甚至超过收回额）
2. **手法激进**：两只重仓票均在涨停板上追入
3. **环境错配**：多龙头同步断板的退潮信号被忽视
4. **融资放大**：中国软件使用融资买入，进一步放大风险

## 与相关概念的关系

### 属于 [[概念/贪婪驱动的纪律撤销]] 的变体

| 概念 | 触发源 | 行为 |
|------|--------|------|
| 贪婪驱动的纪律撤销 | 市场涨势、赚钱效应 | 放宽止损、加仓超出计划 |
| **胜利后全仓再押** | **单笔自身的大幅盈利** | 立即全仓追入新标的，不保留现金 |

二者的核心区别：
- 贪婪驱动的纪律撤销：被市场氛围驱动
- 胜利后全仓再押：被自身胜利驱动（"赚了就是对的，继续这样干"）

### 可能触发 [[概念/正反馈循环的自我破坏]]

如果中国软件/晨曦航空在退潮中亏损，会形成完整的循环：
```
华映赚1.4万 → 过度自信 → 全仓再押 → 退潮亏损 
→ 痛苦 + 后悔 → 恐惧驱动下一次交易 → ...
```

## 防范规则

| 规则 | 说明 |
|------|------|
| **盈利后冷静期** | 单笔大赚后至少保持50%现金过夜，24小时内不新增 |
| **退潮日禁止加仓** | 多龙头断板日 = 只卖不买 |
| **限制单日仓位转换率** | 清仓收回的现金，同一天再投入不超过50% |
