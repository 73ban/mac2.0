# 赚钱后的仓位急剧放大

## 定义

在一笔大幅盈利（通常是单笔 +10% 以上或金额显著）后，将盈利甚至更多资金迅速再投入到新标的中，导致总仓位和风险暴露急剧上升的行为模式。

## 特征

1. **仓位跳跃**：单日仓位从正常水平（如 3-5 万）跃升至 15 万以上
2. **资金零留存**：赚的钱不仅没拿出来，还追加了
3. **风险偏好飙升**：赚钱后追入的标的风险通常高于原持仓
4. **标的切换加速**：不在意对新标的的熟悉程度，急于“锁定下一个机会”

## 核心机制

这是 **[[概念/正反馈循环的自我破坏]]** 的具体表现：

```
大盈利 → 信心膨胀 → 风险偏好↑ → 仓位急剧放大 → 
标的风险也放大（追涨停/融资）→ 潜在亏损↑ → 
若亏损 → 一个月利润被一笔抹平
```

## 经典案例：2024-11-08

**事件链**：
- 卖出华映科技 +14,443元（+26%）
- 当天全仓买入：中国软件 8.5万 + 晨曦航空 6.1万 + 晟楠科技 2.9万
- 总持仓：**17.5万**（相比华映时期约5.6万，放大3倍+）
- 买入支出 175,046元 > 卖出收回 172,125元 ——**赚的钱不仅全投回去，还加了点**

**与 [[概念/贪婪驱动的纪律撤销]] 的关系**：
赚钱是触发贪婪欲望的导火索——赚钱让纪律松动，风险控制失灵，是纪律撤销的典型案例。

## 防范措施

1. **大盈利后强制冷静期**：当日盈利 >10%或 >1 万元，自动触发“仅可减仓不可加仓”规则
2. **盈利资金出金或冻结**：大盈利后至少冻结 50% 利润（不用于当日再投入）
3. **仓位上限不可变**：无论市场情绪和个人信心如何，单日总仓位上限不可因单笔盈利而临时上调
4. **赚钱日是风险最高的一天**：建立认知——赚钱日的风险不一定比亏损日低，赚钱后的追涨更危险
