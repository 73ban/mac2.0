# 旧错误复发

## 核心定义

**旧错误复发**是一个元认知级别的问题：交易者不是不知道这个错误，而是知道之后仍然重犯。它直指 [[概念/知识-执行断裂]] 的核心困境。

> 一个错误被归档进 Wiki，不等于它不再发生。复发的旧错误恰恰证明：文档能记录错误，但不能阻止错误。

## 与“新错误”的本质区别

| | 新错误 | 旧错误复发 |
|---|--------|-----------|
| 认知状态 | 未知领域 | 已知但不执行 |
| 责任 | 经验不足 | 纪律不足 |
| 防御难度 | 低（有防错机制即可阻断） | 高（需要更高级别的执行层约束） |
| 情绪冲击 | 可控 | 严重（“我明明知道还犯”） |

## 复发机制：为什么知道了还会犯

1. **情境触发自动化**：错误不是被理性选择出来的，而是被特定情境自动触发的行为模式
2. **认知过载**：在快速决策的压力下，原始模式覆盖了学到的规则
3. **防错机制的空转**：Checklist 存在，但在关键时刻没有被使用
4. **“感觉良好”的麻痹**：前期盈利降低了警觉性 → 见 [[概念/正反馈循环的自我破坏]]
