# 日内资金闭环

## 定义

**日内资金闭环**是指卖出股票后所得资金在当日立即全部再投入，不保留任何现金缓冲的交易行为模式。

资金流动结构：
```
卖出 A → 资金全额到账 → 当日 100% 买入 B (+ C) → 现金留存 ≈ 0
```

## 核心特征

| 特征 | 说明 |
|------|------|
| 卖出即买入 | 卖出订单执行后，资金立即部署到新的买入标的 |
| 现金留存趋近于零 | 实质满仓状态，无任何防御性现金头寸 |
| "永远在场"心态 | 没有空仓观察期，没有等待确认期 |
| 日间无中断 | 收盘时仓位始终接近满仓 |

## 实证案例

### 2026-05-08 交割单

- 卖出禾盛新材 163,098 元
- 买入麦格米特 74,442 元 + 中超控股 86,724 元 = 161,166 元
- 资金留存仅 **1,932 元**（1.2%）
- → 典型日内资金闭环

## 危害性

1. **剥夺判断时间**：从卖出到买入没有"冷却期"，无法重新评估市场状态
2. **退潮日致命**：若当日为退潮日，满仓等于放大亏损敞口
3. **强化割肉-追涨循环**：卖出旧仓 → 立即买入新仓 → 若再次亏损 → 继续割肉换仓，形成闭环

## 与退潮日的关系

日内资金闭环本身是中性的（牛市中可最大化资金效率），但在**退潮日**执行时构成严重的纪律违反：
- 退潮日应以防守为主，降低仓位
- 卖出后的现金本应是防御工具
- 立即再满仓等于放弃防御
