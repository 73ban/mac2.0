# 板块清仓日

## 定义

**板块清仓日**指在某个交易日集中清空某一板块/方向下所有持仓的行为。这是[[概念/方向切换]]的终点操作，标志着一个方向尝试的正式终结。

## 已知清仓日时间线

| 日期 | 清仓方向 | 涉及标的 | 结果 |
|------|----------|----------|------|
| 🆕 **3/17** | **基建/军工** | [[中国电建-601669]]、[[西部材料-002149]]、[[航天发展-000547]] | 方向切换失败，持有仅 1 有效日 |
| 3/19 | 算力 | （待确认具体标的） | 算力方向集中清仓 |
| 4/21 | 军工/科创 | [[C大普微-301666]]、[[个股档案/麦格米特-002851]]、[[个股档案/中芯国际-688981]]、[[航天发展-000547]] | 方向切换至光伏 |
| 4/24 | 光伏 | [[个股档案/晶科科技-601778]]、[[个股档案/天通股份-600330]] | 光伏持有仅 3 天即被放弃 |

## 🆕 重要修正：基建/军工提前被毙

此前认为 3/19 是第一个板块清仓日（算力）。但 **3/17 交割单** 揭示：**基建/军工方向早在 3/17 就已被全线清仓**，比算力清仓早了 2 个交易日。

这一发现说明：
1. 交易者在 3 月中旬存在 **多个方向的同步试错**（基建/军工 vs 算力）
2. 基建/军工仅持有 1 有效日即被放弃，决策极为仓促
3. 不同方向的清仓时间错开，表明交易者是在 **逐一否定各方向**，而非基于统一的市场判断

## 清仓模式特征

| 特征 | 描述 |
|------|------|
| 触发方式 | 一次性全清，极少分批 |
| 持有期 | 通常极短（1-3 个交易日） |
| 后续行为 | 通常紧随新一轮[[概念/方向切换]] |
| 盈亏 | 多为微亏或打平，但频繁操作累积磨损 |
| 决策质量 | 低——缺乏对方向的充分验证 |

## 与 5 月灾难的关联

频繁的板块清仓日是本金持续磨损的直接证据：
- 每次清仓 = 手续费 + 可能的微亏
- 清仓后立即切换 = 新一轮风险暴露
- 形成"建仓→否定→清仓→再建仓"的消耗循环

详见 [[5月灾难专题]]。
