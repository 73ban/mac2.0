# 恐慌清仓

## 定义

因恐惧而非因纪律触发的大规模/全部清仓行为。与"理性止损"的关键区别在于动机是情绪而非规则。

## 特征

- 不是在预设止损位触发
- 因为"感觉要崩了"而清仓
- 清仓后通常立即后悔
- 常在市场最恐慌时卖出（恰好是最低点）

## 与止损的区别

| 止损 | 恐慌清仓 |
|------|------|
| 有计划、有触发条件 | 无计划、情绪驱动 |
| 执行后心态平和 | 执行后焦虑/后悔 |
| 在预设位置 | 在极端位置 |
| 保护资金 | 锁定亏损 |

## 预防

- 所有持仓必须设止损位
- 止损位写在交易计划里（不做"心理止损"）
- 使用条件单（让系统执行，不靠自己）
- 盘前确认"今天什么情况下我会恐慌？"
