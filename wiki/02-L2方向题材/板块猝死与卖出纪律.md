# 板块猝死与卖出纪律

## 模式定义

**板块猝死**指一个正在上涨的热点板块，因核心人气股突发利空（如减持、监管、基本面暴雷）导致整个板块在极短时间内（通常当日集合竞价）集体大幅下跌或跌停，板块行情瞬间终结。

## 触发条件

| 条件 | 说明 |
|------|------|
| 板块处于加速期 | 已连续上涨多日，市场关注度极高 |
| 核心人气股突发利空 | 减持公告、监管函、业绩暴雷等 |
| 板块情绪脆弱 | 高位获利盘堆积，一有风吹草动即踩踏 |

## 经典案例：物理AI板块（2026.06.11）

- **板块状态**: 周日大V预判，周一爆发（几十个涨停），周二周三持续上涨，处于加速期
- **触发事件**: [[达实智能]]周四突发减持4.3%公告
- **崩盘过程**: 200万手封单跌停 → 整个物理AI板块被带崩
- **来源**: [[2026-06-11-冻毙]]

## 卖出纪律

### 集合竞价阶段（快速反应）

| 信号 | 行动 |
|------|------|
| 核心人气股竞价跌停（无买单） | 立即挂单同板块持仓，争取开盘流动性 |
| 封单量巨大（百万手级别） | 不对反弹抱幻想，优先出逃 |
| 同板块多只个股竞价跌停 | 板块级崩盘确认，全面清仓 |

### 盘中阶段

| 情况 | 行动 |
|------|------|
| 已挂单未成交 | 保持挂单，关注封单量变化 |
| 非减持个股打开跌停 | 评估是否为错杀，可考虑部分留存 |
| 次日继续跌停 | 持续挂单，不因"跌太多了"而犹豫 |

### 事后复盘

- 复盘是否在加速期有减仓动作？
- 是否因"逻辑正确"而放松了止盈纪律？
- 单票仓位和板块仓位是否在约束范围内？

## 与交易体系的关系

- **L4执行控制层 - 卖出规则**: 板块猝死触发无条件止损，不需等待个股技术信号
- **仓位管理**: 板块加速期应逐步降低仓位，而非增加
- **L1市场环境**: 板块猝死后应重新评估市场阶段，可能进入退潮期

## 关键原则

1. **速度第一**: 板块猝死时，跑得快比跑得准重要
2. **不抱幻想**: 减持跌停不等于错杀，流动性枯竭是真风险
3. **事前约束 > 事后补救**: 加速期减仓比跌停时挂单更容易

## 开放性问题

- 板块猝死后，非利空个股是否存在错杀修复机会？如何判断？
- 如何量化"加速期"的结束信号，提前被动减仓而非等待事件触发？
