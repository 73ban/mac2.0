# 追涨式分笔加仓

## 定义

在同一交易日内，对同一标的进行多笔买入，且**买入价格逐笔升高，加仓手数逐笔增大**的行为模式。

## 识别标准

| 特征 | 说明 |
|------|------|
| 价格趋势 | 第二笔 > 第一笔，第三笔 > 第二笔 |
| 仓位趋势 | **越涨买的越多**（第三笔往往最大） |
| 行为时间 | 集中在同一交易日内，通常数小时内 |
| 触发心理 | 确认偏误 + FOMO（恐失机会） |

## 典型案例：韶能股份 (2026-05-14)

```
第一笔：12,400股 @ 9.060
第二笔： 2,400股 @ 9.130  ← 追高
第三笔：26,400股 @ 9.170  ← 远超前两笔之和，价格最高
```

## 与有计划分仓的本质区别

| 维度 | 有计划的分批建仓 | 追涨式分笔加仓 |
|------|----------------|--------------|
| 前置规划 | 盘前已确定仓位分配 | 无计划，盘中临时决定 |
| 价格方向 | 越低越买（左侧）或区间内吸纳 | 越高越买 |
| 仓位节奏 | 均匀或递减 | **递增**（最后一笔最重） |
| 决策依据 | 逻辑/技术位/时间节点 | 股价运动本身 |
| 心理驱动 | 纪律执行 | 贪婪 + FOMO |

## 心理机制

1. **确认偏误**：“股价如我预期在涨，证明我是对的。”
2. **FOMO**：“再不加仓就踏空了。”
3. **锚定效应**：锚定了某个更高的预期价位，当前的任何价格都是“便宜”。
4. **沉没成本倾向**：已投入了前两笔，不追加就“亏了”。

## 防御策略

1. **盘前必须写出买入逻辑**：如果写不出来，禁止开仓。
2. **单日加仓上限**：同一标的单日最多加仓一次，且不超过前一笔仓位的50%。
3. **价格倒挂禁止**：第二笔价格 > 第一笔价格时，禁止加仓（除非有明确的技术突破信号）。
4. **复盘回溯**：每笔追涨加仓必须在当日复盘中有独立解释。
