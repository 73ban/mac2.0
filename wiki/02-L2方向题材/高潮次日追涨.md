# 高潮次日追涨

## 定义

**高潮次日追涨**是指在某板块或概念在T日出现**集体大涨/涨停潮（高潮）**后，T+1日继续追买该板块多只标的的行为。

这是以[[2026-05-22 当日委托]]为典型案例系统化提炼出的错误模式。

## 核心机制

### 为什么是错误？

1. **高潮=筹码情绪的顶点**：大量浮筹在涨停板/大阳线处入场，T+1 分歧概率极高。
2. **卖盘集中释放**：T日获利的短线资金，T+1开盘即构成巨大的潜在抛压。
3. **溢价消失**：高潮后，板块内标的的“情绪溢价”最高，次日往往面临均值回归。
4. **流动性虹吸终结**：高潮日吸引全市场流动性，次日资金流向其他板块，支撑瓦解。

### 与“主线扩散逻辑”的区别

- **有效扩散**：核心标的高位横盘或温和调整，资金向板块内其他低估值/低位标的流动。通常发生在板块**首次被市场认知**或**逻辑二次强化**时。
- **高潮后追涨（本模式）**：板块**已经**出现集体涨停潮，市场情绪过度亢奋后，继续无差别买入任何沾边的标的。这是**扩散逻辑在错误时机的滥用**。

| 维度 | 有效扩散 | 高潮次日追涨 |
|------|---------|-------------|
| 板块位置 | 首次认知/逻辑强化初期 | 集体涨停潮之后 |
| 情绪状态 | 贪婪初期，仍有怀疑者 | 贪婪极致，FOMO 顶峰 |
| 核心标的走势 | 高位横盘或温和调整 | 已连续大涨/涨停 |
| 参与方式 | 低吸/回调买 | 追涨/追高 |
| 结果 | 可能出现第二波 | 大概率站岗 |

## 识别标准

| 信号 | 判断 |
|------|------|
| T日板块内多只涨停（≥3只） | ✅ 高潮成立 |
| T日板块指数大涨 5%+ | ✅ 高潮加强 |
| T日成交量异常放大 | ✅ 高潮筹码发散 |
| T+1日仍计划买入同板块标的 | ⚠️ 触发预警 |
| T+1日无卖出计划 | ⚠️ 风险倍增 |
| T+1日单板块仓位将 >50% | 🔴 强制红灯 |

## 防范规则

1. **T日高潮 → T+1 强制冷却**：板块高潮当日收盘后，次日**禁止**对该板块任何标的开新仓。
2. **冷却期**：至少等待 T+2 日或板块回踩 5 日均线后再评估。
3. **仓位上限**：单板块总仓位 **≤50%**。
4. **卖出纪律前置**：所有持仓隔夜必须绑定条件止损，次日开盘立即生效。

|------|------|------|------|
| 2026-05-22 | PCB | 鹏鼎控股、深南电路、沪电股份、景旺电子 | 5/23 鹏鼎出现较大亏损，详见[[2026-05-23-交割单复盘]] |

## 认知偏误：强逻辑的陷阱

本案例特别值得警惕的是：**[[分析/Rubin-VR200概念板块]]** 逻辑本身极强（PCB 在 VR200 平台 BOM 价值量大幅提升）。但**逻辑越强，越容易让人忽视时机选择**——这是典型的认知偏误：

> “因为逻辑强，所以任何时候买入都是对的。”

**事实**：强逻辑只是在正确的时机才有效。在高潮后入场，等于用最高的价格买最强的逻辑，安全边际为零。
