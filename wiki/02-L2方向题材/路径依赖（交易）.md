# 路径依赖（交易）

路径依赖是解释所有“明知故犯”行为的经济学根源。它描述了当一种行为模式被选定后，由于自我强化机制，会沿着既定方向不断锁定，即使存在更优解也难以切换。

## 1. 自我强化四机制（Arthur-诺斯框架）

| 机制 | 定义 | 在交易中的典型表现 |
| :--- | :--- | :--- |
| **规模经济/沉没成本** | 已投入成本使转换路径代价高昂 | **“等回本”心理**。持仓亏损后的止损决策被已投入的资金绑架。 |
| **学习效应** | 在特定路径上越熟练，越排斥新路径 | **策略锁定**。追板战法在牛市中优化得越完美，越无法适应退潮期。 |
| **协同效应** | 行为与工具、社群形成互补锁网 | **生态锁定**。研究龙虎榜 -> 加入打板群 -> 使用特定软件，全链条绑定单一生态。 |
| **适应性预期** | 基于过去经验形成预期，为过去行为辩护 | **“会回调的”执念**。趋势票卖飞后，越不回调越坚信会回调，最终踏空主升浪。 |

## 2. 路径依赖的形成四阶段（Sydow模型）

1.  **吸引阶段：**追板策略连续成功，产生致命吸引力。
2.  **收敛阶段：**账户中追板交易占比从30%激增至80%以上。
3.  **保持阶段：**环境已变，仍惯性延续，并安慰自己“这次不一样”。
4.  **锁定阶段：**退出成本极高，如5月灾难中的T+1流动性锁死。

> **关键反转警示：** 一次幸运的追板成功（触发事件）会**加速**从吸引到锁定的全进程。**早期小成功往往是未来大损失的催化剂。**

## 3. 打破交易路径依赖四法

| 方法 | 具体操作 | 对抗目标 |
| :--- | :--- | :--- |
| **重置沉没成本** | 强制使用[[概念/交易脱钩原则]]，将当下判断与历史成本分离。 | 沉没成本 |
| **强制策略轮换** | 每月定期用模拟盘练习非主力策略，保持认知多样性。 | 学习效应 |
| **信息源定期清理** | 淘汰近3个月胜率持续下降的付费或免费信息渠道。 | 协同效应 |
| **破坏性实验** | 设置“逆习惯日”，主动做一天逆回购，强行打断“必须交易”的惯性。 | 适应性预期 |
