# 资金趋同效应

## 定义

在超级行情（情绪9-10分）爆发期，全市场的资金不是分散寻找机会，而是**集中流向少数辨识度最高的核心标的**，使这些标的获得远超其他个股的资金支撑和流动性优势。这一效应是[[模式/角度原则]]成立的底层逻辑。

## 核心机制

### 资金行为的趋同性

| 资金类型 | 在超级行情中的行为 | 结果 |
|:---|:---|:---|
| 打板资金 | 排队买入封单最厚、连板最多的标的 | 最强标的的封单越堆越厚 |
| 低吸资金 | 等待分歧时低吸最强龙头 | 龙头分歧时承接能力最强 |
| 踏空资金 | 溢出至方向内弹性最大的标的（20cm+） | 弹性票获得二次加速 |
| 机构/游资 | 仅在核心标的上重仓布局 | 边缘标的无大资金关照 |

### 正反馈循环

```
资金集中 → 封单增厚 → 辨识度提升 → 更多资金涌入 → 角度更陡
    ↑_____________________________________________↓
```

这个正反馈循环使得超级行情中呈现"强者恒强、弱者恒弱"的极端分化。

## 对交易决策的意义

1. **解释了为何龙头最安全**：不是因为龙头不会跌，而是因为全市场的资金都愿意在这里承接
2. **解释了为何后排很危险**：后排没有资金共识，行情一旦出现分化，第一个被抛弃的就是弱势标的
3. **为[[模式/角度原则]]提供了理论基础**：角度最陡 = 资金共识最强 = 最安全

## 与[[概念/安全偏好悖论]]的关系

安全偏好悖论描述用户"避开最强的买安全的"这一行为倾向，而资金趋同效应揭示了一个残酷的事实：用户以为的"安全选择"（后排滞涨股），恰恰是**没有任何资金共识**的标的。在超级行情中，没有共识才是最危险的。

## 适用条件

资金趋同效应**仅在超级行情爆发期（情绪9-10分）显著存在**。在震荡市或退潮期中，资金行为趋于分散或撤退，趋同效应不再成立，此时角度最陡的标的可能成为最先崩塌的标的。

> **核心判断**：行情越火爆，趋同效应越明显；行情越冷清，趋同效应越微弱的甚至反转。
