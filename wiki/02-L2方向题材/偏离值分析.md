# 偏离值分析（Deviation Analysis）

## 目的

每笔交易有两个事前判断：
1. **确定性**（0-10）：我认为挣钱的概率多大
2. **盈亏比**（预期涨幅%）：我认为能挣多少

事后对照实际走势，计算偏离值。**偏离值越小 = 判断越准 = 交易系统越成熟。**

## 偏离值指标体系

### 1. 确定性偏离（Confidence Deviation）

```
确定性偏离 = 实际结果映射分 - 事前确定性评分
```

**实际结果映射分：**

| 实际结果 | 映射分 |
|------|:---:|
| 如期大涨（≥预期涨幅） | 10 |
| 小涨但未达预期 | 7 |
| 横盘不亏不赚 | 5 |
| 小亏（-5%以内） | 3 |
| 大亏（-5%以上） | 1 |
| 天地板/跌停闷杀 | 0 |

**解读：**
- 偏离 -5~0：判断准确
- 偏离 -6~-4：偏高估但方向对
- 偏离 -7 以下：严重高估，需要反思认知盲区

### 2. 盈亏比偏离（RR Deviation）

```
盈亏比偏离 = 实际最大涨幅 - 预期涨幅
```

**解读：**
- 正值：实际比预想好（保守）
- 负值：实际不如预期（乐观）
- 大幅负值：对上涨空间判断有系统性偏差

### 3. 最大回撤偏离（Max Drawdown Deviation）

```
回撤偏离 = 实际最大回撤 - 预期止损线
```

**解读：**
- 正值：回撤超出止损线（纪律问题或判断问题）
- 负值：回撤在预期内（风控有效）

## 统计维度

执行总管定期汇总：

| 维度 | 计算方式 |
|------|------|
| 平均确定性偏离 | 所有交易的 (实际映射 - 事前评分) 均值 |
| 平均盈亏比偏离 | 所有交易的 (实际最大涨幅 - 预期涨幅) 均值 |
| 高确定性胜率 | 确定性≥8 的交易中，实际盈利的比例 |
| 低确定性胜率 | 确定性≤5 的交易中，实际盈利的比例 |
| 盈亏比兑现率 | 实际涨幅 ≥ 预期涨幅 50% 的比例 |

## 使用方法

1. **每笔买入**：在交割单/复盘里声明确定性和盈亏比
2. **每笔卖出后**：Codex 自动生成偏离值分析
3. **每周/月**：汇总统计，识别系统性偏差
