# 网格补仓模式

## 定义

在亏损标的上模仿定投的**持续小额加仓**行为，特征是：

| 特征 | 描述 |
|------|------|
| **补仓频次** | 连续N个交易日反复补仓 |
| **成本锚定** | 围绕某个心理成本位（买入价、整数位）加仓 |
| **仓位扩大** | 从初始轻仓逐步演变为重仓甚至满仓 |
| **杠杆叠加** | 常伴随融资买入，放大风险暴露 |

**本质**：将"摊薄成本"的理性财务概念，错误应用于"拒绝认错"的情绪陷阱。表面上是在优化持仓，实质上是**锚定效应的机械化执行**。

## 与锚定效应的关系

[[概念/锚定效应]] 是认知层面——执念于某个价格或预期。
**网格补仓模式**是行为层面——认知偏差通过"持续小额加仓"这个动作外化为交易行为。

识别公式：
> 锚定效应（认知） + 连续补仓（行为） + 仓位失控（后果） = 网格补仓模式

## 案例 #1：深圳华强（2024年9月12-20日）

| 日期 | 操作 | 持仓变化 | 累计 |
|------|------|----------|------|
| 9/12 | 买入400股 | 400 | 400 |
| 9/13 | 卖出400→买入200（做T） | 净+200? | 500 |
| 9/18 | 买入100 | +100 | 600 |
| 9/19 | 买入200 | +200 | 800 |
| 9/20 | 买入300 | +300 | 800 |

**7个交易日内连续补仓5次**，从500股增至800股，全部融资，仓位约120%+。

**操作逻辑**（用户自述）："华强会反弹，摊薄成本"
**实际结果**：微亏状态，仓位却已打满，完全错过了信创/华为链的主线行情

**诊断**：这是 [[总结/9月→12月三阶段演变]] 第一阶段的典型行为——"知道主线在哪但买旧票"。用户能准确识别信创是"今日最强方向"，但资金全锁在已非主线的深圳华强上。

## 识别信号（自检清单）

- [ ] 同一标的连续3个交易日以上有买入行为
- [ ] 每次买入理由包含"降低成本"或"等反弹"
- [ ] 该标的已非当期市场主线核心
- [ ] 仓位在该标的上持续扩大而非缩小
- [ ] 可用的资金/额度被该标的持续消耗

**≥3项勾选 → 高度警惕网格补仓模式**

## 防治规则

1. **禁止对非主线标的追加仓位**：主线切换后，旧票的每一次补仓都是锚定陷阱
2. **补仓需有新的催化逻辑**：不是"跌了所以便宜"，而是"出现了之前没有的上涨理由"
3. **设定补仓上限**：任何单票的总仓位不得超过计划仓位的120%
4. **强制对照**：每次补仓前，先写出"如果不持有此票，我今天会买它吗？"
