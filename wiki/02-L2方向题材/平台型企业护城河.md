# 平台型企业护城河

## 定义

指公司充当产业链中一个**多客户共享、高附加值、高转换成本**的中间层，通过提供一站式或标准化核心能力，成为下游不可或缺的"底盘"。典型代表：台积电（晶圆代工）、天孚通信（光器件）。

## 天孚通信：光器件"台积电"范式

1.  **不做成品，做核心组件**：天孚不生产光模块，而是为所有光模块厂提供最难做、最关键的光器件/引擎。
2.  **工艺/良率壁垒**：产品的竞争力不是"设计"，而是谁能用高良率低成本地造出来。天孚的陶瓷套管、高速封装能力是多年工艺积累的结晶，难以速成。
3.  **一站式方案**：从材料→零组件→封装测试都能做，客户不需要拼凑多家供应商，转换成本极高。
4.  **受益全行业，不受单一路线**：光模块方案可以硅光、也可以传统III-V族，但无论哪种，都需要高品质光器件。这是典型的"卖铲人"逻辑。

## 如何识别平台型企业护城河？

| 特征 | 说明 |
| :--- | :--- |
| **垂直整合** | 能做产业链多个环节，而不是一个点 |
| **聚焦上游核心** | 把难做的环节做到极致，下游客户离了你就转不动 |
| **多客户共享** | 不赌单一客户，业务分散在下游多个最终用户 |
| **高毛利率+稳定性** | 护城河的量化证据；毛利率长期高于同行且波动小 |
| **高ROE** | 商业模式优秀，轻资产/高附加值/可复制 |

## 与客户集中度风险的辩证关系

天孚通信是一个特殊案例：**护城河极深，但客户集中度极高**。这两者相互制衡：

-   若护城河弱 + 客户集中 = 极度危险（随时被压价、替换）
-   若护城河深 + 客户集中 = 高确定性暂时压倒脆弱性，但终局仍存隐患

因此，即便护城河再深，也需要在估值和仓位管理上为 [[概念/客户集中度风险]] 留出容错空间。
