# 冰点期对抗趋势的补仓

## 定义

在市场情绪进入冰点（情绪指数极低）、标的已明确处于下跌趋势的阶段，对已持有并产生浮亏的股票，在下跌途中追加买入以图摊薄持仓成本的行为。此操作通常以被迫在更低位置止损出局告终。

## 核心特征

| 特征 | 说明 |
|------|------|
| 市场环境 | 情绪指数极低（如2/10），退潮期或冰点日 |
| 标的状态 | 已持有、已有浮亏、处于明确下跌趋势 |
| 操作动机 | 摊薄成本、不想认输、幻想反弹 |
| 操作方式 | 在下跌途中买入（非企稳后） |
| 典型结果 | 继续下跌，被迫止损，扩大亏损 |

## 经典案例：2024-09-13 深圳华强

- 市场情绪指数：2/10
- 持仓：400股@33.72（09-12建仓，已浮亏）
- 补仓：200股@31.57
- 结果：补仓后继续下跌至跌停29.48，被迫T+0卖出
- T+0亏损：-418元
- 总浮亏：-1,696元

## 心理机制

此行为是 [[概念/贪婪驱动的纪律撤销]] 的一种具体表现形式——贪婪在此处以“不想认输、试图通过摊薄成本翻盘”的面目出现。它绕过了纪律：如果客观审视，此时正确的操作应是清仓或至少不动，而非加仓。

## 与正常补仓的区别

| 维度 | 正常补仓 | 冰点期对抗趋势补仓 |
|------|----------|---------------------|
| 市场环境 | 上升趋势中的回调 | 退潮/冰点/下跌趋势 |
| 标的状态 | 逻辑未破坏 | 逻辑已破坏（旧周期） |
| 补仓时机 | 确认企稳后 | 下跌途中 |
| 补仓目的 | 基于逻辑加仓 | 摊薄成本以翻盘 |
| 风险控制 | 有止损计划 | 无，直到被迫止损 |

## 防范规则

1. **冰点日禁止买入**：市场情绪指数低于3分时，暂停所有开仓与补仓操作。
2. **浮亏标的不补仓**：持仓已产生浮亏时，默认方向应为减仓或不动，补仓需有独立逻辑（非“摊薄成本”）。
3. **旧周期标的不留恋**：一轮周期结束后，无论盈亏，坚决出清旧周期标的。
