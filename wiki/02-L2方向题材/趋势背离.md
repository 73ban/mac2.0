# 趋势背离

## 定义

**趋势背离**，是指**营业收入持续增长/维持高位，但毛利率连续多个季度下滑**的异常现象。这是毛利率结构性下行监控框架中的**第一预警信号**。

## 为什么是危险信号

营收增长 + 毛利率下滑 = 公司用**更低的利润率换取规模**，背后只有两种可能：

1. **降价促销维持规模**：产品竞争力下降，不得不让利
2. **成本压力无法转嫁**：上游成本刚性上升，但下游客户不接受提价

两种可能都指向盈利能力的**结构性恶化**，而非周期性的正常波动。

## 判断三步法

```
第一步：确认营收增长是内生还是并购驱动
    ├─ 内生增长 → 信号有效，继续判断
    └─ 并购驱动 → 信号可能失真，需要收购标的的单独数据
    
第二步：拆分毛利率下滑的来源
    ├─ 成本效应驱动 → 判断是周期性（材料涨价）还是结构性（人工刚性）
    └─ 组合效应驱动 → 几乎一定结构性，见[[概念/组合效应恶化]]
    
第三步：交叉验证
    ├─ 经营现金流/净利润比率是否同步恶化？
    ├─ 同行是否存在同样的背离？
    └─ 应收账款周转天数是否变长？（降价促销常伴随账期拉长）
```

## 操作含义

趋势背离确认后：
- **估值锚点必须重新评估**：现有的 [[概念/机构估值锚定]] 可能已不适用
