# 业绩预期差

> **层级定位**：L2财务传导层——连接"基本面变化"与"市场重新定价"的核心传动机制

## 定义

**业绩预期差（Earnings Surprise）** 指公司实际公布的业绩（净利润、营收、ROE等）与市场一致预期之间的偏差。正值表示超预期（Beat），负值表示不及预期（Miss）。

**量化表达式**：
```
SUE = (实际公告净利润 - 预期净利润) / 预测标准差
```

## 为什么是核心概念

在整个交易体系中，业绩预期差处于"基本面逻辑→价格反应"的中间传动层。它解释了**为什么业绩变化会导致股价变化**，以及这种反应的**非对称性**规律。

此前wiki中五个已有概念——[[预期先行业绩滞后]]、[[机构估值锚定]]、[[ROE-TTM高位拐点]]、[[情绪溢价风险]]、[[单季扰动vs方向性转变]]——都与此相关，但缺少对传动环节本身的独立归档。

## 核心规律一：成长股的"盈利鱼雷"效应 ⭐

> **来源**：Skinner & Sloan (2002), "Earnings Surprises, Growth Expectations, and Stock Returns"

**核心发现**：成长股对业绩不及预期的反应，远大于对超预期的反应。

- **Miss这一事实本身**（而非miss的幅度）即触发-10%至-20%的股价暴跌
- miss幅度极微小（低于股价的0.5%）也能引发同等烈度的下跌
- beat仅带来温和上涨，且边际回报递减

**"盈利鱼雷"的命名逻辑**：就像鱼雷一样——致命的不在于接触面积，而在于击中本身。miss打破了成长叙事的连贯性，市场瞬间重新定价整个未来现金流预期。

**与[[单季扰动vs方向性转变]]的关系**：
- 业绩预期差处理"发生了什么"——miss = 鱼雷击中，立即反应
- 单季扰动处理"事件意味着什么"——miss后的事后归因
- **操作原则**：先执行预期差的清仓纪律，再启动扰动归因分析（保住本金优先）

## 核心规律二：分析师系统性乐观偏差

> **来源**：国盛证券 (2022)

**数据实证**：

| 一致预期增速区间 | 低于预期的公司占比 |
|----------------|------------------|
| 10%-15% | 47% |
| 40%-45% | **81%** |

**含义**：
1. 预期增速越高，分析师越倾向于线性外推→过度乐观
2. 市场对此有定价——高预期增速股的估值已包含"大概率miss"的折价
3. 对高增长股而言，"符合预期"某种程度上就是"超预期"

这与[[情绪溢价风险]]直接相关：高估值本身 = 市场对持续超预期的乐观定价。一旦miss发生，溢价瞬间转化为折价。

---

## 核心规律三：不涨的Beat——超预期≠股价上涨

**四种经典例外情景**：

| 情景 | 机制 | 与已有概念关联 |
|------|------|--------------|
| ① 预期已提前定价 | 财报前股价已大涨，small beat=利好出尽 | [[机构估值锚定]] |
| ② 前瞻指引下调 | 当期好但未来差，市场交易未来 | [[预期先行业绩滞后]] |
| ③ 利润beat但营收miss | 降本增效达成，增长动能衰减 | [[ROE驱动力分解]] |
| ④ 估值极度透支 | 高估值下需extraordinary beat才支撑 | [[情绪溢价风险]] |

**操作推论**：beat出现时，按①→②→③→④顺序逐一检查，全部通过才能确认为有效利好。

## 核心规律四：预期差驱动的估值波动

```
超预期阶段 → 盈利上修 + 估值上修 = 戴维斯双击
低于预期阶段 → 盈利下修 + 估值下修 = 戴维斯双杀
```

这与[[concepts/景气度扩散效应]]的互动：
- 持续超预期 → 吸引增量资金 → 拥挤度上升（正常现象）
- **拥挤度本身不可怕，可怕的是拥挤+miss同时发生**
- 拥挤+miss → 被迫卖盘叠加估值下修 → 加速暴跌

这正是[[抱团崩塌四条件]]中**条件四（全民化+信息同质化）的触发机制**：
> 高度拥挤 → 一次意外的miss → 所有人同时发现没人在接盘 → 倾泻式下跌

## 操作推论（速查）

1. **miss = 无条件警惕信号**
   - 对高估值成长股，miss本身就是卖出信号
   - **不等确认、不等归因、先保住本金**
   - 事后用[[单季扰动vs方向性转变]]复盘归因

2. **beat需四检查**（缺一不可）
   - ① 公告前股价是否已透支？
   - ② 前瞻指引是否下调？
   - ③ 营收增速是否同步？
   - ④ 估值是否已进入情绪溢价区？

3. **极端超预期反而警惕**
   - 超预期100%+ → 可能含一次性因素或基准不准
   - 需交叉验证评级机构覆盖数

4. **ROE拐点 + miss = 双重打击**
   - 方向确认 + 意外负面
   - 信号严重程度远高于单独事件

5. **拥挤 + miss = 抱团崩塌导火索**
   - 触发"拥挤→miss→无人接盘→倾泻"加速链条
   - 在拥挤度高位时，对持仓股业绩公告窗口期建立提前减仓预案

## 开放问题

1. **A股业绩预告制度特殊性**：强制/自愿预告制度与美股不同，预告可靠性和修正频率是实操重要变量
2. **Q2验证窗口**：当前AI算力链核心标的（英维克、中际旭创等）Q2财报即将到来，这是业绩预期差框架首次大规模压力测试
3. **数据可得性**：个人投资者能否低成本获取一致预期数据？若不可得，需开发简化版替代指标（如单季度ROE环比变化）
