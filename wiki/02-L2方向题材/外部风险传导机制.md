# 外部风险传导机制

## 定义

当海外市场（尤其是美股）出现系统性暴跌或黑天鹅事件时，通过资金流动、情绪传染、关联资产定价等路径，向A股传导并引发退潮或下跌的风险传递过程。

## 传导链识别

### 基本传导链

```
美股暴跌/经济冲击 → 亚太开盘承压 → 外资流出/A股情绪恶化 → 科技股等风险资产领跌
```

### “链条完整性”判断（2026-06-08 新增）

一次完整的外部冲击通常由多个环节构成，判断其强度需要看**链条是否连续**：

```
美股暴跌（第一环） → 亚洲主要市场熔断级下跌（第二环确认） → 地缘政治等不确定性叠加（第三环）
→ A股科技、成长板块重挫（直接冲击）→ 涨停家数/成交量/封板率提供反证（判断是否全面恐慌）
```

**关键阈值**：

- 若亚洲市场出现熔断级别的传导（如韩国KOSPI熔断），则A股科技股几乎必然成为重灾区。
- 此时需要结合涨停家数、封板率来区分**结构性恐慌**与**全面退潮**：
  - 涨停家数>50且仍有独立逻辑股封涨停 → 属于中等偏弱退潮，观望即可，不宜恐慌清仓。
  - 涨停家数稀少、跌停扩散 → 可能触发全面退潮，需执行[[退潮日仓位管理规则]]。

## 2026-06-08 案例

- 美股黑色星期五 + 韩国股市熔断 + 中东局势升级 → 三重外部利空共振。
- 科创50-3.63%，科技股全线下跌。
- 但56家涨停，中科三环、埃斯顿、金安国纪等强势股依然封板，说明传导未引发系统性恐慌。
- 半日缩量40%，资金以观望为主，未踩踏。
- 对照链条完整性判断：外部三重共振，但内部仍有局部独立逻辑，因此判断为中等偏弱退潮，执行“不恐慌清仓、观望等待”的计划。

## 与相关概念的关系

- [[退潮日-1]]：外部传导往往成为退潮的触发器，需结合情绪指数和结构性分化来判定退潮级别。
- [[退潮日结构性行情]]：在外部冲击退潮时，如何区分独立逻辑股的抗跌和退潮反弹卖点。

## 风险提示

外部传导的“预期差”往往表现为：A股低开消化后快速修复，或持续踩踏。判断修复力需要监测：午后量能是否回补、机器人等风向板块是否回流、亚洲市场是否止跌。
