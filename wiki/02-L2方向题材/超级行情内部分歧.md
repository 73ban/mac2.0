# 超级行情内部分歧

**定义**：在史诗级逼空行情的高潮阶段，市场情绪达到极端亢奋（情绪分10分），但首次出现非ST跌停为零、高位连板股断板等异常信号，同时成交额仍在环比放大（+25%以上），形成一种“放量加速中的分歧”。这与退潮期的缩量崩塌有本质区别。

## 识别信号
| 指标 | 典型表现 |
|:---|:---|
| 情绪分 | 10分（极值） |
| 非ST跌停 | 0家（异常值，不可持续） |
| 高位连板股 | 部分掉队/断板（如海能达6→7板断板） |
| 大盘成交额 | 环比放量25%以上（加速放量） |
| 主线龙头 | 继续一字板/巨单封死（天风证券封单15亿） |
| 指数 | 可能触及极端涨幅（创业板指涨停） |

## 与退潮分歧的本质区别
| 特征 | 超级行情内部分歧 | 退潮期分歧（如5/22） |
|:---|:---|:---|
| 量能 | 持续环比放量 | 缩量或量平价跌 |
| 主线龙头 | 封单不减 | 开板或巨量换手 |
| 后续走势 | 可能二次加速或进入高位震荡 | 大概率转入持续下跌 |

## 操作意义
1. **调仓窗口**：是清理掉旧方向弱票、向最强主线集中的最佳时机
2. **勿恐慌**：不能因高标断板而全面离场，否则可能踏空二次加速
3. **绝对主线仓位倾斜**：执行力弱的分散持仓应在此时纠正
4. **识别真分歧**：若量能不再放大且主线龙头开板，则升级为真分歧预警

## 案例：2024-09-27
- 情绪10分，成交额14556亿（+25%），零跌停
- 海能达断板、常山北明未能连板
- 天风证券继续一字，东方财富138亿成交首板
- 证券主线绝对统治，但交易者却分散至CXO和半导体
- **应操作**：将中芯国际等非核心仓位转换为天风或东财的弹性仓位
