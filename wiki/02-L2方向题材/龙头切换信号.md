# 龙头切换信号

## 定义

龙头切换信号是指在市场关键转折日（尤其是行情爆发日），**旧周期高标集体崩塌**与**新主线龙头集体爆发**同时发生的现象。这是资金大规模迁徙的盘面语言——旧龙断板之日即新龙崛起之时。

## 核心识别特征

1. **旧龙跌停**：上一周期的高辨识度标的出现跌停（一字跌停或天地板）
2. **新主线涨停潮**：一个新的板块方向出现多股涨停的集体爆发
3. **成交量暴增**：全市场成交显著放大，表明增量+存量资金同时切换方向
4. **时间同步**：两者在同一交易日发生，而非渐变过渡

## 典型案例：2024-09-24

| 要素 | 表现 |
|------|------|
| 旧龙崩塌 | 保变电气（12天10板→天地板）、南天信息（一字跌停）、深圳华强（一字跌停） |
| 新主线爆发 | 证券+多元金融近20股首板涨停 |
| 成交量 | 5510亿→9713亿（+76%） |
| 催化 | 央行+证监会联合发布会（历史级别） |

## 与退潮日的区别

- [[概念/退潮日]]：高位股集体崩跌，但**没有**新的主线方向同时爆发 → 市场整体降温
- 龙头切换：旧龙崩跌 + 新方向爆发同步发生 → 资金迁徙，是牛市/强反弹中的典型信号
- 龙头切换往往发生在行情**爆发初期**，退潮日往往发生在行情**末端**

## 操作含义

- **正向**：识别到龙头切换信号 → 果断调仓至新主线
- **负向**：未能识别或滞后 → 继续持有旧龙（可能遭遇持续下跌）或偏离主线选股
- 924当日用户的三笔操作均未参与证券/多元金融首板，属于负向案例
