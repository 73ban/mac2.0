# PVM毛利桥分析

## 定义

**PVM分析**（Price-Volume-Mix Analysis），又称**毛利桥（Margin Bridge）**，是将毛利率/毛利额变化分解为三大效应贡献的量化归因框架：

```
毛利率变动 = 价格效应 + 成本效应 + 组合效应
                (P)        (V/C)      (M)
```

## 三大效应详解

### 1. 价格效应（Price Effect）
衡量产品**售价变化**对毛利率的直接影响。

| 效应方向 | 含义 |
|---------|------|
| 正效应 | 提价能力强，品牌溢价/供不应求 |
| 负效应 | 竞争降价/被动促销/产品老化 → ⚠️ |

### 2. 成本效应（Cost Effect）
衡量**单位生产成本**变化的影响。

| 效应方向 | 含义 |
|---------|------|
| 正效应 | 成本下降（良率提升/规模效应/上游降价） |
| 负效应 | 成本刚性上升（材料涨价/人工上涨/新产线折旧）→ ⚠️ |

### 3. 组合效应（Mix Effect）
衡量**产品/客户结构变化**的影响。详见 [[概念/组合效应恶化]]。

| 效应方向 | 含义 |
|---------|------|
| 正效应 | 高毛利产品占比上升 → 产品竞争力强 |
| **负效应** | **高毛利产品被低毛利替代 → 🔴 结构性下行最直接的量化指征** |

## 可视化：瀑布图

使用**毛利桥（Margin Bridge）** 瀑布图直观展示：

```
上期毛利率 ──→ +价格效应 ──→ +成本效应 ──→ +组合效应 = 本期毛利率
  (起点)      (绿柱/红柱)    (绿柱/红柱)    (绿柱/红柱)     (终点)
```

红色=负贡献（利润流失来源），绿色=正贡献（利润增长引擎）。

## 实战判断矩阵

| 价格效应 | 成本效应 | 组合效应 | 综合判断 |
|---------|---------|---------|---------|
| 正 | 正 | 正 | ✅ 三优，产品力+定价力+成本控制均强 |
| 负 | 负 | 负 | 🔴 三杀，结构性危机，必须启动减仓 |
| 正 | 负 | 负 | ⚠️ 提价对冲不了结构与成本恶化，警惕后续转弱 |
| 负 | 负 | 正 | ⚠️ 产品升级换挡期，关注升级能否持续 |
| 负 | 正 | 负 | 🔴 低成本红利耗尽+产品力下降，双杀信号 |

## 数据要求与限制

**理想状态**：需要分产品线的收入+成本+销量数据
**现实约束**：A股上市公司（尤其中小公司）可能仅披露分部收入，成本/毛利拆分不完整

**兜底规则**：
- 数据不足时，至少关注公司定期报告中的**分产品/分业务毛利率变化**
- 若公司未拆分，观察同行业竞争对手的拆分数据作为替代参考
- 标注"数据不足以支撑完整PVM"时，框架判断降级为"定性观察"
