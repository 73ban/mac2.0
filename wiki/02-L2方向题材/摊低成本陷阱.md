# 摊低成本陷阱

## 定义

在退潮期或下跌趋势中，以"摊低成本"为由**部分保留亏损仓位**的操作行为。其本质是[[概念/锚定效应|成本锚定]]与[[概念/协同亏损|协同亏损]]的组合陷阱——用"摊低成本"的叙事合理化不肯认亏，为下一次协同亏损制造温床。

## 核心机制

```
持股亏损 → 拒绝全部认错 → "卖掉一部分，留一部分摊低成本"
    → 保留仓位继续亏损 → 叠加新亏损的锚定效应
    → 触发割肉-追涨循环 → 协同亏损扩大
```

### 为什么"摊低成本"在退潮日尤其致命？

1. **环境不配合**：退潮日全市场下跌，保留仓位大概率继续亏损
2. **锚定强化**：保留的仓位让成本锚点持续存在，影响后续判断
3. **协同亏损温床**：保留的亏损仓位为下一次"割肉→追涨补亏"循环提供了起点
4. **自我合理化**："我在摊低成本"是一种叙事，让人感觉在"做正确的事"，实质上是在拒绝认错

## 经典案例：巨轮智能 2024-12-23

- **背景**：巨轮智能此前已是"三战两败"的持续亏损载体
- **当日操作**：卖出300股（-11.4%），保留400股"摊低成本"
- **同日对比**：佳先股份、日出东方全部清仓——说明用户**有能力执行完整清仓**，但对巨轮选择性打折扣
- **暴露问题**：对最疼的亏损票最难割舍——情感依附 > 纪律执行

## 破解策略

1. **定义刚性**：清仓 = 全部卖出。不存在"卖一半摊低成本"的选项
2. **识别叙事**：当脑中浮现"摊低成本"这个词时，立即识别为锚定效应的自我合理化
3. **反向测试**：如果这只股票不在持仓中，你现在会买入吗？如果不会，就应该全部卖出
4. **退潮日零容忍**：退潮日不得保留任何亏损仓位——参见[[策略/退潮票早盘清仓]]

## 开放性观察

> 巨轮智能从"三战两败"（12月早期）到"保留摊低成本"（12/23），已成为跨越整个12月的持续亏损载体。这个模式是否暴露了用户对特定股票的情感依附（"舍不得它跌完"）？这是超越技术层面的心理问题。
