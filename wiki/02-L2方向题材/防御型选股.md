# 防御型选股

## 定义

在市场主线不明朗或退潮期，以"不亏钱"为首要目标的选股策略。

## 选股标准

| 维度 | 标准 |
|------|------|
| 行业 | 公用事业（电力/水务）、必选消费、医药 |
| 估值 | PE<20，PB<2 |
| 股息 | 股息率>3% |
| 波动 | 近期涨跌幅<±3% |
| 流动性 | 日均成交>1亿 |
| 催化剂 | 有明确事件支撑（分红/业绩预告/政策）|

## 操作方式

- 仓位：单票≤10%，总仓≤30%
- 持有期：1-5个交易日
- 卖出：涨5%止盈，跌3%止损
- 不在防御票上加仓

## 典型标的

- 公用事业：华能国际、大唐发电、韶能股份
- 必选消费：中国联通（防御+高股息）

## 防御≠安全

- 防御票也会跌 — 只是波动小
- 防御票不能变成"拿住不动" — 也要设止损
- 防御票不是长期持有 — 是过渡期工具
