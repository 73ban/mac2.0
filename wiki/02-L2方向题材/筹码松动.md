# 筹码松动

## 核心定义

> **筹码松动 = 原本锁仓的筹码开始释放抛压，主力/大资金的锁仓意愿在下降。**

本质不是"股票不好"，而是**内部人的信心在瓦解**。这形成了一个独立于基本面逻辑的技术面卖出信号系统。

## 铁律

> **筹码松动 = 资金在跑，逻辑再好也没用。**

这与[[概念/强预期弱现实]]互补：强预期弱现实讲的是"逻辑不成立"，而筹码松动讲的是"即使逻辑成立，资金走了就是走了"。技术面压倒基本面。

## 三级分级体系与操作对应

### 🟡 轻度松动
| 特征 | 描述 |
|------|------|
| 涨停板打开次数增多 | 以前一字板，现在反复炸开又封上 |
| 封单越来越薄 | 封单金额从10亿→3亿→5000万递减 |
| 成交量持续放大 | 缩量板→放量板→爆量板 |

**操作：继续持有，但降低预期，设好止盈。**
> 这是最早的警告——市场在告诉你"别太乐观"。

### 🟠 中度松动
| 特征 | 描述 |
|------|------|
| 封不住涨停 | 封板后反复打开，尾盘勉强封住或炸到收盘 |
| 分时图频繁跳水 | 拉升后快速回落，V型变U型再变倒V |
| 高位换手率飙升 | 5%→15%→25%+（正常换手5-15%） |
| 龙虎榜出逃 | 游资席位从净买入变成净卖出 |

**操作：减仓一半，剩下看次日表现。**
> 中线资金的信心已经动摇，必须大幅降低风险敞口。

### 🔴 重度松动（即将见顶）
| 特征 | 描述 |
|------|------|
| 高开低走/冲高回落 | 早盘冲高3-5%，下午跌绿收盘 |
| 跌停板出现 | 龙头开始有人按跌停 |
| 跟风股先崩 | 龙头还没事，跟风票已经-7%~-10% |
| 板块联动消失 | 昨天还一起涨，今天只有龙头撑场面 |
| 利好不涨 | 出了利好反而高开低走 |

**操作：全部清仓，不要犹豫。**
> 此时的犹豫是[[错误/锚定回追]]的最强心理驱动——"之前涨那么好，应该还能起来"。但复盘数据反复证明：重度松动后不走的，最终都以更差的价格割肉。

## 量价关系矩阵

| 阶段 | 成交量 | 涨幅 | 含义 |
|------|--------|------|------|
| 正常 | 缩量或温和放量 | 涨停 | 筹码锁定好 |
| 初松 | 成交量翻倍 | 涨停但炸开 | 有人在卖 |
| 松了 | 爆量（3倍以上） | 勉强涨停/收长上影 | 大量资金出逃 |
| 崩了 | 放量 | 大跌/跌停 | 没人接了 |

## 分时图形态

| 强度 | 形态 |
|------|------|
| **强** | 9:25高开→直线拉到涨停→封死到收盘（一字或T字） |
| **松** | 高开→拉到涨停→炸开→回封→再炸→尾盘勉强封住 |
| **弱** | 高开→冲高→回落→翻绿→一路走低 |

## 与退潮日的关系

**筹码松动是退潮日在个股层面的微观先行指标。**

| 层次 | 概念 | 触发时机 |
|------|------|---------|
| 宏观 | [[概念/退潮日]] | 市场整体情绪转向 |
| 微观 | **筹码松动** | 个股/板块内部资金结构瓦解 |

退潮日中个股必然出现筹码松动，但筹码松动可能先于退潮日出现。这意味着：**等到确认退潮再行动，可能已经晚了。**筹码松动提供了一条更早的逃生通道。

## 板块内部分化信号

> "龙头还撑在涨停，跟风股已经-5%~-8% → 板块筹码开始松动"

这与[[概念/内部缩圈]]直接对应——板块内部分化是内部缩圈的早期信号。但筹码松动的视角不同：内部缩圈是一个宏观叙事（"资金在板块内部集中到头部"），筹码松动则给出了**具体的识别标准**（跟风票跌幅阈值、时间顺序）。

## 经典案例

### 日丰股份 2024-12-18

5板当天涨停打开**9次**，典型重度筹码松动。次日核按钮带崩。
> 用户当时未持仓此票，但此案例与12月交易记录时间高度重叠（华扬联众一字跌停在12/11-12/12，视觉中国正反馈循环在12/16起），说明重度松动信号在当时市场环境中多次出现。

## 开放问题

1. **量化阈值适用性**：成交量翻倍/爆量3倍的阈值是否适用于不同市值股票？大盘股与小盘股的分时特征可能不同。
2. **退潮+松动双重触发**：宏观退潮+微观筹码松动同时触发时，是否需要强化为"无条件清仓"？（文档倾向于"是"——但需明确定义为策略规则）
3. **与[[策略/退潮票早盘清仓]]的优先级**：如果退潮日信号未触发但筹码已重度松动，是否仍然执行清仓？
