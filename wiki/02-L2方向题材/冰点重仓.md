# 冰点重仓

## 定义

**冰点重仓**是指在市场情绪处于冰点/退潮期时，仍然以重仓（≥50%）出击的交易行为。这是认知-执行断裂的典型表现。

## 核心特征

| 特征 | 说明 |
|------|------|
| 市场状态 | 冰点/退潮期，情绪指数≤3/10 |
| 仓位水平 | ≥50%（含融资杠杆） |
| 标的选择 | 前期高标/龙头（持续下跌中） |
| 买入逻辑 | 缺失或模糊（"跌多了""博反弹"） |
| 止损计划 | 缺失 |
| 持仓周期 | 未定义 |

## 为什么危险？

1. **退潮期胜率极低**：冰点环境下，任何买入的胜率都被压缩
2. **前期高标补跌风险**：退潮期杀高标是规律，而非例外
3. **融资放大亏损**：杠杆使小幅下跌产生成倍亏损
4. **无止损=无底线**：一旦继续下跌，浮亏快速扩大

## 经典案例：2024-09-12 深圳华强

- 市场冰点，情绪3分，成交5156亿地量
- 融资60%仓位买入深圳华强（5日跌21%）
- 建仓日浮亏-2.9%
- 无买入逻辑记录，无止损计划

详见：[[交易起点/2024-09-12-冰点融资买入深圳华强]]

## 防范规则

- **退潮日强制空仓或≤10%试探仓**
- **冰点不做左侧抄底**
- **拒绝"跌多了"的锚定思维**（参见[[概念/锚定效应]]）
- **任何买入必须先写逻辑+止损+持仓周期**
