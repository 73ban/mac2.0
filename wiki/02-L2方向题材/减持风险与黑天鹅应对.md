# 减持风险与黑天鹅应对

## 定义

**减持黑天鹅**指上市公司大股东突然宣布大规模减持股份，导致股价崩盘、甚至拖垮整个板块的不可预判外部冲击。其核心特征是：事件发生前无法通过常规分析预判，发生后影响巨大且迅速。

## 典型特征

| 特征 | 说明 |
|------|------|
| 不可预判性 | 减持公告在发布前不公开，无技术面或基本面前兆 |
| 连锁反应 | 板块核心人气股减持可带崩整个板块，即使其他个股基本面未变 |
| 速度极快 | 集合竞价即封跌停，盘中无逃生窗口 |
| 破坏力大 | 大比例减持（如4%+）可导致连续跌停 |

## 经典案例：达实智能（2026.06.11）

- **背景**: 物理AI板块周日被大V推荐，周一爆发（几十个涨停），周二周三持续上涨
- **事件**: 周四突发达实智能减持4.3%公告
- **结果**: 200万手封单跌停，整个物理AI板块崩盘
- **教训**: 板块逻辑正确、走势良好，但单一个股黑天鹅足以摧毁一切。参见[[2026-06-11-冻毙]]

## 应对策略

### 1. 仓位约束（最核心）

| 规则 | 逻辑 |
|------|------|
| 单票仓位上限 ≤ 总仓位的X% | 单票黑天鹅破坏力可控 |
| 同板块总仓位上限 ≤ 总仓位的Y% | 防止板块连锁反应 |
| 加速爆发后的仓位应逐步降低 | 高估值隐含风险增大 |

### 2. 黑天鹅预警信号

虽不可完全预判，但以下信号可提高警惕：
- 近期有解禁到期的个股
- 大股东质押比例高的个股
- 前期涨幅已大的核心人气股
- 板块已连续加速3天以上

### 3. 盘中应对

| 情况 | 应对 |
|------|------|
| 集合竞价跌停 | 立即挂单，争取流动性 |
| 板块整体崩盘 | 非减持个股未必需要立刻清仓，但应评估错杀修复机会 |
| 次日继续跌停 | 保持挂单，不抱幻想 |

## 与交易体系的关系

- **L4执行控制层**: 单票仓位上限是硬约束，不可因"逻辑正确"而突破
- **L3个股选择**: 评估个股时需检查减持风险因素（解禁、质押、涨幅）
- **卖出纪律**: 黑天鹅触发后，止损速度是第一位的

## 开放性问题

- 减持黑天鹅后，基本面未变的同板块个股是否存在错杀修复机会？
- 如何在板块爆发后第三天评估追高性价比？是否有客观指标（如连板高度、换手率）可提前预警？
