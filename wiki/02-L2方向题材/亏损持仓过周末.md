# 亏损持仓过周末（Holding Losers Over Weekend）

## 定义

在周五收盘时，账户中仍持有处于亏损状态的仓位，承受周六至周日的2天消息面和情绪面缺口风险。

## 核心风险

| 风险维度 | 描述 |
|----------|------|
| **时间缺口** | 2天（48小时+）无法进行任何操作，市场发生任何事情都只能被动承受 |
| **消息风险** | 周末可能出利空公告、监管问询、行业政策变化 |
| **情绪累积** | 持仓者的恐慌在周末发酵，周一开盘形成集中抛压 |
| **复利破坏** | 一次周末大幅低开可摧毁数周积累的利润——如思美传媒单周末亏损26% |

## 典型案例：思美传媒（2024-10-31 → 11-04）

```
10/31（周四）断板日买入 7.61元 → 未止损
11/01（周五）持仓 → 未止损
11/02-03（周末）→ 无法操作
11/04（周一）低开 → 5.65-5.73割肉
结果：单周末亏损 26%，约 -7,082 元
```

这个案例不仅是"持仓过周末"的问题，更是"[[概念/断板日买入]] + 持仓过周末"的致命组合——断板票在周末后面临的抛压尤其大。

## 交易纪律

| 规则 | 内容 |
|------|------|
| **周五强制审核** | 每周五收盘前15分钟，强制检查所有持仓的盈亏状态 |
| **亏损仓位清仓** | 任何处于亏损状态的仓位（特别是断板票、高位票、退潮票），周五收盘前必须清仓 |
| **例外条件** | 仅允许深度盈利（浮盈>10%）且基本面逻辑完好的仓位过周末 |
| **问自己** | "如果周一开盘低开-5%，我能接受吗？如果不能，现在就卖。" |
