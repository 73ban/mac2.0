# 复盘空洞化

## 定义
**操作频率上升的同时，复盘深度系统性地下降**，形成“越操作越不记录、越不记录越乱操作”的负向螺旋。是系统性纪律崩溃的前兆信号。

## 产生机制
1. **短期盈利或焦虑驱使高频交易** → 复盘时间被挤压
2. **复盘深度下降 → 缺乏对操作的检验** → 错误不被纠正
3. **错误积累 → 更大的亏损焦虑** → 更频繁地调仓换股
4. **循环加速** → 复盘彻底沦为形式

## 5月12日证据
复盘模板中 12 个分析维度：
| 维度 | 状态 |
|------|------|
| 赚钱效应 | ❌ 缺失 |
| 盘面关键信号 | ❌ 缺失 |
| 热门板块拆解 | ❌ 缺失 |
| 连板天梯 | ❌ 缺失 |
| 资金流向 | ❌ 缺失 |
| 消息面 | ❌ 缺失 |
| 关键反思 | ⚠️ “当日操作逻辑待回顾”（等于承认未记录） |
| 明日操盘要点 | ⚠️ “待定” |
| 关注方向 | ⚠️ “待定” |

**仅 4 个维度有数据**（盘前背景、今日操作、资金概览、心态/纪律），完成度 **33%**。

## 与正常复盘的对比
正常复盘应包含：赚钱效应判断 → 盘面信号解读 → 板块拆解 → 资金流向验证 → 消息面校验 → 逻辑记录 → 明日计划。5月12日这一链条**完全断裂**。

## 干预策略
1. **复盘数据完整度纳入硬性纪律**：低于 70% 时次日强制减仓至半仓以下
2. **简化复盘模板的备选方案**：当时间不足时，至少填写“赚钱效应+盘面关键信号+关键反思”三个核心维度
3. **每周复盘检查**：周末统计每日复盘完成度，连续两日 < 70% 启动“轻仓周”
