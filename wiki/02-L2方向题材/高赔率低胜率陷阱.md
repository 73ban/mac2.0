# 高赔率低胜率陷阱

## 定义

**高赔率低胜率陷阱**指交易者被一个机会的巨大潜在收益所吸引，而系统性地低估了其失败概率，从而导致整体期望值为负的行为模式。表面上看“赚一次抵十次”，但实际上“亏十次才能赚一次”。

## 核心公式

期望值 = (胜率 × 盈利) - (败率 × 亏损)

**陷阱的关键**：你的大脑自动放大了“盈利”的数字，却把“胜率”四舍五入到了50%以上。而真相往往是胜率远低于50%。

```
你看到的：20%涨幅 × 50%胜率 - 10%跌幅 = 正期望
真实的：20%涨幅 × 15%胜率 - 10%跌幅 = 负期望（-5.5%）
```

## 交易 vs. 赌博的分界线

| 维度 | 交易 | 赌博（高赔率低胜率） |
|------|------|------|
| 优势来源 | 市场结构、信息差、模式复利 | 偶然运气 |
| 胜率 | 可量化、可持续 | 不可量化、不确定 |
| 赔率 | 在合理框架内 | 被夸大幻想 |
| 仓位 | 受限于风险预算 | 容易无限制加仓 |
| 决策依据 | 模式识别+环境匹配 | “感觉要反弹了” |
| 止损 | 预先设定且执行 | “再等等，会回来” |

## 中粮资本10/15案例

```yaml
标的：中粮资本（002423）
机会类型：9板断板后的“修复博弈”
真实胜率：极低（环境退潮+不在主线+获利盘压力+梯队压缩）
真实赔率：不对称——下跌空间>上涨空间
仓位：6.4万（>盈利操作的4.8万）
决策模式：赌博（“赌不会A杀”）
```

**这是典型的高赔率低胜率陷阱**：
- 你追逐的是一个“可能反弹10-20%”的故事
- 但你忽略了这个故事的失败概率>80%
- 真正的期望值是负的
- 仓位却比盈利交易更大

## 为什么它会反复出现

1. **近因效应**：最近赚了钱（国民技术+10.3%），大脑告诉你“我能判断对”
2. **可得性启发**：你见过断板后修复的案例（如双成药业），就高估了修复的概率
3. **损失厌恶的镜像反转**：不是怕亏，而是怕踏空——“万一修复了呢？”
4. **筹码已经动了**：一旦买入，沉没成本让你更难客观评估

## 识别清单

在做任何一笔交易前，自查以下问题：

- [ ] 我知道这笔交易的**真实胜率**吗？（不是感觉，是基于历史统计）
- [ ] 如果胜率只有20%，我还愿意做吗？
- [ ] 我的仓位是否超过了我对这个判断的“确信程度”？
- [ ] 我是在“把握机会”还是在“防止踏空”？
- [ ] 如果有同等吸引力的机会出现，我是否有资金参与？（资金机会成本）
- [ ] 如果亏损，这笔钱对我的账户意味着什么？

**如果其中任何一项让你犹豫，那这笔交易可能就是高赔率低胜率陷阱。**
