# 拖油瓶

## 定义

**拖油瓶**是指持续亏损、持续持有、反复出现在仓位中、未能果断清仓的低效仓位。它不是一次性的错误买入，而是一种**持续占据仓位和注意力的慢性失血**。

## 核心特征

| 特征 | 说明 |
|------|------|
| 持续亏损 | 从持有到最终清仓，累计亏损逐步扩大 |
| 持续持有 | 多次交易日中出现，未被及时处理 |
| 隐性成本 | 占用资金、占据仓位、消耗注意力 |
| 转化困难 | 持有者倾向于等待反弹，而非果断清仓 |
| 连锁影响 | 最终亏损清仓后容易触发[[概念/割肉-追涨循环]] |

## 典型案例：红四方（603395）

- **时间跨度**：2024年12月初 → 12月10日清仓
- **每日亏损累积**：多次出现在持仓中，持续浮亏
- **最终结局**：-3.0%清仓（-553元）
- **连锁效应**：清仓资金立即追入永安行（涨停价23.01）、华扬联众（涨停价18.38）、汇金科技（涨停未封追买）
- **核心教训**：如果在亏损-1.5%时果断清仓，可能避免（1）更大的亏损（2）追涨停换仓带来的风险暴露

## 与联动概念的机制

```
拖油瓶持续亏损
  → 占用资金、消耗注意力、积累挫败感
  → 最终被迫清仓（通常是亏损扩大的时候）
  → 急于扳回 → [[概念/割肉-追涨循环]] 启动
  → 清仓资金追入高位涨停票
  → 新仓位面临更高风险（一字炸板/次日无溢价）
```

## 预防策略

1. **设定最大容忍亏损线**：超过阈值，无条件清仓
2. **每日持仓审计**：每个持仓必须有持仓理由，理由消失即清仓
3. **拖油瓶一经识别，次日早盘必须处理**：不允许拖延（参考[[策略/退潮票早盘清仓]]的精神）
4. **清仓后冷静期**：亏损清仓后至少等待30分钟再开新仓，打断[[概念/协同亏损]]链条
