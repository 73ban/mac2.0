# 隔夜信息风险——盘后利空与信息不对称

## 定义

**隔夜信息风险**是指在交易时段内基于某一逻辑（通常是利多）建仓后，收盘后出现**反向信息**（如公司公告、政策变动、关联方表态），导致次日开盘即面临亏损扩大的风险。

其核心机制是**信息不对称**：交易时段内掌握的信息不足以预判盘后利空，而盘后利空在T+1开盘时直接作用于价格。

## 与一字跌停流动性归零的关系

隔夜信息风险是[[概念/一字跌停流动性归零]]的**上游近亲**：

```
隔夜信息风险（T日盘后利空发酵）
  → T+1低开/急跌
  → 极端情况：一字跌停流动性归零
```

区别在于：
- **隔夜信息风险**：强调"信息不对称"这一**成因**
- **一字跌停流动性归零**：强调"无法卖出"这一**后果**

## 5/18 典型案例：兆易创新 + 合肥城建

### 时间线

| 时间 | 事件 | 用户认知 |
|------|------|----------|
| T日盘中 | 长鑫Q1净利330亿，存储板块暴涨 | 利多强烈，板上追入兆易创新 |
| T日盘中 | 合肥城建一字板涨停 | 视为存储板块强度信号 |
| T日盘后 | 合肥城建异动公告：投资长鑫仅6,000万、不享有控制权 | 信息尚不可知 |
| T+1预期 | 存储板块分歧，一字板可能开板 | 兆易创新面临扩大亏损风险 |

### 风险链条

```
T日：基于"利多单向"认知建仓
  → T日盘后：关联方（合肥城建）主动切割
  → T+1开盘：板块预期反转
  → 持仓暴露于信息不对称的惩罚中
```

### 教训

1. **盘中涨停≠逻辑可隔夜**：一字板票可能盘后发布异动公告
2. **关联标的的异动公告风险**：不仅关注持仓标的，还要关注板块内一字板票的公告风险
3. **退潮日+消息面的脆弱性**：退潮日中消息面的利多-利空反转更容易引发板块踩踏

## 可预判的信号（有限）

某些信号可能预示隔夜信息风险：

| 信号 | 说明 | 可靠度 |
|------|------|--------|
| 关联标的连续一字板 | 异动公告概率上升 | 中 |
| 题材无实质映射 | 见[[概念/题材无实质映射]] | 中 |
| 公告窗口期（季报/异动触发） | 强制披露时点 | 高 |
| 退潮日中的高位题材 | 资金敏感，利空放大 | 中 |
