# 板块崩塌独苗补跌陷阱

## 模式定义

当某个板块出现系统性暴跌（跌幅>5%）时，板块内仅剩的涨停“活口”在次日极大概率（>70%）补跌。此类活口看似是“弱转强”或“龙头挺进”的信号，实则是极高风险的陷阱——市场情绪崩坏后，孤独的涨停股不具备独立走强的生态。

## 典型案例：诚邦股份（2026年5月21日）

| 维度 | 描述 |
| :--- | :--- |
| 板块状态 | 存储芯片板块 -5.28%，系统性暴跌 |
| 活口状态 | 诚邦股份 4板涨停，板块内唯一活口 |
| 次日预期 | 极高概率补跌 |
| 危险信号 | 无板块支撑 → 封单可能瞬间崩塌 → 流动性归零 |
| 陷阱逻辑 | 逆势涨停吸引跟风 → 次日低开补跌锁死流动性 |

## 识别特征

- 板块整体跌幅 >5%
- 板块内涨停家数 ≤2（最好是唯一活口）
- 该活口无基本面催化支撑
- 连板高度 ≥3板（位置危险）
- 市场整体处于退潮预警状态（见 [[概念/退潮预警信号]]）

## 为什么这是陷阱而非机会

即使是板块内“最强”的活口，在板块崩塌后也面临：
1. **补跌动能**：板块整体趋势向下，独苗难以逆势
2. **流动性枯竭**：次日低开后没有承接盘
3. **情绪传染**：退潮期高位板整体脆弱，独苗会被连累
4. **虚假弱转强**：看似扛住板块暴跌，实则是最后一棒

这与 [[概念/弱转强的生存门槛]] 直接关联：环境不配合时，弱转强信号是反向指标。

## 交易纪律

当识别到板块崩塌独苗时：
- **立即止盈/止损**，不等到次日
- **禁止追入“逆势活口”**，无论封单多强
- **警惕次日一字跌停**：流动性归零风险真实存在
