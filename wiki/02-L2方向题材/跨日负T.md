# 跨日负T

> **定义**：日内或跨日卖出后，以更高价买回同一标的，形成"低卖高买"的负T。核心驱动是恐慌→贪婪的跨日转换。

## 经典案例（2024-10-10）

**中芯国际**
- 10/9买入 → 10/10卖出价：74-76
- 10/10买回价：78-79
- **买回价比卖出价高4-5元**

**寒武纪**
- 10/9买入价：380
- 10/10卖出价：333-349
- 10/10买回价：377
- **买回价比卖出价高28-44元**

## 心理机制

```
盘中暴跌 → 恐慌清仓（卖在低位）
     ↓
数分钟/几小时后反弹 → 踏空焦虑
     ↓
追高买回（买在高位）
     ↓
次日回落 → 继续亏损
```

## 与恐慌-贪婪交替的关系

跨日负T是[[概念/恐慌-贪婪交替]]的跨日翻版。区别在于：
- **日内恐慌-贪婪交替**：同一交易日内发生，如10/9同标的卖出后追回
- **跨日负T**：清仓（恐慌）与追回（贪婪）发生在不同交易日，中间可能隔夜

两者共享同一个防范规则：**同一标的卖出后，30分钟内禁止买回，且当日禁止以高于卖出价的价格重新买入该方向任何标的**。

## 防范规则

1. **反向操作禁令**：同一标的/同一方向卖出后，当日禁止重新买入
2. **价格红线**：禁止以高于卖出价买入同一标的（包括次日）
3. **清仓冷静期**：全线清仓后至少等待一个完整交易日再重新建仓同一方向
