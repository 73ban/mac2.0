# AI算力概念板块

## 概念定义
**AI算力/智算中心** 是指支撑人工智能模型训练与推理的计算基础设施，包括AI服务器、数据中心（IDC）、边缘计算节点、算力调度平台等。随着大模型技术爆发，算力基建化成为确定性趋势。

## 逻辑主线
```
大模型需求爆发 → 算力基建化 → IDC扩建/边缘节点部署
                              → AI服务器需求增长（华为昇腾生态）
                              → 算力调度平台价值提升
```

## 风险点
- **高度主题内聚**：同日集中建仓四只标的形成方向性暴露，主线回调时集体承压
- **估值与情绪驱动**：多数标的受AI主题情绪影响大，基本面验证存在时滞
- **融资放大风险**：拓维信息使用融资杠杆，放大了板块回调时的亏损幅度
