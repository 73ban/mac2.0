# 补涨与轮动逻辑

> **定义**：区分两种本质不同的参与逻辑：**补涨**（板块内核心标的的前期涨幅修复）与 **新预期差**（新的产业逻辑首次被市场关注），两者在持续性、参与方式和风险特征上完全不同。

## 核心区分

| | 补涨逻辑 | 新预期差逻辑 |
|---|---|---|
| **本质** | 板块内已有方向的空间修复 | 新的产业映射逻辑被首次定价 |
| **驱动** | 横向比较：板块涨了我没涨 | 纵向逻辑：新催化剂→新认知→新买入 |
| **持续性** | 修复到位后空间消失 | 逻辑能持续发酵，可能走出趋势 |
| **参与方式** | 轻仓试探，高开不追 | 等分歧后的低吸建仓 |
| **风险** | 修复就是补涨上限 | 逻辑是否被市场认可的不确定性 |
| **5/29案例** | 深南电路（AI服务器PCB核心） | 高澜股份/英维克（AI散热新逻辑） |

## 补涨标的的风险特征

- 补涨逻辑的**上限**是板块其他标的已有的涨幅 — 修复到位后边际动力消失
- 补涨 ≠ 它能成为新的龙头 — 板块内龙头地位是资金结构决定的，补涨不改变地位
- 高开补涨是最大陷阱 — 高开已经兑现了预期差，追入只剩风险

## 与新预期差的差异

- 新预期差是**从0到1** — 逻辑是新的，天花板是打开的，持续性和弹性都优于补涨
- 但新预期差需要**市场确认** — 逻辑虽然新，如果当天没有资金跟进，就是逻辑自嗨

## 使用纪律

- 补涨预期差：轻仓试探（1-2成），高开超过3%放弃，快进快出
- 新预期差：等分歧确认（次日或当日下午企稳），才进入持仓模式
- 不要在“补涨修复”上投入“新逻辑”级别的仓位和时间
