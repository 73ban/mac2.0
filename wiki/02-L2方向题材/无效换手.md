# 无效换手

## 定义
**无效换手** 指在同一交易日内，对同一只股票（或通过换仓）进行买卖，但**买卖价差为零或极微薄**，扣除手续费、印花税后净亏损，且不产生任何正收益的交易行为。

## 核心特征
1. **同价或微差价**：买入与卖出价格几乎完全一致（如 ±0.01 元）
2. **净损手续费**：成交金额较大，但价差无法覆盖佣金和印花税，实际亏损
3. **决策矛盾**：买入后立即卖出，或卖出同日又买入，缺乏明确的持仓逻辑
4. **换仓驱动**：常因追逐新的“机会”而强行切换，表现为旧仓位的低质量出清

## 典型案例：2026-05-13 永杉锂业
- **买入**：1,900 股，23.21 元，金额 44,099 元  
- **卖出**：2,500 股，23.21 元（其中 1,900 股为当日买入，600 股为旧仓）  
- **价差**：0 元  
- **净亏损**：手续费 5.07 + 6.67 + 印花税 29.01 ≈ 40.75 元  
- **推测动机**：卖出回收资金约 58,025 元，随后买入美利云 66,120 元，疑似为腾出仓位追逐新目标，同时牺牲了永杉锂业的持仓逻辑。

## 行为解释
- **贪婪驱动**：看到美利云的“机会”后，急于换仓，不择时机地抛出永杉锂业，甚至接受同价卖出。
- **决策疲劳/冲动**：未等待合理的止盈止损点，买入后无理由离场，反映出交易计划的缺失。
- **高频换手模式**：与 [[总结/9月→12月三阶段演变]] 中记录的“退潮修复期的高频换仓+追涨停模式加速”一致，属于高成本、低收益的行为模式。

## 认定标准
| 条件 | 说明 |
|------|------|
| 买入→卖出时间 ≤ 1 个交易日 | 日内回转或次日开盘立即卖出 |
| 净价差 ≤ 0.1%（或绝对价差 ≤ 0.05 元） | 几乎无价格贡献 |
| 净损益为负 | 扣除费用后亏损 |
| 无明确止损或止盈条件 | 非计划内交易 |

## 防范措施
1. **禁止同价卖出**：除非触及硬止损，否则不在买入价附近卖出。
2. **换仓冷静期**：新开仓必须提前至少 15 分钟决策，避免盘中临时追涨换仓。
3. **每日换手率上限**：限定当日买卖股数或金额，防止高频转圈。
4. **复盘检查**：每日交割单必须检查是否存在“无效换手”，记录原因并计入错误库。

## 与现有概念的关系
- 若同价卖出源于恐慌，则交叉 [[概念/恐惧驱动的纪律执行]]  
- 若为追新标的，则属于 [[概念/贪婪驱动的纪律撤销]]  
- 经常出现在退潮期、修复期等方向不明的市场环境中
