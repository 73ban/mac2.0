# 联动锚点——正向的以A判B决策框架

## 定义

**联动锚点**是一种**理性决策框架**：用一个关联标的的走势作为另一个标的操作决策的**参考信号**，以此提高判断的确定性。这与[[概念/锚定效应]]（非理性的、由成本价锚定驱动的行为偏差）形成**正向对照**。

### 核心公式

```
标的A的走势反馈 → 验证/否定 对标的B的假设 → 决定B的操作方向
```

## 与负面锚定效应的对比

| 维度 | 联动锚点（正向） | [[概念/锚定效应]]（负面） |
|------|----------------|------------------------|
| 锚定对象 | 关联标的的**实时走势** | 自身的**历史价格/成本** |
| 逻辑基础 | 板块协同/产业链联动 | 心理偏差（沉没成本、损失厌恶） |
| 驱动 | 理性推理 | 情绪驱动 |
| 典型操作 | "京能封板稳住→大唐水下低吸安全" | "亏了10%等回本再卖" |
| 结果 | 提高判断确定性 | 延迟止损、扩大亏损 |

## 5/18 应用案例：京能电力→大唐发电

### 锚定链条

```
京能电力（600578）
  4连板换手板（4次开板），电力板块龙头
    ↓ 走势反馈
大唐发电（601991）
  绕10日异动博弈标的
    ↓
京能4→5成功封板  →  大唐水下低吸安全（情景B）
京能崩盘闷杀     →  大唐立即减仓/止损（情景D）
```

### 为什么这是理性的？

1. **因果逻辑清晰**：京能是电力板块的4板龙头，其强弱直接反映板块情绪
2. **信号可观测**：京能的封单量、是否开板是客观可验证的
3. **操作可执行**：京能的反馈直接映射到具体的ABCD情景选择

## 构建联动锚点的条件

| 条件 | 说明 | 不满足的风险 |
|------|------|------------|
| **真实的板块/产业链关联** | 两只标的之间存在可验证的业务或题材逻辑关联 | 伪关联导致错误锚定 |
| **锚点标的有先行指标属性** | 锚点标的走势能**先于**或**同步**反映板块情绪（通常是大市值或高连板龙头） | 滞后指标无参考价值 |
| **单向映射清晰** | A的走势→B的操作有一个明确的"如果-那么"规则 | 模糊映射无法指导操作 |

## 与绕异动博弈的组合

在[[概念/绕10日异动博弈]]中，联动锚点是**安保条件**——没有锚点时，绕异动博弈退化为纯赌博：

- 锚点稳住 → 情景B（水下低吸做T）有安全垫
- 锚点崩 → 情景D（止损）的信号明确
