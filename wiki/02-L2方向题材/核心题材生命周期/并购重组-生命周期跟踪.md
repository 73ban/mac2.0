# 并购重组 / 事件驱动 - 生命周期跟踪

更新时间：2026-06-13

## 当前定位

并购重组是小资金短线必须重点跟踪的方向之一，因为容易出现一字、连板和预期差。但它对信息真实性要求最高，必须坚持 S1/S2 优先。

```yaml
theme: 并购重组/事件驱动
current_stage: 每日动态筛选
truth_grade: 必须S1/S2，S3/S4不交易
preferred_modes:
  - 并购重组预期差
  - 一字定方向扩散
  - 连板空间破局
  - 竞价弱转强
forbidden:
  - 小作文重组
  - 连续一字高潮后追后排
```

## 筛选重点

| 维度 | 要求 |
|---|---|
| 信息来源 | 公告、交易所、公司披露、权威媒体 |
| 资产质量 | 是否有产业价值、稀缺性、想象空间 |
| 市值弹性 | 小市值更容易短线弹性 |
| 市场理解 | 市场是否还没充分定价 |
| 竞价表现 | 9:20 不撤，9:25 仍强 |
| 板块扩散 | 是否带动同类重组或主线方向 |

## 作战室三类票

| 类型 | 特征 | 打法 |
|---|---|---|
| 一字核心 | 公告超预期，大单一字 | 不追买，观察方向和扩散 |
| 换手核心 | 有重组逻辑且换手承接强 | 半路/打板确认 |
| 预期差票 | 市场尚未充分理解 | 小仓观察，确认后提高 |

## 主升确认条件

1. 公告或正式披露强。
2. 题材不只是单票，而有扩散。
3. 一字核心封单真实。
4. 换手核心承接强。
5. 当日市场情绪支持接力。

## 降级信号

1. 公告弱于预期。
2. 重组小作文被证伪。
3. 一字撤单。
4. 高位重组票开板大面。
5. 连板情绪退潮。

## 龙虾/老虎输入要求

每天必须给大鸟：
```text
今日并购重组公告清单
最重要3只重组票
公告原文链接或摘要
是否S1/S2
9:15/9:20/9:25竞价表现
是否带动板块扩散
```

## 禁止

1. 无公告只靠传闻。
2. 连续一字后买后排。
3. 情绪退潮还做接力。
4. 不设退出条件。
