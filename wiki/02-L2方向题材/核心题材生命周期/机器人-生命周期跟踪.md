# 机器人 / 人形机器人 - 生命周期跟踪

更新时间：2026-06-13

## 当前定位

机器人是高关注度方向，容易受产业会议、订单、技术路线、特斯拉链等催化影响。短线关键不在“长期空间大”，而在是否有核心票带队和板块梯队。

```yaml
theme: 机器人/人形机器人
current_stage: 轮动到主线候选，需核心确认
truth_grade: S2/S3混合
preferred_modes:
  - 新主线去伪存真
  - 强板块前排半路
  - 主线核心低吸
  - 低位补涨前排
forbidden:
  - 只看长期空间追后排
  - 无核心确认做补涨
```

## 题材结构

| 角色 | 个股 | 作战用途 | 需要确认 |
|---|---|---|---|
| 核心链条 | 三花智控、拓普集团、绿的谐波 | 判断机器人是否被资金认可 | 是否持续走强，是否进入热门榜 |
| 零部件/弹性 | 大族数控、信维通信、东山精密 | 题材扩散和弹性 | 是否跟随核心，而不是孤立异动 |
| 交叉方向 | AI硬件、消费电子、工业自动化相关票 | 题材扩散 | 是否被市场明确归因到机器人 |
| 风险锚 | 三花智控、绿的谐波 | 判断题材承接 | 核心回落无修复时降级 |

## 主升确认条件

1. 核心票持续走强，而不是单日脉冲。
2. 板块有多个分支共振。
3. 热门榜出现机器人方向集群。
4. 题材分歧后能修复。
5. 有明确产业催化或订单预期。

## 降级信号

1. 核心票冲高回落。
2. 只有低位小票乱动。
3. 机器人消息很多但板块不涨。
4. 资金回流 AI/半导体，机器人变成轮动套利。

## 作战室打法

1. 主升确认前，只做前排，不做后排。
2. 核心趋势票优先低吸，不盲目追高。
3. 低位补涨要看核心是否继续打开空间。
4. 如果只是轮动，隔日预期要降低。

## 禁止

1. 长期想象力替代短线资金确认。
2. 后排补涨当龙头。
3. 核心不强时硬做机器人。
