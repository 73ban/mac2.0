# 隐性大亏心理遮蔽

## 定义

**隐性大亏心理遮蔽** 是指：已清仓的重大亏损被"清仓=结束"的心理防御机制从日常复盘视野中清除，不再被主动审视，但其心理后遗症（恐惧、急于扳回、对后续决策的扭曲）持续影响新持仓的行为模式。

## 核心机制

```
重大亏损 → 清仓（"终于结束了"） → 心理归档 → 不再复盘 →
→ 亏损的心理后遗症持续影响新决策
→ 但用户注意力被新持仓的浮亏吸引（注意错位）
```

### 关键特征

1. **视觉消失：** 持仓量=0，从自选列表或持仓页面消失
2. **心理归档：** "这笔已经结束了，看新的"
3. **未消化：** 亏损的教训未被提取，亏损的情绪未被处理
4. **隐形传导：** 恐惧或扳本心理渗透到后续持仓，但用户不自知

## 实证：韶能股份（2026-05-15）

- 亏损额：-36,903元（当前可见最大单笔亏损）
- 持仓状态：已完全清仓（持仓量=0）
- 注意力现状：当前持仓中两只股票仅浮亏合计-6,576元，但占据用户的主要关注
- **注意错位：** 真正伤筋动骨的亏损已经"消失"，注意力却集中在两个相对较小的浮亏上

## 与相关概念的区别

| 概念 | 亏损状态 | 心理机制 | 影响路径 |
|------|----------|----------|----------|
| **隐性大亏心理遮蔽** | 已清仓 | 清仓=归档，不再复盘 | 心理后遗症隐形传导 |
| [[概念/协同亏损]] | 可已清仓可仍在仓 | 前一笔亏损扭曲后一笔决策 | 行为链式传导 |
| [[概念/锚定效应]] | 仍在仓 | 成本价作为参照点 | 阻碍止损决策 |

三者可叠加：韶能大亏（隐性遮蔽）→ 扭曲线性心态（协同亏损）→ 持有数据港/美利云时以成本锚定拒绝止损（锚定效应）。

## 防范措施

1. **清仓≠结束：** 每笔清仓交易强制写入[[错误/]]或复盘日志
2. **亏损金额排序审查：** 定期按已实现亏损排序，确保大亏不被"持仓=0"遮蔽
3. **交割单回溯：** 每月回溯交割单，标记已清仓的重大亏损并追问原因
4. **新开仓前自检：** "我现在的决策是否受到之前某笔大亏的影响？"
