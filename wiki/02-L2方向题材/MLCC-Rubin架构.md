## 二、题材逻辑

### 2.1 为什么炒

```
英伟达下一代Rubin架构 vs 当前GB300：
┌────────────┬──────────┬──────────┐
│ 增量方向   │ 市场认知 │ 炒作位置 │
├────────────┼──────────┼──────────┤
│ PCB        │ 高       │ 高位     │
│ 存储       │ 高       │ 高位     │
│ MLCC       │ 低 ←     │ 低位 ←   │ ← 我们的机会
└────────────┴──────────┴──────────┘
```

MLCC是英伟达AI服务器中用量最大的被动元件，Rubin架构对MLCC的数量和规格要求大幅提升。但散户第一反应是PCB/存储，MLCC认知度最低→**预期差最大**。

### 2.2 预期差来源

| 维度 | 内容 |
|------|------|
| 散户认知差 | 散户知道PCB/存储，不知道MLCC也是Rubin增量 |
| 位置差 | PCB/存储龙头已高位，MLCC标的相对低位 |
| 空间差 | MLCC单机价值量增量可观，但市场未充分定价 |

### 2.3 持续性判断

| 因素 | 判断 |
|------|------|
| 产业真实度 | ⭐⭐⭐⭐⭐ 真实增量，非概念炒作 |
| 市场认知阶段 | 🟡 发酵中期，尚未完全扩散 |
| 龙头高度 | 风华已从~36涨到~60+，但仍有空间 |
| 资金参与度 | 🟢 6/9风华涨停+洁美联动，资金认可 |
| 时间窗口 | 预计仍有1-3周活跃期 |
