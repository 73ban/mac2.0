# 先行指标监控体系

## 定位

**L1.5 层级** — 介于宏观风格判断（L1）与中观景气追踪（L2）之间的**时间序列预警层**。

现有 Wiki 的"宏观-中观-微观"分层是**空间维度**，本体系补充**时间维度**：不仅知道"看什么"，更知道"什么时候看、谁先动"。

## 核心定义

根据指标变动轨迹与目标周期（宏观经济、行业景气、股价走势）的时间关系，将指标分为三类：

| 类型 | 定义 | 角色 |
|------|------|------|
| **先行指标** | 比目标周期更早发生转折 | 预警——趋势可能来临 |
| **同步指标** | 与目标周期同时转折 | 确认——趋势已经到来 |
| **滞后指标** | 比目标周期更晚转折 | 验证——趋势确实确立 |

理论基础（Leeuw 1991）：
1. **生产-消费时滞**：上游指标的变动远早于终端需求数据
2. **调整成本差异**：反应快的指标必然先行
3. **预期敏感性**：消费者信心、股指等在实际经济活动前反应
4. **外部冲击传导**：政策传导路径中先受影响的环节成为先行信号

## 三层部署框架

### L1 宏观层：普林格周期

- **先行指标**：社融增速（信贷周期）
- **同步指标**：工业增加值（生产周期）
- **滞后指标**：CPI（通胀周期）

联动检验 [[概念/风格撕裂]]：若社融触底反弹但CPI未攀升 → 风格撕裂获经济周期层面支撑。

详见 [[概念/普林格周期六段论]]

### L2 行业层：先行指标链

**最具实操价值的落地应用——PCB行业**：

```
覆铜板价格/出货量 → PCB厂商季度营收 → AI服务器订单
   [先行 1-2季度]      [同步]            [下游催化]

PCB设备订单（芯碁微装LDI）
   [先行 6-9个月]
```

**预警前置**：在深南电路季报出来之前，提前1-2个季度通过覆铜板/设备订单判断方向。

详见 [[数据/PCB行业先行指标追踪]]

### L3 技术层：领先vs滞后互补

| | 领先指标 | 滞后指标 |
|---|---------|---------|
| 代表 | RSI、KDJ | MA、MACD |
| 适用 | 震荡市 | 趋势市 |
| 当前AI算力链 | **禁用RSI判断顶部** | **持股参考均线系统** |

**互补纪律**：
- 领先指标负责"找信号" —— 提供预警线索
- 滞后指标负责"验信号" —— 确认趋势转化
- 配合量能和价格形态 —— 三维验证

与 [[概念/缩量蓄力放量突破]] 互补：量价模式判断中军介入时机，指标体系提供趋势环境指引。

## 核心操作纪律

> **先行指标只负责预警，不负责决策。真正动手仍应等待同步指标确认。**

本质原因：
- 领先期数不稳定，会随经济结构变化漂移
- 任何基于历史领先期数的量化阈值都面临 [[概念/过度拟合风险]]
- 先行指标解决"什么时候该开始关注"，而非"什么时候该动手"
