# 消息催化预期差评分

> 建立：2026-06-09 | 维护：codex
> 
> **核心目的：从每日数百条公告/新闻中，筛出对明日股价有真实影响的消息，量化预期差。**

---

## 一、什么是预期差

> **预期差 = 市场尚未定价的利好/利空 — 已被消化的消息**

你关心的不是"出了什么消息"，而是"出了什么市场还没反应过来的消息"。

## 三、预期差评分（1-5分）

### 5分 — 重大预期差（次日大概率涨停）
```
条件（满足任一）：
- 签订重大合同（金额超年营收30%）
- 与英伟达/华为/特斯拉级别企业宣布合作
- 并购重组获批复/完成
- 财报利润超预期100%以上
- 获得重大牌照/批文（如创新药获批）
```

### 4分 — 显著预期差（次日大概率高开）
```
条件（满足任一）：
- 业绩预增50%以上
- 获得重要客户订单
- 新产品/技术发布
- 行业政策明确利好
- 美股同板块标的涨超10%
```

### 3分 — 中等预期差（可能影响走势）
```
条件（满足任一）：
- 业绩预增20-50%
- 获得普通订单
- 管理层增持
- 机构密集调研
- 板块轮动到该方向
```

### 2分 — 弱预期差（大概率已定价）
```
条件（满足任一）：
- 例行公告
- 股东大会通知
- 小股东减持
- 已知消息的延续
```

### 1分 — 无预期差
```
- 纯噪音、无实质内容
```

## 四、预期差判断三问

对每条消息，hermes 必须回答：

```
① 这个消息市场知道了吗？
   - 盘后出的 → 不知道 → 预期差大
   - 盘中已反应 → 知道 → 预期差小

② 明天开盘会有人因为这个消息买吗？
   - 会 → 预期差大
   - 不会 → 预期差小

③ 这个消息能持续影响几天？
   - 持续1天以上 → 可以追
   - 当天冲高回落 → 不能追
```

## 六、消息催化 → 交易决策

| 预期差 | 交易策略 | 仓位 |
|--------|----------|------|
| 5分 | 竞价直接扫，可以追高 | 可达满仓 |
| 4分 | 开盘观察5分钟，确认方向后上车 | 可达5成 |
| 3分 | 放入关注池，等分歧点低吸 | 不超过3成 |
| 2分 | 不作为主要交易依据 | — |
| 1分 | 忽略 | — |

### 关键原则

```
- 5分催化 + 属于今日主线板块 → A类标的
- 当天追不上 → 第二天分歧低吸（分歧日上车）
- 一致日只能卖不能买
- 消息催化票的3天窗口：D+1追 → D+2分歧可加 → D+3不涨就出
```

## 七、hermes 每日收盘后跑

```
1. 抓取当日所有公告（巨潮/东财）
2. 抓取当日重要新闻（财联社/华尔街见闻）
3. 按预期差评分筛出 ≥3分 的消息
4. 对齐关注池，标注是否在热门股TOP150内
5. 产出 03-消息催化股.md
```
