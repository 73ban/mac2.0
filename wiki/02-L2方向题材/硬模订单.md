# 硬模订单

> 产业链验证层级中的确定性分水岭，标志着从设计验证阶段进入准量产阶段

## 核心定义

**硬模订单**是人形机器人（及其他高端制造）产业链中，供应商收到的具备准量产意义的模具订单。它区别于前期的PPA框架协议（宽泛入围）和软模（设计验证），标志着产品设计已冻结、工艺已锁定，供应商被正式纳入量产计划。

## 在产业链验证层级中的位置

参见 [[人形机器人产业链验证层级]] 完整框架：

```
PPA框架协议 → 软模 → ★硬模★ → 明确交付计划 → 业绩兑现
   题材期      验证期    准量产      加速期        业绩期
```

硬模是**确定性分水岭**：
- 拿到硬模的企业退出概率大幅降低
- 意味着从几十家竞争收窄到少数核心供应商
- 投资重心从“入围”转向“份额”

## 硬模 vs 软模/PPA

| 维度 | PPA | 软模 | 硬模 |
|------|-----|------|------|
| 性质 | 宽泛入围 | 设计验证 | 准量产 |
| 确定性 | 低 | 中 | 高 |
| 淘汰率 | 高 | 中高 | 低 |
| 量纲 | 无 | 模糊 | 有排产计划 |
| 投资阶段 | 题材炒作 | 题材→确认 | 确认→加速 |

## 案例分析：特斯拉V3版机器人（2026年6月）

- 部分核心供应商已收到硬模订单
- 获得明确到9月份的周/月度排产计划
- 投资评估重心转向“份额与兑现”

## 投资应用

1. **识别核心资产**: 只关注已获硬模的公司，过滤掉仍处PPA/软模阶段的炒作标的
2. **追踪份额**: 在硬模企业中，根据排产计划推断各家的份额占比
3. **跟踪兑现**: 硬模→交付→确认收入，是业绩可追踪的链条
4. **注意风险**: 硬模≠立即贡献业绩，需时间转化为财报数字

## 批判性审视

- 硬模订单信息通常来自产业链调研（二阶信息），非公告级别
- 需交叉验证：是否有公告？是否有公开报道？
- 在A股市场，“硬模传闻”可能被利用于出货
- 特斯拉供应链信息通常高度保密，公开信息有限
- 硬模到量产仍有距离（工艺爬坡、良率提升等）
