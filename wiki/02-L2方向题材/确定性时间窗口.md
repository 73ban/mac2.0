# 确定性时间窗口

## 定义
已知未来某日会发生的高确定性事件（如IPO审核、政策发布、重大会议），在事件发生前存在一段明确的抢筹时间窗口。此窗口内资金有明确的做多动机，方向确定性远高于纯概念催化。

## 与“利好兑现”的关系
- **窗口期内**：资金抢筹，方向偏多
- **事件当天/后**：可能出现利好兑现（[[策略/利好兑现与靴子落地]]），即“buy the rumor, sell the news”

## 确定性分级
| 类型 | 确定性 | 案例 |
|------|------|------|
| 硬事件（IPO审核、政策落地） | 最高 | 长鑫科技5/27上会 |
| 产业趋势（供需缺口、技术迭代） | 中等 | PCB铜箔紧缺到2028年 |
| 概念催化（试飞、小作文） | 较低 | SpaceX星舰试飞成功 |

## 案例：长鑫科技 2026-05-27 科创板上会
```
T-2（周一）：抢筹窗口 → 确定性最强
T-1（周二）：最后抢筹日
T（周三）：上会结果决定方向
    → 通过：利好兑现，部分资金了结
    → 被否：利空，大幅回调
```

## 使用规则
1. **只在窗口期内交易**：事件落地前是安全期
2. **事件当天减仓或清仓**：除非有超预期成分
3. **优先级高于概念催化**：硬事件方向 > 概念催化方向
4. **配合纪律**：即使窗口期内，走势不对也要止损——不因“确定性”放弃纪律
