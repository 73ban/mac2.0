# 挂单过高的机会成本

## 定义

当持有亏损票时，出于“少亏一点”的心理将卖出挂单设在高于当前市场流动性能接受的价格，导致挂单无法成交、亏损继续扩大的现象。

## 机制

```
成本价 → 错定在“少亏”心理价 → 挂单脱离盘中波动区间 → 不成交
→ 当日亏损扩大 → 次日继续恶化 → 最终被迫在更低价格割肉
```

**核心错误**：将“我想卖多少钱”替代了“市场愿意给多少钱”。

## 案例：2026-05-26 综艺股份

| 参数 | 数值 | 含义 |
|------|------|------|
| 成本价 | 7.47 | 买入价 |
| 挂单价 | 6.87 | 试图-8.0%止损 |
| 当日最高 | 6.56 | 市场实际最高价 |
| 价差 | 0.31（+4.7%） | 挂单价完全脱离盘中流动性 |
| 收盘价 | 6.56 | 亏损-12.24%，比挂单价低4.7% |

**机会成本**：如果早盘以市价（约6.56）清仓3000股，可回收约19,680元，亏损约2,730元。但因挂单未成交，当日亏损继续扩大。

## 心理根源

- [[概念/锚定效应]]：被成本7.47锚定，6.87“看起来可以接受”
- [[概念/恐惧驱动的纪律执行]]（负面）：恐惧确认更大亏损 → 选择“多挂一点” → 结果失去全部流动性

## 与规则的关系

[[策略/退潮票早盘清仓]] 明确要求“退潮票不挑时间，早盘必须出”——隐含要求是**市价成交**，而非限价挂单。综艺股份的挂单失败是该规则的**反面教材**：挑价格 = 挑时机 = 无法成交。

## 规则嵌入

> **退潮票早盘清仓补充条款**：清仓必须以对手盘最优价格（买一价）成交，禁止限价挂单。
