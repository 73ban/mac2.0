# 极致分化

## 定义

**极致分化**是指缩量（或存量博弈）市场中，赚钱效应高度集中在少数高辨识度个股，而绝大部分个股阴跌或横盘的市场状态。

核心特征：
- **强者恒强**：连板股、龙头股批量涨停或连板
- **弱者阴跌**：非核心票大面积下跌或零涨幅
- **成交量萎缩**：市场整体缩量，增量资金不足
- **抱团效应**：存量资金向确定性最高、辨识度最高的标的集中

## 市场信号

### 可以识别的信号
1. 指数窄幅震荡或微跌，但连板梯队反而完整或强劲
2. 涨跌家数极度悬殊（如沪市483涨 vs 1,784跌）
3. 成交额持续萎缩，当日无放量迹象
4. 出现多个连板龙头共振（如此日 电光4板 + 南京3板 + 中百涨停）

### 历史案例

| 日期 | 关键特征 | 连板龙头 | 结果 |
|------|----------|----------|------|
| 2024-12-25 | 沪市5,222亿，483涨/1,784跌，4-3-2板梯队完整 | 电光科技、南京化纤、中百集团 | 待跟踪 |
| 待补充 | | | |

## 交易含义

### 机会面
- 高辨识度龙头获得**确定性溢价**（参见[[概念/确定性溢价]]），资金抱团使其连板概率更高
- 精准上车龙头（如此日买入电光科技）是高质量操作

### 风险面
- **流动性陷阱**：缩量环境下一旦情绪反转或放量杀跌，高标股流动性归零风险急剧放大（参见[[概念/一字跌停流动性归零]]）
- [[股票/华扬联众]]的教训：一字跌停封死后无法出逃，缩量环境下的连板是双刃剑
