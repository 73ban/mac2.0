# 供给端扰动

## 定义

**供给端扰动**指上游资源国政策变动（如出口限制、资源税上调、环保停产等）引发的短期供给收紧预期，其逻辑链为：

```
资源管控收紧 → 短期供给趋紧 → 矿价走强 → 矿企库存价值重估 → 业绩预期改善
```

## 核心矛盾：逻辑正确 ≠ 交易机会

完整的传导链需要 **6-12 个月**，短期股价反应的是**情绪预期**而非**业绩兑现**。这是[[概念/预期先行业绩滞后]]的经典场景。

## 催化力度四维评估模板

| 维度 | 关键问题 |
|------|----------|
| 事件级别 | 占全球/国内供给的比例多高？边缘扰动还是重大冲击？ |
| 确定性 | 政策是否已落地？还是传闻阶段？ |
| 持续性 | 短期情绪还是中期趋势？实际产能收缩需要多久？ |
| 催化强度 | 是新题材首次催化，还是老题材的审美疲劳式扰动？ |

**评分标准**：3项以上“中-高”才值得纳入交易观察。若“持续性”和“催化强度”均低，则**坚决不参与**。

## 参与纪律：双信号确认

1. **现货/期货价格同步走强**：情绪预期需要有现实价格背书
2. **资金共识形成**：主线级别的龙头涨停、中军放量

**在双信号确认之前，执行“知道→归档→观察”流程。最大风险不是错过机会，而是分散对主线的注意力和仓位。**
