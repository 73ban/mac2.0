# 踏空焦虑

> **定义**：错失行情启动窗口后产生的心理压力，驱动交易者用错误姿势追赶，形成"踏空→焦虑→过度补偿→方向跑偏→再次踏空"的恶性循环。

---

## 核心机制：四阶段循环

```
阶段1: 踏空           阶段2: 焦虑升级
行情已经启动，           "再不追就彻底
自己仓位不足             错过了"
     ↓                      ↓
阶段4: 再次踏空    ←   阶段3: 过度补偿
方向跑偏，新持仓          用分散姿势匆忙
没吃到主升浪              追进多个方向
```

## 在9月操作中的表现

| 日期 | 循环阶段 | 具体表现 |
|:---|:---|:---|
| 9/24-25 | 阶段1：踏空 | 启动日轻仓/踏空，错过两根阳线 |
| 9/26 | 阶段2→3：焦虑→过度补偿 | 建仓3只3方向，天风仅21%——想用20%仓位赚回100%踏空的涨幅 |
| 9/27 | 阶段4→1：再次踏空 | 清2换2偏离主线，天风→16%，新仓未吃到大涨幅 |
| 9/30 | 循环加速 | 再度换仓，天风→14.7%，方向再度偏离 |

### 关键诊断
> **踏空焦虑不是"害怕错过"——是"害怕追不上"。** 它的驱动力不是贪婪，是一种**落后感**："别人都赚了，我还在后面"。这种感觉让交易者做两件致命的事：
> 1. **提高交易频率**（用操作填焦虑）
> 2. **降低选择标准**（只要是涨的就行，不管是否主线）

## 踏空焦虑 vs 正常追赶

| | 踏空焦虑驱动的追赶 | 理性追赶 |
|:---|:---|:---|
| **仓位** | 分散多个方向，单票仓位不足 | 集中核心方向，单票重仓 |
| **标的** | 什么涨追什么，偏离主线 | 只追主线核心龙头 |
| **心态** | 紧张、匆忙、怕错过 | 冷静、有计划、按系统 |
| **结果** | 再次跑输，焦虑加剧 | 逐步跟上，焦虑消退 |

## 识别信号

- [ ] 买票前没有完整逻辑，只说"它涨得好"
- [ ] 同一天买进多只不同方向的票
- [ ] 每只票仓位都在20%左右，没有核心重仓
- [ ] 买完后立刻焦虑"会不会追高了"
- [ ] 盘后复盘发现自己说不出每只票的买入理由

## 干预机制

### 踏空焦虑触发 → 强制暂停
1. **识别**：发现自己在用分散姿势追赶（同日买≥3只不同方向票）
2. **暂停**：停止任何新开仓，当日剩余时间只观察
3. **重聚焦**：回答三个问题——
   - 现在的主线到底是什么？
   - 主线的核心龙头是什么？
   - 我为什么没有重仓它？是没识别、不敢、还是仓位已在别处？
4. **重新分配**：次日开盘前做仓位重分配计划，核心方向≥50%

### 超级行情的特别规则
> 在超级行情中，踏空焦虑是**必然出现**的——因为没有人能完美抓住起点。关键不是消除焦虑，而是**不被焦虑改变姿势**。

- **承认踏空**：对，我错过了第一根阳线。这是事实，不是失败。
- **拒绝追赶**：用正确姿势（核心龙头重仓）只吃剩下的涨幅，也好过用错误姿势追回全部涨幅。
- **对标检查**：如果我没有踏空焦虑，我会买什么？买多少？——然后照做。

## 踏空板（市场合力板）

### 什么是踏空板？

当市场突然出现大的盘面反转（如利好刺激、情绪逆转），前排核心票纷纷上板，量能急剧放大。未能一早抢到前排的踏空资金，会在下午选择**有辨识度的核心票**进行打板——这就是踏空板，也叫做**市场合力板**。

### 形成逻辑

1. 盘面突发反转，前排上板封死
2. 踏空资金焦虑，但又不想追后排杂毛
3. 资金自发聚焦**近期辨识度高、或前期核心龙头**的票
4. 市场合力将其拉升上板

### 交易要点

- **买入方式**：打板买入
- **选票标准**：辨识度 > 消息面 > 技术形态。资金只会聚焦"大家都认识"的票
- **溢价特征**：踏空板第二天溢价通常不错，负反馈不大，**盈亏比较高**
- **本质**：这不是追高——这是市场合力驱动下的确定性买点

### 踏空板 vs 焦虑式追涨

| | 踏空板 | 焦虑式追涨 |
|:---|:---|:---|
| **标的** | 高辨识度核心票 | 什么涨追什么 |
| **姿势** | 打板买入 | 追高半路 |
| **逻辑** | 市场合力确认 | 害怕错过 |
| **结果** | 溢价稳定，盈亏比高 | 买在日内高点 |

---

> **核心洞察**：踏空焦虑最危险的地方不是它让你感到痛苦——而是它让你**看起来在努力**。频繁换仓、多方下注、积极操作——这些在震荡市是勤奋，在超级行情里是慢性自杀。但踏空板恰恰相反——它不是焦虑驱动的乱追，而是识别市场合力后的**确定性打板**。
