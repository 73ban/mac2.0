# 超短模式对比：鄂华少 vs 小彬彬

## 两种交易宇宙的对比

| 核心维度 | 鄂华少 超短模式 | 小彬彬 波段体系 |
| :--- | :--- | :--- |
| **资金体量** | 推测中小体量（追求年翻倍） | 5000万实盘 |
| **核心周期** | 1-2天（隔日） | 1周 至 4个月 |
| **利润来源** | 情绪溢价、隔日溢价、轮动低吸博弈 | 产业趋势的完整波段（20%-100%+） |
| **选股逻辑** | 新题材龙头、市场明牌总龙头 | 产业瓶颈环节中容量最大的核心票 |
| **买卖点** | 首日上车、低吸蹲守、不追高接力 | 研究透后等合适的买点，忽略日内波动 |
| **核心纪律** | 不重仓单票、严格止损、克服确认偏误 | **不做T、不死扛、不追高、不频繁换股** |
| **胜率/赔率** | 追求高胜率，小盈小亏累计 | 胜率50-60%，靠极高盈亏比赚钱 |
| **心态/认知** | 克服"不肯认亏"心理，开盘不及预期→卖出 | 必须研究透才敢重仓，忘掉上一笔交易 |
| **信息处理** | 关注消息热度，大V是信息源不是决策器 | 靠公开信息自己研究，几乎不看盘 |

## 形象比喻

- **鄂华少 = 狩猎者**：在情绪森林里快进快出，赚市场情绪波动的钱
- **小彬彬 = 农民**：选定肥沃土地（产业趋势），播种后耐心等待作物生长

## 对我们的核心启示

### 1. 一致性是第一准则
你的性格和资金量更匹配哪种模式？
- 做超短（鄂华少）→ 坚决止损不格局
- 做波段（小彬彬）→ 坚决不做T、不频繁换股
- **不能在两种模式之间反复横跳**

### 2. 禁止模式漂移
> 在高潮次日用波段的逻辑（产业趋势好）去接超短的盘（情绪退潮），是5月22日错误的根源。

详见 [[错误/模式漂移]]。

### 3. 体系净化："不做T"纪律
小彬彬强烈反对做T，认为"越做越觉得扯淡"。需统计数据验证做T是否正期望。这一矛盾记录在 [[模式漂移#待验证问题]]。
