# 假正反馈

## 定义

**假正反馈**：赚钱→信心膨胀→放大仓位→亏损→再加仓→大亏的正反馈循环。

与 [[概念/浮盈麻醉剂]] 互补——浮盈麻醉剂管"不舍得卖"，假正反馈管"卖了赚钱的去追更大的"。

## 机制

```
盈利（+12.9%）→ "我赚钱了，可以赌更大" → 单日加仓179%
    → 次日市场逆转 → 全线砍仓亏损 → 信心受挫 → 急于翻本 → 再加仓 → 大亏
```

## 经典案例

### 案例1：有研新材 11/21→11/22（最强证据）

| 时间 | 操作 | 结果 |
|------|------|------|
| 11/21 | 卖有研+12.9%（+2,550元） | 赚钱，信心膨胀 |
| 11/21 | 单日加仓179%（4.7万→13.1万） | 放大风险敞口 |
| 11/22 | 93%个股暴跌 | 全线砍仓-3,324元 |

**精确数据链**：4.7万→13.1万（+179%）→-3,324元。因果链条完整，时间戳精确。

### 案例2：10月柏星龙、佳先股

同样的模式：赚钱→加仓→亏回去→再加仓→大亏。跨周期一致性确认。

## 与现有概念的互补关系

| 概念 | 管什么 | 触发条件 |
|------|------|------|
| [[概念/浮盈麻醉剂]] | 不舍得卖 | 持仓中浮盈 |
| **假正反馈** | 卖了追更大 | 已兑现盈利后 |
| [[概念/盈利归因错误]] | 错误归因 | 将单次成功视为能力 |

三者共同构成"盈利后的纪律崩溃"全家桶。

## 防控铁律

1. **盈利后24小时冷静期**：大赚后当日不得加仓超过50%
2. **单日加仓上限**：无论盈利多少，单日加仓不得超过前日持仓的50%
3. **盈利归因检查**：赚了钱先问"是市场给的还是我判断对的"
4. **假正反馈信号**：当出现"我赚钱了可以赌更大"的念头时，立即停止交易

## 识别信号

- [ ] 刚卖掉赚钱的票，立刻想追另一只
- [ ] 仓位比昨天大了50%以上
- [ ] "反正赚了，亏一点也没关系"的心理
- [ ] 买的票不是研究过的，而是"看起来要涨"
