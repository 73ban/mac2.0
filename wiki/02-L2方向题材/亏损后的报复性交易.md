# 亏损后的独立决策

## 核心原则

**亏损后能否开仓，不取决于"刚亏了多少钱"，而取决于"现在有没有确定性买点"。**

买卖是两个独立决策。上一笔交易的盈亏，不影响下一笔交易的判断——除非它干扰了你的动机。

## 区分：报复性交易 vs 独立买回

| | 报复性交易 | 独立买回 |
|:---|:---|:---|
| **动机** | "必须今天赚回来" | "这个买点符合我的模式" |
| **标准** | 降低了选票门槛 | 同样严格的确定性要求 |
| **仓位** | 比平常更大（想翻本） | 按确定性正常匹配 |
| **标的** | 什么涨追什么 | 核心追踪池内的标的 |
| **心态** | 紧张、怕再亏 | 冷静、按计划执行 |

**关键判断：如果这笔买入在盈利时也会做，它就是独立交易。如果只因为亏损了才做，它就是报复性交易。**

## 触发识别

以下信号出现任意两项，说明动机已经偏了：
- [ ] 买入理由里包含"把刚才亏的赚回来"
- [ ] 仓位明显大于平常
- [ ] 标的在核心追踪池之外，临时扫描涨幅榜选的
- [ ] 买入后立刻紧张地盯着价格
- [ ] 不确定这笔交易的逻辑是什么，只知道"它在涨"

## 正确做法

1. **割肉=纪律执行**，本身就是对的
2. 割肉后，用同样的标准审视盘面——有没有确定性买点？
3. 有 → 正常开仓，仓位按确定性匹配
4. 没有 → 不交易
5. 绝对不要因为"刚亏了"就降低选票标准

> **亏钱本身不是问题。亏钱后降低标准、用翻本动机驱动交易，才是问题。**
