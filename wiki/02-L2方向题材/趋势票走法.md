# 趋势票走法

## 定义

标准趋势票走法：**不断创新高 → 阶段盘整 → 继续突破创新高**

完整循环：拉升→消化→再拉升

## 案例：光迅科技（002281）

- 创新高后盘整数日
- 继续突破走高

盘整期的意义：为下一个台阶蓄力，而非走弱。

## 持仓意义

- **盘整期≠走弱** — 不能因几天不涨就清仓
- **需要消费耐心** — 等待下一个台阶的蓄力完成
- **核心判断依据** — 逻辑（而非价格波动）

## 与止损的区分

- 趋势票盘整 → 正常 → 持有
- 逻辑破位/板块退潮 → 异常 → 卖出

关键：**区分正常盘整和真正走弱。** 正常盘整 = 缩量、不破关键均线；真正走弱 = 放量破位。
