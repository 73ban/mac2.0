# 趋势主升参与模式

## 定义

趋势主升参与模式是针对“机构主导、大市值趋势股引领”的行情结构而设计的参与框架。它与[[模式/绕异动临界点参与规则]]形成互补：后者适用于游资主导的连板情绪周期，前者适用于大资金配置驱动的趋势主升周期。

## 场景识别：何时启用此模式

| 指标 | 连板情绪周期（用绕异动） | 趋势主升周期（用此模式） |
|------|----------------------|------------------------|
| 领涨标的市值 | 小市值（<100亿） | 大市值（>500亿） |
| 资金性质 | 游资+散户 | 机构+外资 |
| 连板高度 | 5板+ | 2-3板以内 |
| 涨停结构 | 连板高标引领 | 大票中军稳步推升 |
| 典型信号 | 绕异动博弈 | 中军不放量滞涨 |

## 交易框架

### 买点
- **核心原则**：不追高，等待分歧低吸
- **最佳买点**：回踩5日线或10日线不破时介入
- **次优买点**：连续缩量调整2-3日后企稳回升
- **禁止操作**：涨停板追板（趋势票涨停次日大概率分化）

### 卖点
- **减仓信号**：放量滞涨（成交量为前日1.5倍以上但涨幅<1%）
- **清仓信号**：跌破20日线或关键趋势线，或出现单日放量大跌（>5%且量>2倍均值）
- **滚动操作**：主升途中可利用5日线/10日线做T，保留底仓

### 仓位
- **分批建仓**：首次介入不超过目标的1/3仓位，确认趋势后逐步加仓
- **仓位上限**：单只趋势中军不超过总仓位40%（区别于情绪票的更高集中度）
- **加仓条件**：回踩均线不破+缩量企稳，可加至目标仓位

### 持有周期
- **核心逻辑**：赚取产业趋势的钱，而非情绪溢价
- **目标持有**：2-4周（远长于连板票的1-3天）
- **不退出的条件**：产业逻辑未变 + 中军未放量滞涨 + 主线未切换

## 与“绕异动”模式的场景分工

```
开盘判断：涨停榜是大票还是小票主导？
  ├── 大票+低连板 → 趋势主升周期 → 使用「趋势主升参与模式」
  └── 小票+高连板 → 连板情绪周期 → 使用「绕异动」模式
```

**禁忌**：两种模式不可混用。不能用绕异动的“博弈监管规避”逻辑去分析中兴通讯，也不能用趋势中军的“分批建仓持有”逻辑去参与连板高标。

## 风险警示

- **主升确认失败的条件**：次日中兴放量分歧+富联放量滞涨，则重新评估
- **贝塔误判为阿尔法**：主升行情中盈利可能来自市场贝塔，警惕[[概念/修复速度麻醉效应]]
