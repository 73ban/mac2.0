# 分化→主升确认模式

## 定义

一个趋势板块从"内部分化严重"到"全面共振上涨"，往往标志着主升浪的正式确认。分化期是筛选真龙的过程，共振日是真龙带飞全板块的信号。

## 模式路径

```
分化期（D-1）                →        主升确认（D）
━━━━━━━━━━━━━━━━━━          ━━━━━━━━━━━━━━━━━━━
板块内部严重分化                    板块内部全线共振
· 趋势龙头独自新高                    · 龙头继续高举高走
· 后排标的跌停或杀跌                  · 新龙出现+20%涨停、连板
· 板块分歧极大                       · 板块涨停数暴增
· 逻辑被质疑                         · 逻辑被全面认可

操作策略：不参与，观察              操作策略：确认时追龙头/分歧时低吸
```

## MLCC实证：6/1 → 6/2

| 维度 | 6/1（分化期） | 6/2（主升确认） |
|------|-------------|---------------|
| **龙头** | 风华高科趋势新高（独撑） | 风华高科继续高举高走 |
| **后排** | 宏和科技跌停、艾华集团跌停 | 达利凯普+20%（射频MLCC全球前五） |
| **连板** | 无 | 火炬电子2板涨停（英伟达VR200） |
| **逻辑** | MLCC需求暴增但分歧大 | 逻辑链完整：英伟达→MLCC→OEM涨价→高盛背书 |
| **资金** | 犹豫 | 达利凯普/火炬涨停→资金涌入确认 |

## 提炼规律

1. **分化期特征**：
   - 趋势龙头独立走强（证明龙头逻辑硬）
   - 后排大跌或跌停（洗掉炒作资金）
   - 板块内部赚钱效应和亏钱效应并存

2. **确认信号**（需同时满足）：
   - 龙头继续新高/连板
   - 出现新龙（首板涨停+逻辑有辨识度）
   - 赚钱效应扩散至板块其他标的
   - 外部催化逻辑进一步强化

3. **操作窗口**：
   - 分化期：观察为主，不追高买入——分歧太大，假突破概率高
   - 确认日：可以追板块内最强的票（但已面临高位风险）
   - **最佳策略**：确认后的首次分歧→低吸龙头（等板块共振后第一次回调）

## 交易含义

**不要试图在分化期押注哪只票能走出来**。等确认信号明确后，追最强的那只，或等确认后第一次分歧低吸。这避免了在分歧期赌方向的错误模式。
