# 概念：三背离格局

## 定义

**三背离格局**是指在市场过渡期出现的三种结构性扭曲同时存在的高风险状态：

1. **指数与个股背离**：指数微跌或不跌（权重护盘），但市场大多数个股（如4000+）下跌
2. **龙头与板块背离**：主线核心龙头持续涨停，但所在板块指数走弱或下跌
3. **板块间方向背离**：退潮方向（如科技大票）持续失血，脉冲方向（如资源）单日暴拉

这种格局的本质是**资金在极度扭曲的结构中运行**，不可持续。一旦护盘力量撤离或龙头断板，极易引发“指数补跌 + 高标断板”的**共振下杀**。

## 三大背离详解

### 背离一：指数 vs 个股
- **现象**：指数微跌（上证-0.17%）或翻红（创业板+0.54%），但全市场超4000股飘绿
- **成因**：权重股（大金融、大市值科技等）被资金护盘，掩盖了个股层面的普遍亏钱效应
- **危险**：一旦护盘资金撤出，指数可能**补跌**，引发恐慌性抛售

### 背离二：高标 vs 板块
- **现象**：旧主线龙头连续涨停（宝鼎科技14天9板），但板块指数开始下跌（PCB-0.58%）
- **成因**：这是[[概念/内部缩圈]]的极限表现——资金极度集中于绝对龙头，放弃跟风股
- **危险**：龙头一旦断板，因缺乏板块支撑，可能引发**踩踏式下跌**

### 背离三：退潮 vs 脉冲
- **现象**：科技大票/算力持续阴跌（10日主力净流出3000亿级），资源股单日暴涨（铝+5.52%）
- **成因**：主力资金进行**高切低**的风向切换，但新方向尚未确立
- **危险**：若新方向事件驱动不及预期，脉冲回落，旧方向又已失血——**两头踏空**

## 识别信号

当以下信号**同时出现3个以上**，应高度警惕三背离格局：

- [ ] 涨跌比极度扭曲（>1:3）
- [ ] 指数平稳或微跌，但跌停家数显著增加
- [ ] 旧主线龙头涨停但板块指数走弱
- [ ] 出现新的低位方向单日暴拉（涨超5%）
- [ ] 连续多日主力净流出
- [ ] DDX/DDY双负（主力卖、散户接）
- [ ] 中高位连板股出现频繁“硬封”（多次炸板后强封）

## 风险路径

三背离格局的常见演变路径：

```
三背离格局
    │
    ├─→ 新方向确认接力（如资源连板）→ 成功过渡，风格切换
    │
    ├─→ 旧龙头断板 + 指数补跌 → 共振下杀，全面退潮
    │
    └─→ 扭曲持续但方向不明 → 震荡轮动，操作难度极大
```

## 操作原则

处于三背离格局中，核心原则是：

> **不做方向性押注，等待市场选择方向。**

1. **降低仓位**：总仓位降至30%以下
2. **拒绝追高**：中高位跟风股、脉冲方向的首日暴拉，不追
3. **聚焦绝对龙头**：若参与残局博弈，只做龙一
4. **设防共振下杀**：提前设好止损，一旦龙头断板 + 指数走弱，清仓
