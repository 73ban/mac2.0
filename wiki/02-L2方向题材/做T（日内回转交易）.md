# 做T（日内回转交易）

## 定义

在持有底仓的基础上，进行日内高抛低吸操作，赚取差价。T仓必须当日平仓，不增加净持仓敞口（理论上的正确做法）。

## 首次成功记录

2026-05-28 深科技做T：

- 买入均价约42.07元，卖出均价约43.06元
- 1700股×(43.06-42.07) ≈ **+2,040元差价**
- 这是用户交易记录中罕见的做T成功案例

## 操作规范

| 规范 | 内容 |
|------|------|
| T仓必须当日平 | 做T的买入部分必须在当日卖出，不得变相加仓 |
| T仓不改变持仓方向 | 做T是降本操作，不是加仓理由 |
| 区分T仓盈亏与持仓风险 | T仓赚了≠持仓安全（见下方风险警示） |

## 🚨 核心风险：做T成功感遮蔽持仓风险

这是本概念最重要的警示：

- 做T赚了2,040元 → 产生"我操作很好"的错觉
- 但净持仓2600股、浮亏-5.01% → **实质上是追高被套**
- 做T的成功感可能产生"过度自信"效应 → 降低止损警觉 → 小亏变大亏

**关联风险路径**：
做T成功 → 过度自信 → 忽视持仓浮亏 → 不执行止损 → 浮亏扩大 → [[概念/贪婪驱动的纪律撤销]] 的变体
