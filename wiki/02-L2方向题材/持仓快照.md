# 持仓快照

## 定义

持仓快照（Portfolio Snapshot）是指在特定时间点（通常为交易日收盘后）对个人投资组合全部头寸的静态记录。其核心要素包括：股票代码、持仓数量、持仓成本、当前市价、浮动盈亏、仓位占比。

## 在复盘链条中的定位

持仓快照是整个复盘闭环中**不可或缺的中间环节**，处于"交易流"与"库存评估"的交汇点：

```
交割单（交易流） → 持仓快照（库存状态） → 风险评估 → 次日决策
```

| 复盘要素 | 数据来源 | 能回答的问题 |
|----------|----------|-------------|
| 交割单 | 当日成交记录 | "我做了什么？" |
| **持仓快照** | **当日收盘头寸** | **"收盘后我持有什么？风险敞口多大？"** |
| 市场快报 | 当日市场数据 | "市场给了什么信号？" |

**没有持仓快照的复盘是有缺陷的复盘**：

- 只知道买了什么、卖了什么，不知道收盘后实际持有什么
- 无法计算板块集中度风险（如 PCB 持仓占比 80% 无法量化）
- 无法评估次日开盘前的 VaR（风险价值）

## 理想格式

| 字段 | 示例 | 必要性 |
|------|------|--------|
| 股票代码 | 002463 | 必须 |
| 股票名称 | 沪电股份 | 必须 |
| 持仓数量 | 10,000 | 必须 |
| 持仓成本 | 18.50 | 必须 |
| 最新市价 | 19.20 | 必须 |
| 浮动盈亏 | +3.78% | 必须 |
| 仓位占比 | 35% | 必须 |
| 所属板块 | PCB/半导体 | 建议 |
| 买入日期 | 2026-05-20 | 建议 |
