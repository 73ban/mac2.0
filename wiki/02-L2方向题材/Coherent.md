# Coherent Corp.

> **一句话定位**：全球光电子与半导体材料龙头，**6英寸磷化铟（InP）晶圆平台的量产先驱**，率先实现6英寸产线良率超越3英寸产线的关键跨越。

## 1. 公司简介

Coherent Corp. 是全球领先的光电子和半导体制造商，业务覆盖材料、网络、激光器三大板块。在 AI 算力产业链中，Coherent 处于 **最上游的物理层**——其制造的 InP 晶圆和激光器芯片，是光模块和 CPO 交换机的核心原材料。

## 2. 技术中心：6英寸 InP 晶圆平台

### 2.1 关键里程碑

| 节点 | 时间 | 状态 |
|------|------|------|
| 6英寸平台良率超越3英寸 | 2026年5月 | ✅ 宣布达成 |
| 6英寸产线认证推进中 | 2026年 | 🔄 200G EML / CW激光器等多产品 |
| 内部InP产能翻一番 | 2026年底前 | 🔄 进行中 |
| 产能再翻一倍以上 | 2027年 | 📅 规划中 |
| 大部分生产迁移至6寸 | 2027年及以后 | 📅 远期 |
| 芯片成本降低目标 | — | >60% |
| 单晶圆产出提升 | — | 4x |

### 2.2 生产布局

| 晶圆厂 | 地点 |

## 4. 投资与产能驱动

- **NVIDIA战略投资**：**20亿美元**，直接推动光网络基础设施扩展
- **扩产节奏**：2026年底产能翻一番 → 2027年再翻一倍以上
- **目标市场**：AI数据中心光互连（800G/1.6T/3.2T）、6G无线网络

## 5. 财务追踪锚点

| 指标 | 关注逻辑 |
|------|----------|
| **毛利率（Gross Margin）** | 6寸良率带来的降本贡献，最终应在毛利端显著体现——这是验证“良率-成本双杀”是否兑现的最硬指标 |
| **营收增速与产能释放匹配度** | 扩产投资是否被足够的终端需求消化 |
| **CAPEX 与产能节奏** | 验证2026年底/2027年扩产目标是否如期推进 |

## 6. 风险评估

### 6.1 短期风险
- **良率声明的可信度**：缺乏下游客户（NVIDIA）或第三方独立良率测试、验证
- **宣传与实际量产的落差**：官方汇报存在正面引导动机

### 6.2 中长期风险
- **竞争追赶**：欧洲（Eindhoven）、日本财团、Lumentum的6英寸追赶可能缩短先发优势窗口
- **替代技术路线**：薄膜铌酸锂（TFLN）技术可能在长距、超高速率场景形成架构替代
- **下游需求不达预期**：若AI算力投资放缓，CPO和InP的市场空间可能低于预期
