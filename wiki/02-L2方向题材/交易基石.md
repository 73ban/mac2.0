# 交易基石

> **4条不可动摇的哲学原则** —— 所有交易决策的底层逻辑。

## 原则 1：确定性 > 盈亏比 > 稳定盈利

**决策优先级链**。每次交易前必须自问：
1. 这次交易的确定性有多高？
2. 盈亏比是否合理？
3. 长期执行能否稳定盈利？

**→ 仓位由确定性决定**：
- 高确定性 → 重仓
- 低确定性 → 轻仓或不参与
- 退潮/混沌 → 无确定性 → 空仓

## 原则 2：仓位与确定性挂钩

- **单票仓位**：由个股确定性决定
- **总仓位**：由市场环境 + 个股确定性双重校准
- **核心推论**：退潮日市场无确定性 → 总仓位归零

## 原则 3：交易模式、纪律、执行力三者统一

**这是避免"知行分离"的关键**。
- **模式**：告诉你做什么
- **纪律**：约束你不做什么
- **执行力**：确保你做到

三者缺一不可。有模式无纪律=随意交易；有纪律无执行=纸上谈兵。

## 原则 4：预判与盘面统一

- **尊重盘面实际走势，不执念**
- 盘面不配合预判 → 放弃预判
- 宁可错过，不执着于"我认为应该涨"

## 四原则在决策链中的位置

```
确定性判断 → 仓位校准 → 模式选择 → 纪律约束 → 盘面验证
    │            │          │          │          │
   原则1        原则2      原则3      原则3      原则4
```
