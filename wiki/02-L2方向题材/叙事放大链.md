# 叙事放大链

## 定义

外部信号在传播链中逐级失真、从"技术事实"演变为"投资神化"的过程。是[[概念/外部信号处理框架]]和[[概念/信息传播生态健康度]]的核心子概念。

## 英维克案例的完整放大链

```
原始事实：英维克进入英伟达MGX生态（技术兼容认证）
    ↓ 第一级放大（券商研报）
"进入英伟达NPN供应商名单" → "进入GB200/300 NVL72供应商名录"
    ↓ 第二级放大（财经媒体+论坛传播）
"通过NPN Tier 1认证" → "成为英伟达GB300液冷解决方案领先供应商"
    ↓ 第三级放大（市场共识形成）
"英维克=英伟达大陆唯一核心供应商" → "拿下中国区近50%份额"
    ↓ 估值表达
PE 107x — 市场为"英伟达直接一供50%份额"定价
```

## 放大链的驱动因素

1. **卖方研报利益驱动**：30余篇券商研报维持推荐评级，需要强化看多叙事
2. **市场情绪自我强化**：股价上涨 → "英伟达概念"标签 → 资金涌入 → 更高估值 → 需要更强的叙事 → 叙事升级
3. **信息不对称利用**：英伟达不公开AVL → 信息真空 → 容易被填充以"最乐观的解释"
4. **公司模糊回应的默许**：英维克以"保密条款"回避正面确认，市场将其解读为"默认利多"

## 诊断框架

当评估任何标的的"概念叙事"时，执行以下检查：

| 检查点 | 问题 | 英维克案例 |
|--------|------|-----------|
| **L1信源存在** | 公司正式公告中有量化描述吗？ | ❌ 无 |
| **概念定义追溯** | "Tier 1""核心供应商"最早来自哪里？ | L3网站+券商研报 |
| **公司回应态度** | 公司是主动确认还是回避模糊？ | 始终回避 |
| **信息源利益冲突** | 传播者是否有维持股价/评级的利益？ | 券商有维持推荐评级的利益 |
| **逐级放大痕迹** | 从"技术事实"到"投资神化"有几次跳变？ | MGX→RVL→AVL→50%份额 |

## 防范机制

1. **强制跨信源交叉验证**：任何声称"XX供应商"的说法，必须在L1级别找到量化证据
2. **追溯源头**：找到叙事链的最原始信号，对比当前传播版本
3. **放大倍数估计**：估算"原始信号→当前定价"之间的放大倍数，判断是否合理
4. **逆叙事推演**：如果这个叙事被证伪/弱化，当前估值还有多少安全边际
