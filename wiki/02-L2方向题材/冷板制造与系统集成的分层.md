# 冷板制造与系统集成的分层

## 定义

液冷散热赛道的**内部分层框架**——将液冷产业链分为上游冷板制造与中游系统集成两个不同环节，两者在技术壁垒、竞争格局、价值分配上存在根本差异。

## 两层结构

### 上游：冷板制造（高价值 + 高技术壁垒）

冷板是直接贴附于 GPU 芯片表面的精密散热金属块，通过内部微通道结构将热量传导给冷却液。

| 特征 | 说明 |
|------|------|
| **技术壁垒** | 精密微通道加工（Skived Fin / 真空钎焊）、热流工程设计，热阻 < 0.03°C/W |
| **竞争格局** | 寡头格局：Cooler Master + AVC 占 80-90% 份额 |
| **价值分配** | 冷板 ASP 较传统散热高 4 倍以上，利润集中 |
| **NVIDIA 绑定** | Rubin 起集中采购，与 GPU 强绑定，开关权在 NVIDIA |
| **代表厂商** | Cooler Master（份额 50%+）、AVC（30-40%）、Auras（第二梯队） |

### 中游：系统集成（低价值 + 竞争分散）

系统集成商负责 CDU、管路、歧管、冷板的整体设计与运维。

| 特征 | 说明 |
|------|------|
| **技术壁垒** | 系统工程 + 运维能力，但冷板核心组件外购 |
| **竞争格局** | 较分散，国内外多家厂商竞争 |
| **价值分配** | 利润空间可能被上游冷板价格锁定 |
| **NVIDIA 绑定** | 与 NVIDIA 无直接组件级绑定，冷板采购受 NVIDIA 集中采购政策影响 |
| **代表厂商** | 英维克（002837）、高澜股份（300499） |

## 为什么这个分层重要

1. **修正单一认知**：此前将"液冷"视为一个整体，将英维克/高澜视为核心标的。实际上，真正的技术高地和利润高地在上游冷板。
2. **解释利润分配结构**：NVIDIA 集中采购冷板意味着冷板的定价权从集成商转移到上游，集成商利润可能被压缩。
3. **指导标的追踪**：AVC 等冷板供应商的产能/订单/财务数据是更直接的需求景气度指标。
4. **可迁移框架**：这个分层逻辑可应用于其他硬件环节——光模块（芯片 vs 封装）、MLCC（粉体 vs 元器件）。

## 对现有持仓的核心推论

- [[股票/英维克-002837]]：系统集成商定位 → 长期利润率受上游冷板集中度压制 → Q2 验证需增加"冷板利润分配"维度 → 能否向上游垂直整合是关键变量。
- [[股票/高澜股份-300499]]：同理，集成商定位 → 利润空间受限 → GB300 认证悬案与 Auras 的对比更显被动。
- [[股票/中航光电-002179]]：国内冷板制造商 → 若英维克绑定中航光电，则可部分突破利润锁定。

## 微笑曲线映射

在液冷散热环节：
- **左侧（上游冷板）**：高价值、高壁垒、寡头格局 —— 微笑曲线高点。
- **中部（系统集成）**：中等价值、竞争分散、利润受两端挤压 —— 微笑曲线低点。
- **右侧（运维服务）**：可能具备长期粘性价值，但当前尚未形成独立利润点。

详见 [[概念/液冷产业链微笑曲线]]。

## 操作纪律

- 评估液冷赛道时，**先看冷板厂商景气度**（AVC 营收/产能），**再看集成商订单**（英维克合同负债）。
- 当冷板厂满产扩产时，液冷赛道需求确认——集成商收入端受益，但毛利率需单独评估。
- 英维克/高澜能否突破冷板利润锁定，取决于是否能自研冷板或深度绑定国内冷板制造商（中航光电等）。
