# 博弈二波与双头的安全边际

> **本概念提供仓位与风控框架。选股与择时的具体量化标准，请参见[[龙回头]]战法。**

## 核心原则：三条红线

博弈二波的本质是在龙头股主升浪结束后，博弈其充分回调后的反抽或新主升。这属于高风险操作，必须严格遵守以下三条安全边际原则：

### 1. 价格低——不追高，等回调到位

买入价格必须显著低于第一波高点。具体量化标准参见[[龙回头]]的"回调深度"：
- 回调至第一波涨幅的50%左右（黄金分割位0.382-0.618）
- 不破20日均线或关键支撑位
- 回调量能必须缩至第一波峰值量能的1/3以下（证明主力未出逃）

### 2. 信号先——等右侧确认，不预判底部

- 必须有明确的止跌企稳信号（十字星、单针探底、缩量小阳线）
- 必须有放量阳线覆盖回调阴线的再启动确认
- 在大盘共振环境下（大盘下跌导致被动回调），信号的可信度更高

### 3. 仓位分——分批建仓，不孤注一掷

- 初始试探仓位不超过计划总仓位的30%
- 确认趋势延续后再加仓
- 不因"确定性高"而一次性打满

## 与龙回头的关系

- **[[龙回头]]** = 选股 + 择时框架（如何识别龙头、判断回调充分性、确认启动信号）
- **本概念** = 仓位 + 风控框架（如何在二波博弈中控制风险、防止高位接盘）

两者形成完整的"二波博弈体系"：龙回头解决"什么时候买、买什么"，安全边际解决"买多少、怎么止损"。

## 大唐发电式参与

以大唐发电为典型案例的博弈二波参与模式：

| 维度 | 要求 |
|------|------|
| 价格 | 回调至第一波涨幅的50%附近 |
| 信号 | 企稳K线 + 放量阳线确认 |
| 仓位 | 分三批建仓，间隔1-2天 |
| 止损 | 跌破20日均线或回调低点立即离场 |

## 风险警示

- 二波博弈 ≠ 高位接盘。若回调破位，必须止损，不能将短线博弈做成长线套牢。
- 弱势市场禁用。退潮期龙头股走A杀的概率大增。
- 警惕出货假象。放量下跌的回调不是洗盘，是出货。
- 与[[错误/T+1锁仓流动性风险]]联动：启动日追入可能面临当日冲高回落的锁仓风险。
