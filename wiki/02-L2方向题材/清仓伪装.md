# 清仓伪装

## 定义

清仓伪装指交易者通过卖出动作制造“止损”“减仓”“出清”的错觉，但实际上风险敞口并未减少或反而扩大。其本质是 **用清仓的表象掩盖仓位管理的不诚实**。

## 已识别变体

### 类型 A：换方向全清

- **表现**：清掉所有持仓，立即等额（或更大）全仓买入其他方向。
- **诊断**：看似执行了止损，实则只是换了赌桌。
- **案例**：2024-10 月中旬某次出清后当日追入反向板块。

### 类型 B：同票卖出后加倍买回（新增完整样本）

- **表现**：卖出某票 → 立刻（数秒～数分钟）以更大仓位买回同票或同方向。
- **核心机制**：卖出由恐慌触发，反悔由“不想错失反弹”驱动，仓位逆势膨胀。
- **终极样本**：2024-10-31~11-01 银邦四进——
  - 10/31 14:xx 卖出 600 股 @18.92
  - 10/31 14:xx 立刻融资买回 2000 股 @18.92（仓位翻 3.3 倍）
  - 11/1 出清日全部清仓 @18.00
  - 结果：2,000 股的亏损远大于原 600 股，且净亏 -4.9%。
- **新洞见**：此序列是**清仓伪装 + 强迫性交易的叠加**，最终走向真清仓，说明伪装的终点仍是亏损。

### 类型 C：分仓集中变体（待补充案例）

- 卖出多只分散持仓，集中到单票或少数票，名义上“集中兵力”，实则为翻本下注。

## 识别清单

| 问题 | 是/否 |
|------|-------|
| 卖出后 5 分钟内再次买入？ | |
| 新买入的仓位 ≥ 原卖出仓位？ | |
| 新标的与原标的为同一只、同一板块或与翻本有关？ | |
| 买入决策发生于卖出之前（即早就想买，只是用卖出腾资金）？ | |

两题以上为“是”，即高度疑似清仓伪装。

## 防范规则

1. **卖出后 60 分钟禁入期**（此规则可拦截所有类型 A/B）。
2. **同票反向禁入**：卖出后 24 小时内禁止买入同票。
3. **卖出日报备**：记录卖出原因与计划再入条件，无计划者不得开新仓。
