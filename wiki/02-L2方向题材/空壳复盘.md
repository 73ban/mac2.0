# 空壳复盘

## 定义

**空壳复盘** 是指复盘的外部框架（模板、维度标题）完整保留，但核心内容全部缺失或仅以“数据缺失”“待定”填充的状态。它是纪律松懈的早期预警信号，处于实质崩溃（停止复盘）与形式崩溃（放弃模板）之间的中间态。

## 特征识别

| 特征 | 说明 |
|------|------|
| 模板完整 | 所有复盘维度的标题和框架仍在 |
| 内容空洞 | 多个关键维度标注“数据缺失”或完全空白 |
| 操作极简 | 仅记录最基本的买卖数据，无逻辑说明 |
| 反思缺失 | “待定”“待回顾”代替了实质分析 |
| 资金归零 | 通常伴随可用本金为 0.00 或极小值 |

## 与纪律断裂的关系

空壳复盘是纪律从实质向形式退化的关键拐点：

1. **健康的复盘**：框架 + 实质内容（数据、分析、反思）
2. **空壳复盘**：框架仍在，但内容流失 → **预警阶段**
3. **形式崩溃**：连模板也放弃，完全停止复盘
4. **实质崩溃**：不再关注盘面，复盘失去意义

空壳复盘的狡猾之处在于：从外部看，“仍在坚持复盘”；但从内部看，已丧失了看见自己的能力。保留模板是过去建立的惯性的最后痕迹。

## 可能的成因

- 盘前无准备 + 盘中无关注 + 盘后无动力的三重叠加
- 空仓期或被迫停手的倦怠
- 连续亏损后的回避心理
- 意志力耗尽但还未完全放弃的过渡状态

## 应对策略

1. **识别为空壳后立即触发干预**：空壳复盘不应被视为“至少还在复盘”，而应被视为“纪律危机”。
2. **回溯当日盘面**：即使当日无暇顾及，事后仍应补充盘面关键信号。
3. **强制执行最低复盘标准**：至少完成赚钱效应、盘面信号、操作逻辑三项，否则不计入有效复盘。
4. **与退潮纪律联动**：如果空壳复盘出现在退潮期，应主动降仓或空仓。
