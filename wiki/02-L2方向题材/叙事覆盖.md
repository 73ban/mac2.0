# 叙事覆盖

## 定义

当盘面现实与持仓预期发生冲突时，思维自动构建新叙事以**覆盖/否定/淡化**不利信号，从而合法化持仓冲动。

**核心结构**：
1. 旧叙事被盘面证伪（如"利好驱动追高"→退潮确认）
2. 立即构建新叙事（如"逻辑未坏""低开是上车机会"）
3. 新叙事覆盖旧现实的紧迫性
4. 为不退场甚至加仓提供正当性

## 🆕 新增案例：5/21退潮判断的自我否定

### 完整链条

| 步骤 | 内容 | 时间点 |
|------|------|--------|
| 旧叙事 | "三重利好→科技股机会"→开盘追高中国长城、综艺股份 | 09:47-09:52 |
| 盘面证伪 | 评分6分→2分，板块全军覆没 | 上午→下午 |
| 退潮确认 | "量化砸盘+获利兑现造成的股灾体感" | 收盘 |
| 新叙事构建 | "这种兑现不改变中期逻辑，低开反而是二次上车窗口" | 收盘后 |
| 叙事覆盖 | "逻辑未坏"覆盖"筹码崩解" | 当晚 |
| 危险操作 | 计划次日加仓做T | 计划中 |

### 关键特征

1. **退潮判断与新叙事共存**：退潮已确认，却不影响"次日低吸"计划
2. **用抽象逻辑否定具体筹码信号**："中期逻辑未坏"否认了当天真金白银的砸盘
3. **自我合理化**："前提是逻辑没坏、个股没破位"——预设了永远不会被否定的条件

## 与[[概念/量化集中兑现]]的关系

量化集中兑现代来一个危险的叙事窗口：
- **事实**：量化+获利盘集中砸盘 → 筹码崩解
- **叙事覆盖**："逻辑未坏 + 筹码兑现 = 短期回调 = 低吸机会"

叙事覆盖的危险在于：它把一个**需要离场的信号**重新解释为一个**可以加仓的机会**。

## 防范

1. **退潮一旦确认，不构建任何次日加仓叙事** — 铁律
2. **区分"逻辑"与"筹码"**：逻辑未坏≠不会跌
3. **存档矛盾时刻**：当出现两个互相矛盾的判断，必须存档并事后验证
4. **遵守规则而非叙事**：[[策略/退潮票早盘清仓]] — 不挑时间
