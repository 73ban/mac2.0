# 题材催化节奏

## 定义

事件驱动题材的交易往往遵循可识别的**时间节点节奏**：事件预告 → 预期发酵 → 提前埋伏 → 事件兑现 → 出货/回落。理解这一节奏是参与题材交易的核心能力。

## N1X Computex案例

```
2026年5月
  ├── 联合预告"PC新时代" → 预期发酵（约1-2周）
  ├── Computex前一周 → 提前埋伏窗口
  ├── Computex黄仁勋keynote → 事件兑现（预期 vs 现实）
  ├── 若真机落地 → 情绪延续，寻找超预期
  └── 若仅为PPT → 落地即利空，出货
```

## 通用的“五阶段”框架

| 阶段 | 特征 | 策略 |
|------|------|------|
| ①消息萌芽 | 少数人知悉，盘面无明显反应 | 难参与，需信息优势 |
| ②预期发酵 | 板块资金涌入，优质票走强 | 可轻仓试错 |
| ③提前埋伏 | 事件前1-3天，埋伏盘进入 | 需有预案应对“利好出尽” |
| ④事件兑现 | 实际结果vs预期，盘面剧烈反应 | 关键决策点：加仓/减仓/清仓 |
| ⑤情绪消化 | 兑现后1-3日，情绪回归理性 | 退潮观察，不宜追高 |

## 核心风险

- **“预期跑得比现实快”** — 股价已提前反映乐观预期，事件兑现时即使利好多也是出货信号
- **“延期”或“落空”** — PPT demo而非真机，情绪落地即利空
- **天花板逻辑** — 出货1000万台vs 2亿大盘，实质拉动有限
