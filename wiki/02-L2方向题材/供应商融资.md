# 供应商融资

## 定义

**供应商融资**是指设备制造商向上游材料商付款、同时向客户提供贷款/股权投资/信用担保，客户再用这些资金购买该设备商的设备的一种产业链融资模式。

本质上是用自己的钱制造自己的收入，当循环最终不指向真正付费的终端盈利客户时，就会形成致命陷阱。这是[[概念/中间厂商流动性陷阱]]的**产业链融资维度极端版本**。

## 核心机制

### 虚假销售收入循环
```
设备商借钱给客户 → 客户购买设备 → 设备商确认收入
↑________________________________________↓
          客户的还款依赖融资收入 → 循环断裂
```

**判断标准**：财务循环是否最终指向盈利的终端付费客户。

- **良性循环**：微软/谷歌等盈利客户产生真实需求 → 购买AI芯片 → 设备商获得真实收入
- **恶性循环**：初创公司没有盈利 → 设备商借钱给初创公司 → 设备商确认收入 → 初创公司无法还款

## 历史案例：2000年电信泡沫

### 朗讯科技
- 供应商融资15亿美元，坏账率从**2.6%→60%**
- Winstar破产时超8亿美元未偿
- "在某个时刻，朗讯已经不是在卖设备，而是在免费赠送并称之为销售"

### 北电网络
- 供应商融资14亿美元，融资条件高达设备价格的130%
- 坏账率从**25.5%→80%**（九大设备商最剧烈）
- 估值从1360亿美元→140亿美元，最终倒闭

### 行业全景
- 九大设备商合计至少**256亿美元**供应商贷款
- 光纤利用率仅**10%**，带宽价格跌**90%**
- 运营商CAPEX占比从35%骤降至10%

## 核心监控指标（迁移至当前AI算力链）

1. **终端客户质量**：主要买家是否具备独立盈利能力？
2. **融资条件透明度**：是否存在"被真正银行拒绝"的客户获得融资？
3. **产能利用率追踪**：GPU实际利用率vs装机量（2000年光纤利用率10%的等价指标）
4. **内部循环比例**：供应商融资收入/总收入的趋势
