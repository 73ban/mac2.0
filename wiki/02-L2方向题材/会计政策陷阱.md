# 会计政策陷阱

## 定义

**会计政策陷阱**，指因会计处理口径变更导致的毛利率"假象"，不反映实际盈利能力的恶化或改善。

## 三种常见陷阱

### 1. 收入确认时点变更
- **机制**：收入从完工百分比法改为某一时点确认，导致收入在不同报告期之间漂移
- **影响**：毛利率异常升高或下降，但不反映经营实质
- **识别**：年报附注"重要会计政策变更"，关注是否涉及收入确认方法

### 2. 成本归集口径变化
- **机制**：部分费用从"销售费用/管理费用"重新分类至"营业成本"
- **影响**：毛利率被直接拉低，但净利率可能不变（费用减少抵消）
- **识别**：毛利率下滑但期间费用率同步下降 → 极可能是口径调整

### 3. 存货计价方法变更
- **机制**：FIFO与LIFO的切换，在通胀/通缩环境下导致毛利率剧烈波动
- **影响**：存货计价方法变更会影响营业成本，进而影响毛利率
- **识别**：年报附注"存货计价方法"，关注是否发生变更

## 对抗会计政策陷阱的操作步骤

```
步骤1：每次分析毛利率变动前，先查看年报"重要会计政策变更"部分
步骤2：观察毛利率与期间费用率的同步变化
    ├─ 毛利率↓+费用率↑ → 真实恶化
    ├─ 毛利率↓+费用率↓同幅度 → 可能是口径调整
    └─ 无法判断 → 查看净利润率是否同步变化
步骤3：如果确认是口径调整，使用调整前口径做同比
```

## 个人投资者实操要点

- 年报附注是必读内容（"重要会计政策和会计估计变更"节）
- 东方财富、同花顺等平台通常有"财务指标变动原因"栏目，关注"因会计政策变更导致"
- 最简单的方法：对比毛利率变化与净利率变化，如果方向不一致要警惕
