# 大题材前排套利切入规则

> **14号交易模式**（新增）——大题材爆发时，前排买不到的后排套利先手，等分歧再切入前排。

## 一、核心逻辑

**前提**：大题材具有持续性，即使买了后排，也能挣钱套利——这是确定性比较高的选择。

**两段式操作**：
```
第一阶段：前排一字/秒板买不到 → 买同方向后排套利（先上车）
第二阶段：前排分歧/开板 → 卖掉后排，切入前排（换座到龙头）
```

**本质**：这是“确定性能参与行情”和“最终持有最强标的”之间的桥梁。不因为前排买不到就空仓踏空，也不因为买了后排就满足于后排。

## 二、适用条件（必须同时满足）

| 条件 | 判断标准 | 说明 |
|:---|:---|:---|
| **大题材** | 政策级别高（央行/政治局/国家级产业政策）+ 板块涨停≥8只 | 小题材不适用，后排可能直接闷杀 |
| **持续性确认** | 板块内有1只以上一字板锁死 + 次日有2连板出现 | 一日游题材排除 |
| **前排买不到** | 最强标的已封死涨停，排板无望 | 能买到前排就直接买前排，不需要套利 |
| **后排有辨识度** | 后排标的必须是同方向内“次强”——有辨识度、有资金关注，而非纯跟风 | 买最弱后排=送钱 |

## 三、操作流程

### 第一天：套利上车

1. 确认题材强度和前排封单（天风一字/中粮一字/国海换手板等）
2. 前排排板同时，选定方向内次强标的下单：
   - 优选：当天封板但反复炸板回封的（有换手、次日溢价概率高）
   - 次选：当天未涨停但量能放大、处于板块内涨幅前30%的
   - **禁止**：买板块内跌幅最大/无资金关注的纯跟风
3. 仓位控制：**套利仓位 ≤ 总仓位的30%**（这不是主仓位，是占座费）
4. 当天收盘持有：前排排板仓位（如有）+ 后排套利仓位

### 第二天：分歧切换

**情景A：前排开板/分歧 → 切换信号触发**
- 卖出后排套利仓（大概率有浮盈——因为题材有持续性）
- 买入前排龙头（开板分歧即是买点）
- 前排仓位提升至主仓位水平

**情景B：前排继续一字封死 → 继续持有**
- 后排套利仓继续持有（享受板块持续性的溢价）
- 继续排板前排（但不要因为排不到就加仓后排）
- 第三天判断同上

**情景C：前排炸板未回封 + 后排走弱 → 题材持续性不足，清仓离场**
- 同时清掉前排排板和后排套利仓
- 这说明“大题材”判断可能有误，不应继续加仓

## 五、禁止事项

| 禁止行为 | 原因 |
|:---|:---|
| ❌ 买了后排就不再切前排 | 后排套利是过渡手段，不是最终目标。满足于后排=放弃龙头溢价 |
| ❌ 后排仓位超过30% | 套利仓不是主仓位，过重会绑架资金，分歧时无法切前排 |
| ❌ 小题材/一日游题材用此规则 | 后排可能第二天直接低开-5%，没有套利空间 |
| ❌ 后排选了板块内最弱的 | 弱后排连套利价值都没有，第二天可能不是浮盈而是亏损 |

## 六、案例映射

**正向假设**（如果9/24按此规则操作）：
```
9/24: 天风证券一字买不到 → 买入国海证券（换手板/次强）
9/25: 天风分歧开板 → 卖出国海（浮盈）→ 买入天风（龙头）
结果：9/24参与行情 + 9/25持有龙头
```

**实际9/24-26对照**：
- 9/24买了中航产融（未涨停的弱后排）→ 无前排意识
- 9/25止损后买了安阳钢铁/天茂集团 → 仍是边缘标的
- 9/26终于切换到天风证券 → 但已是第3板，且仓位偏轻

如果当时有这个规则，整个操作序列会从“边缘→边缘→龙头（迟）”，变成“次强→龙头（早）”——少走弯路。

## 七、与其他模式的适用环境

| 市场阶段 | 是否适用本规则 |
|:---|:---|
| 主升爆发（情绪8-9分） | ✅ **最佳适用场景** |
| 主升分化（情绪6-7分） | ✅ 前排分歧时直接切入，无需后排套利 |
| 震荡轮动（情绪5分） | ⚠️ 前提判断难度加大，后排套利风险增加 |
| 退潮期（情绪≤4分） | ❌ 不适用——此时应空仓，不是套利 |
