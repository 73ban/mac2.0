# LPO与光模块技术路线

## 定义

光模块技术路线涵盖从传统可插拔到下一代光电共封装的多种互联形态。当前主流技术路径包括四种核心方案，其中**LPO（线性驱动可插拔，Linear-drive Pluggable Optics）** 是当前最关键的技术过渡路径——它在保持可插拔灵活性的同时大幅降低功耗，已通过英伟达认证，是新易盛进入英伟达供应链的核心技术通行证。

## 四种核心技术路线

| 技术 | 全称 | 阶段 | 功耗 | 可插拔性 | 代表厂商 |
|------|------|------|------|----------|----------|
| **可插拔** | Pluggable Optics | 主流量产 | 基准 | ✅ 是 | 新易盛、中际旭创 |
| **LPO** | Linear-drive Pluggable Optics | 规模量产 | 降低20-30% | ✅ 是 | 新易盛（已通过英伟达认证） |
| **LRO** | Linear Receive Optics | 早期量产 | 部分降低 | ✅ 是 | 新易盛（薄膜铌酸锂方案） |
| **NPO** | Near-Packaged Optics | 研发/送测 | 显著降低 | ❌ 否 | 新易盛（6.4T已发布） |
| **CPO** | Co-Packaged Optics | 预研/生态建设 | 最低 | ❌ 否 | 多厂商布局中 |
| **XPO** | eXtreme-density Pluggable Optics | 发布/送测 | 中等 | ✅ 是 | 新易盛（12.8T行业首款） |

## LPO的战略意义

### 为什么LPO是关键路径

1. **英伟达认证壁垒**：LPO是当前进入英伟达供应链的核心技术要求。新易盛LPO产品已通过英伟达认证，这是其深度绑定英伟达的基础
2. **功耗优势**：相比传统可插拔方案功耗降低20-30%，满足AI数据中心对能效的严苛要求
3. **可插拔保留**：保持了可插拔的灵活性优势，无需改变数据中心运维习惯
4. **技术过渡桥梁**：在CPO生态成熟前（预计3-5年），LPO是最优过渡方案

### LPO对竞争格局的影响

- **新易盛**：通过LPO认证率先切入英伟达供应链，获得1.6T光引擎订单
- **中际旭创**：同样布局LPO，两家在英伟达供应链中形成双供格局
- **后发者**：LPO认证需要时间，形成实质性的进入壁垒

## CPO代际切换风险

### CPO是什么

光电共封装（Co-Packaged Optics）将光引擎与交换芯片直接封装在一起，彻底消除传统可插拔的连接损耗，功耗可再降50%以上。被视为光互联的终极形态。

### 核心不确定性

1. **时点争议**：产业共识认为CPO大规模商用需3-5年，但英伟达可能加速推进
2. **生态依赖**：CPO需要交换机厂商、光引擎厂商、芯片厂商的协同生态，不是单一厂商可推动
3. **良率瓶颈**：当前CPO良率远低于可插拔方案，成本居高不下
4. **运维变革**：CPO不可插拔，对数据中心运维模式是颠覆性改变

### 对"易中天"格局的冲击

如果CPO在2027-2028年大规模商用：
- **新易盛**：已布局CPO/NPO/XPO，但1.6T可插拔的先发优势可能被重置
- **中际旭创**：作为龙一，有更多资源投入CPO研发
- **天孚通信**：作为光引擎组件商，在CPO时代可能获得新的价值定位

→ CPO是"易中天"格局最大的长期不确定性因素。

## 新易盛技术路线全景

```
当前量产：800G / 1.6T（可插拔）+ LPO + LRO（铌酸锂）+ 硅光
   ↓
研发送测：3.2T（预研）+ 6.4T NPO + 12.8T XPO
   ↓
远期布局：CPO（专利储备）+ 薄膜铌酸锂调制器 + 硅光集成
   ↓
技术目标：1.6T CPO功耗再降20%，良率提升至95%以上
```
