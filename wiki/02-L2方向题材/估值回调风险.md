# 估值回调风险

## 定义

由于前期股价涨幅巨大导致市盈率处于历史高位，若未来业绩增速未达市场预期，股价可能因估值向下修正而大幅下跌的风险。在极端情况下，业绩增速放缓和估值收缩同时发生，形成"戴维斯双杀"。

## 典型案例：中际旭创（截至2026年5月）

| 指标 | 数据 |
|------|------|
| 2025年股价涨幅 | 397.51% |
| 市盈率（TTM） | 一度达76.55倍 |
| 市值 | 超9000亿 |
| 隐含预期 | 已包含2027-2028年高增长 |

> 来源：[3][8]

## 触发条件

1. **业绩增速放缓**：季度营收/净利润增速低于市场一致预期
2. **毛利率下滑**：竞争加剧或产品结构降级
3. **下游需求不及预期**：大客户资本开支削减
4. **竞争格局恶化**：竞争者突破技术壁垒或通过客户认证
5. **市场风格切换**：资金从高估值成长股轮动至价值股

## 对[[概念/趋势主升参与模式]]的警示

中际旭创的案例揭示了趋势参与中的**反向风险**——与[[错误/趋势票卖飞综合症]]对应的，是趋势票追高后的中期套牢。

### 纪律要求

| 情境 | 操作 |
|------|------|
| 趋势完好 + 估值合理 | 按[[概念/趋势主升参与模式]]低吸分批建仓 |
| 趋势完好 + 估值高位 | 仓位减半，止盈上移，不新增仓位 |
| 趋势破位 + 估值高位 | 纪律止损，不等基本面验证 |
| 趋势破位 + 估值合理 | 观察基本面是否出现退化信号 |

### ⚠️ 特别警示

对"趋势中军"的深度基本面研究，可能产生[[概念/修复速度麻醉效应]]的变种——因为数据翔实而产生信仰，在需要止损时用"价值投资"自我合理化。记住：

> **基本面的好，不替代交易纪律的执行。**
