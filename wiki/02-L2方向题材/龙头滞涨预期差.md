# 龙头滞涨预期差

## 定义
当**板块整体大涨**时，板块中逻辑最硬的核心标的**涨幅显著落后于板块平均**，这种偏差构成预期差——前提是滞后并非由已知利空导致。

## 识别要点

| 要素 | 标准 |
|------|------|
| 板块涨幅 | 板块指数日内+2%以上 |
| 龙头涨幅偏差 | 龙头涨幅低于板块涨幅2个百分点以上 |
| 催化支撑 | 龙头是当日催化事件的**直接受益方** |
| 排雷 | 排除市场已知利空（如业绩预警、减持公告、监管问询） |

## 与[[概念/预期先行业绩滞后]]的区别
- **预期先行业绩滞后**：时间维度的滞后（预期先行，业绩要等）
- **龙头滞涨预期差**：空间维度的滞后（板块涨了，个股没涨）
- 两者可叠加：滞涨龙头如果还有未兑现的业绩预期，构成双重预期差

## 风险提示

### 假性滞涨（需排除）
- 标的已被剔除板块指数权重
- 标的存在未被文档覆盖的重大利空
- 滞涨是资金出货的结果，而非暂时的忽略

### 补涨窗口关闭
- 板块次日回调，龙头补涨逻辑被破坏
- 验证窗口以**当日开盘~收盘**为核心，T+1为最后观察窗口

## 案例：深南电路 (2026-05-29)
- 板块：PCB +3.48%
- 标的：深南电路 +0.56%（差值2.92个百分点）
- 催化：戴尔AI服务器+757%（深南电路为AI服务器PCB核心供应商）
- 优先级：高
- 待验证：滞涨原因是否涉及市场已知利空

## 执行纪律
- 预期差介入后，**当日未兑现即降级**，T+2未兑现即出清
- 不允许将"龙头滞涨预期差"转化为"长期持有逻辑"
