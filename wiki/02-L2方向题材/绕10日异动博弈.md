# 绕10日异动博弈

## 定义

**绕10日异动博弈**是一种利用A股10个交易日内区间涨幅偏离值达100%触发停牌核查规则进行的策略博弈。当个股偏离值接近触发阈值时，存在两种典型路径：

| 路径 | 机制 | 操作 |
|------|------|------|
| **资金抢筹触发** | 资金合力推高使其触及100%偏离值 → 触发异动停牌 | 封板时卖出（在停牌前离场或博弈复牌溢价） |
| **资金压盘回避** | 主力主动压价避免触发 → 股价回落后再蓄力 | 水下低吸做T，博弈后续抢筹 |

**核心优势**：所有博弈边界是**可量化**的——触发价、涨停价、偏离值百分比均有精确数值，不依赖纯情绪/题材判断。

---

## 量化框架

### 关键指标计算

```
10日异动偏离值 = (当日收盘价 / 期初最低收盘价 - 1) × 100%
触发阈值 = 100%
触发价 = 期初最低收盘价 × (1 + 100%)
```

### 大唐发电（5/18）实例

| 指标 | 数值 |
|------|------|
| 期初最低收盘（4/30） | 4.16 |
| 5/18收盘 | 7.93 |
| 10日偏离值 | **89.83%** |
| 触发阈值 | 100% |
| 触发价 | **8.33** |
| 距触发需涨 | **+5.04%** |
| 涨停价 | 8.72 |
| 涨停价是否覆盖触发价 | ✅ 8.72 > 8.33（封板必触发） |

## 情景分析模板（ABCD框架）

这是本策略的核心操作框架，可复用于任何接近异动触发阈值的标的：

### 情景A：抢筹拉板（最佳路径）
- **触发条件**：高开+快速封板
- **市场含义**：资金选择合力触发异动
- **操作**：**封板即卖点，不贪**。触发价=离场信号
- **反纪律风险**：[[概念/贪婪驱动的纪律撤销]]——"封板了还能再涨"的幻想

### 情景B：水下震荡（蓄力路径）
- **触发条件**：低开/水下，未触发异动
- **市场含义**：资金压盘回避触发，但未出货
- **操作**：低吸做T（加仓→日内冲高→减回原仓位），降低持仓成本
- **安保条件**：需要联动锚点确认（如板块龙头稳住）

### 情景C：冲高回落（出货路径）
- **触发条件**：拉高不封板→回落
- **市场含义**：资金无意或无力触发异动
- **操作**：高位减半仓，尾盘视情况回补

### 情景D：直接崩盘（失败路径）
- **触发条件**：联动锚点闷杀+板块全线崩溃
- **操作**：**立即止损**，不抱幻想

## 生存条件

与[[概念/交易模式的生存条件]]一致，绕异动博弈需要以下前置条件：

| 条件 | 说明 | 不满足的风险 |
|------|------|------------|
| **偏离值接近阈值**（>85%） | 只有在足够接近时才存在博弈空间 | 偏离值过低=触发概率极低，不值得持仓 |
| **涨停价覆盖触发价** | 封板必然触发，提供明确卖点信号 | 涨停价 < 触发价 = 封板也不触发，失去博弈锚点 |
| **联动锚点存在** | 同板块/同逻辑的龙头作为先行指标 | 无锚点=纯赌博 |
| **市场非极端退潮** | 需要有基本流动性支撑 | 极端退潮中资金无心博弈异动 |

## 盈亏比分析

- **做T路径**（B情景）：成本每降低0.1元=约1.3%安全垫，多次做T可显著降低风险
- **触发卖出路径**（A情景）：7.80→8.33=**+6.79%**，盈亏比取决于触发概率
- **失败路径**（D情景）：最大亏损取决于止损纪律

**核心风险**：触发价≠必然触发。偏离值89.83%意味着距离触发仅+5.04%，但市场可能选择压盘路径。

## 与退潮日纪律的边界测试

5/18复盘显示用户在明确退潮日（缩量4,532亿、53家跌停）以绕异动量化逻辑开仓大唐发电。这提出了一个未解决的问题：

> "量化逻辑"是否构成退潮日不开新仓纪律的有效豁免条件？

**正向论据**：绕异动边界可计算、触发价明确、有联动锚点——确定性高于一般的题材/情绪交易。

**反向论据**：5/18同日开仓的兆易创新（板上追入未封，-3.19%）显然不满足"确定性高"的标准，却也在退潮日开仓。这说明用户对"豁免"边界的把握尚不清晰。

**待T+1验证**才能得出结论。见[[概念/退潮日]]。
